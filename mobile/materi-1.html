Pengenalan Kotlin

Apa itu Kotlin?

Kotlin adalah bahasa pemrograman modern yang dikembangkan oleh JetBrains. Pada tahun 2017, Google mengumumkan Kotlin sebagai bahasa resmi untuk pengembangan Android. Kotlin berjalan di atas Java Virtual Machine (JVM) dan dapat digunakan bersama dengan Java.

Ringkas

Kode lebih sedikit, lebih mudah dibaca

Aman

Null safety bawaan

Interoperable

100% kompatibel dengan Java

Tool-friendly

Dukungan IDE yang luar biasa

fun main() {
    // ===== VAR (Mutable) =====
    // var = variable, bisa diubah nilainya
    
    var skor: Int = 0
    println("Skor awal: $skor")
    
    skor = 100  // OK - bisa diubah
    println("Skor setelah diubah: $skor")
    
    skor = skor + 50  // Menambah nilai
    println("Skor setelah ditambah 50: $skor")
    
    // Contoh lain
    var pesan = "Hello"
    println("\nPesan awal: $pesan")
    
    pesan = "World"  // OK - bisa diubah
    println("Pesan setelah diubah: $pesan")
    
    // PENTING: Tidak bisa mengubah TIPE DATA
    // pesan = 123  // ERROR! Type mismatch
    
    // ===== Perbandingan val vs var =====
    println("\n=== Simulasi Counter ===")
    
    val namaGame = "Space Invaders"  // Nama game tidak berubah
    var nyawa = 3                     // Nyawa bisa berkurang
    var level = 1                     // Level bisa bertambah
    
    println("Game: $namaGame")
    println("Nyawa: $nyawa, Level: $level")
    
    // Simulasi bermain
    nyawa = nyawa - 1  // Kehilangan nyawa
    level = level + 1  // Naik level
    
    println("Setelah bermain:")
    println("Nyawa: $nyawa, Level: $level")
}

Best Practice

Selalu gunakan val sebagai default. Gunakan var hanya jika Anda benar-benar perlu mengubah nilai variabel tersebut. Ini membantu membuat kode lebih aman dan mudah dipahami.

    // Sintaks: { parameter -> body }

    println("Double: ${double(5)}")

    val printInfo = { name: String, age: Int ->
        println("=== Info ===")
        println("Name: $name")
        println("Age: $age")
        "Info printed"  // Return value (last expression)
    }
    val result = printInfo("John", 25)
    println("Result: $result")
    
    // === Lambda dengan Collections ===
    println("\n=== Lambda dengan Collections ===")
    val numbers = listOf(1, 2, 3, 4, 5)

    val doubled = numbers.map { it * 2 }
    println("Doubled: $doubled")

// === Fungsi yang menerima fungsi sebagai parameter ===
fun calculate(a: Int, b: Int, operation: (Int, Int) -> Int): Int {
    return operation(a, b)
}

// Fungsi dengan lambda sebagai parameter
fun performOperation(
    x: Int,
    y: Int,
    operation: (Int, Int) -> Int
): Int {
    println("Performing operation on $x and $y")
    return operation(x, y)
}

// === Fungsi yang mengembalikan fungsi ===
fun createMultiplier(factor: Int): (Int) -> Int {
    return { number -> number * factor }
}

fun createGreeter(greeting: String): (String) -> String {
    return { name -> "$greeting, $name!" }
}

// === Contoh Higher-Order Function untuk repeat ===
fun repeatAction(times: Int, action: (Int) -> Unit) {
    for (i in 1..times) {
        action(i)
    }
}

// === Higher-Order Function dengan nullable ===
fun executeIfNotNull(value: String?, action: (String) -> Unit) {
    value?.let { action(it) }
}

// === Higher-Order Function untuk timing ===
fun measureTimeMillis(block: () -> Unit): Long {
    val start = System.currentTimeMillis()
    block()
    return System.currentTimeMillis() - start
}

// === Higher-Order Function untuk try-catch ===
fun  tryCatch(
    block: () -> T,
    onError: (Exception) -> T
): T {
    return try {
        block()
    } catch (e: Exception) {
        onError(e)
    }
}
