Praktikum: Firebase untuk Android Kotlin

Tujuan Pembelajaran

  • Memahami ekosistem Firebase dan layanan-layanannya
  • Mampu menghubungkan project Android ke Firebase Console
  • Mampu mengimplementasikan Firebase Authentication (register & login)
  • Mampu melakukan operasi CRUD data menggunakan Cloud Firestore
  • Memahami perbedaan Firestore dan Realtime Database
  • Mampu mengupload dan mengambil file menggunakan Firebase Storage

Prasyarat

Pastikan Anda sudah memahami materi dari praktikum sebelumnya:

  • Activity, Intent, dan dasar-dasar UI Android
  • RecyclerView dan Adapter
  • Kotlin dasar (data class, lambda, coroutines)
  • Memiliki akun Google untuk mengakses Firebase Console

Apa itu Firebase?

Firebase adalah platform Backend-as-a-Service (BaaS) milik Google yang menyediakan berbagai layanan backend siap pakai untuk aplikasi mobile dan web. Dengan Firebase, developer tidak perlu membangun server sendiri untuk fitur-fitur umum seperti autentikasi, database, dan penyimpanan file.

Yang akan dibahas di praktikum ini

Praktikum ini fokus pada 4 layanan Firebase yang paling sering digunakan dalam pengembangan aplikasi mobile: Authentication, Cloud Firestore, Realtime Database, dan Firebase Storage. Setiap bagian dilengkapi contoh kode sederhana dan hands-on activity.

Kombinasi yang Umum Dipakai

Dalam satu aplikasi, ketiga layanan ini biasanya dipakai bersama-sama, bukan pilih salah satu:

  • Firestore — menyimpan data profil user, termasuk URL foto profil
  • Storage — menyimpan file foto profil yang sebenarnya
  • Realtime Database — menyimpan status online/offline user secara real-time

Contoh: saat user upload foto profil → file dikirim ke Storage → URL-nya disimpan ke Firestore → status "sedang upload" ditampilkan via Realtime Database.

Setup Firebase

Ada lima langkah yang harus dilakukan secara berurutan: membuat project Android baru, membuat project di Firebase Console, menghubungkan keduanya, menambahkan dependencies, lalu menguji koneksi.

3. Hubungkan Project Android ke Firebase

Ada dua cara menghubungkan Android ke Firebase: menggunakan Firebase Assistant di Android Studio (cara termudah), atau secara manual lewat Firebase Console.

Cara A — Firebase Assistant (Direkomendasikan)

  1. Di Android Studio, klik menu Tools → Firebase — panel Firebase Assistant muncul di kanan.
  2. Pilih layanan, misalnya Authentication → Authenticate using a custom authentication system.
  3. Klik Connect your app to Firebase — browser terbuka untuk login Google → pilih project Firebase yang sudah dibuat → klik Connect. Jika berhasil, statusnya berubah menjadi ✓ Connected.
  4. Klik Add the Firebase Authentication SDK to your app — dependencies otomatis ditambahkan ke build.gradle.

Cara B — Manual via Firebase Console

  1. Di Firebase Console, klik ikon Android (</>) di halaman Project Overview untuk mendaftarkan aplikasi Android baru.
  2. Masukkan Android package name yang sama persis dengan applicationId di build.gradle.kts (Module: app), misalnya com.example.praktikumfirebase → klik Register app.
  3. Download file google-services.json yang disediakan, lalu pindahkan ke folder app/ di project Android Studio.
  4. Pastikan file berada di app/google-services.json (bukan di root project) — switch view ke Project di Android Studio untuk memverifikasi lokasinya.

Package Name Harus Sama Persis

Package name di Firebase Console harus identik dengan applicationId di build.gradle.kts (Module: app). Jika berbeda, aplikasi tidak akan bisa terhubung ke Firebase dan akan crash saat dijalankan.

4. Tambah Dependencies Firebase

Jika kamu menggunakan Cara A (Firebase Assistant), plugin Google Services dan dependency firebase-auth sudah otomatis ditambahkan. Kamu hanya perlu menambahkan dependency untuk layanan lain yang belum ada, yaitu Cloud Firestore.

Sudah Otomatis oleh Firebase Assistant

Setelah mengikuti Cara A, file build.gradle.kts kamu sudah berisi:

  • Plugin google.gms.google.services di Project level dan Module level
  • Dependency firebase.auth di Module level
  • File google-services.json sudah ada di folder app/

Jangan tambahkan ulang hal-hal di atas — akan menyebabkan duplikat dan error saat build.

Tambah Dependency Firestore

Karena materi ini juga menggunakan Cloud Firestore, ada dua file yang perlu diupdate.

Langkah 1 — Tambahkan versi dan library di gradle/libs.versions.toml:

gradle/libs.versions.toml 
[versions]
# ... versi lainnya ...
firebaseAuth = "24.1.0"
firebaseFirestore = "25.1.1"  // ← tambahkan ini

[libraries]
# ... library lainnya ...
firebase-auth = { group = "com.google.firebase", name = "firebase-auth", version.ref = "firebaseAuth" }
firebase-firestore = { group = "com.google.firebase", name = "firebase-firestore", version.ref = "firebaseFirestore" }  // ← tambahkan ini

Langkah 2 — Tambahkan dependency di build.gradle.kts (Module: app):

build.gradle.kts (Module: app) — tambahkan Firestore 
dependencies {
    implementation(libs.androidx.core.ktx)
    implementation(libs.androidx.appcompat)
    implementation(libs.material)
    implementation(libs.androidx.activity)
    implementation(libs.androidx.constraintlayout)
    implementation(libs.firebase.auth)
    implementation(libs.firebase.firestore)  // ← tambahkan ini
    testImplementation(libs.junit)
    androidTestImplementation(libs.androidx.junit)
    androidTestImplementation(libs.androidx.espresso.core)
}

MainActivity.kt — uji coba koneksi Firebase 
import android.graphics.Color
import android.os.Bundle
import android.util.Log
import android.widget.Button
import android.widget.TextView
import androidx.appcompat.app.AppCompatActivity
import com.google.firebase.Firebase
import com.google.firebase.auth.auth

class MainActivity : AppCompatActivity() {

    // Tag untuk Logcat
    private val TAG = "FirebaseTest"

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        val tvStatus      = findViewById<TextView>(R.id.tvStatus)
        val btnCekKoneksi = findViewById<Button>(R.id.btnCekKoneksi)

        btnCekKoneksi.setOnClickListener {
            cekKoneksiFirebase(tvStatus)
        }

        // Cek otomatis saat app pertama dibuka
        cekKoneksiFirebase(tvStatus)
    }

    private fun cekKoneksiFirebase(tvStatus: TextView) {
        try {
            // Coba inisialisasi Firebase Auth
            val auth = Firebase.auth

            // Jika berhasil, auth tidak akan null
            val statusUser = if (auth.currentUser != null) {
                "Login sebagai: ${auth.currentUser?.email}"
            } else {
                "Tidak ada user yang login"
            }

            // Tampilkan hasil di layar
            tvStatus.text = "Firebase terhubung!\n$statusUser"
            tvStatus.setTextColor(Color.parseColor("#2e7d32"))

            // Catat di Logcat
            Log.d(TAG, "Firebase Auth berhasil diinisialisasi")
            Log.d(TAG, "Status: $statusUser")

        } catch (e: Exception) {
            // Jika gagal, tampilkan pesan error
            tvStatus.text = "Koneksi gagal!\n${e.message}"
            tvStatus.setTextColor(Color.parseColor("#c62828"))

            Log.e(TAG, "Firebase gagal diinisialisasi", e)
        }
    }
}

Pengenalan Firebase Authentication

Firebase Authentication menyediakan layanan backend siap pakai untuk mengautentikasi pengguna. Mendukung berbagai metode login: email/password, Google, Facebook, GitHub, nomor telepon, dan lainnya.

Cara Kerja Firebase Auth

Saat user berhasil login, Firebase mengembalikan objek FirebaseUser yang berisi informasi user (uid, email, displayName, dll). UID ini unik per user dan digunakan sebagai identifier di database.

Sebelum bisa menggunakan login email/password, aktifkan dulu di Firebase Console:

Authentication → Sign-in method → Email/Password → Enable → Save

RegisterActivity.kt

RegisterActivity.kt 
import android.content.Intent
import android.os.Bundle
import android.view.View
import android.widget.Button
import android.widget.EditText
import android.widget.ProgressBar
import android.widget.Toast
import androidx.appcompat.app.AppCompatActivity
import com.google.firebase.Firebase
import com.google.firebase.auth.auth

class RegisterActivity : AppCompatActivity() {

    private val auth = Firebase.auth

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_register)

        val etEmail    = findViewById<EditText>(R.id.etEmail)
        val etPassword = findViewById<EditText>(R.id.etPassword)
        val btnRegister = findViewById<Button>(R.id.btnRegister)
        val progressBar = findViewById<ProgressBar>(R.id.progressBar)

        btnRegister.setOnClickListener {
            val email    = etEmail.text.toString().trim()
            val password = etPassword.text.toString().trim()

            // Validasi input
            if (email.isEmpty() || password.isEmpty()) {
                Toast.makeText(this, "Email dan password tidak boleh kosong", Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }
            if (password.length < 6) {
                Toast.makeText(this, "Password minimal 6 karakter", Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }

            // Tampilkan loading
            progressBar.visibility = View.VISIBLE
            btnRegister.isEnabled = false

            // Daftarkan user ke Firebase
            auth.createUserWithEmailAndPassword(email, password)
                .addOnCompleteListener(this) { task ->
                    progressBar.visibility = View.GONE
                    btnRegister.isEnabled = true

                    if (task.isSuccessful) {
                        // Registrasi berhasil
                        val user = auth.currentUser
                        Toast.makeText(this, "Registrasi berhasil! UID: ${user?.uid}", Toast.LENGTH_LONG).show()
                        // Arahkan ke halaman login atau home
                        startActivity(Intent(this, LoginActivity::class.java))
                        finish()
                    } else {
                        // Registrasi gagal — tampilkan pesan error
                        Toast.makeText(this, "Gagal: ${task.exception?.message}", Toast.LENGTH_LONG).show()
                    }
                }
        }
    }
}

import android.os.Bundle
import android.view.View
import android.widget.Button
import android.widget.EditText
import android.widget.ProgressBar
import android.widget.Toast
import androidx.appcompat.app.AppCompatActivity
import com.google.firebase.Firebase
import com.google.firebase.auth.auth

class LoginActivity : AppCompatActivity() {

    private val auth = Firebase.auth

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_login)

        val etEmail    = findViewById<EditText>(R.id.etEmail)
        val etPassword = findViewById<EditText>(R.id.etPassword)
        val btnLogin   = findViewById<Button>(R.id.btnLogin)
        val btnGoRegister = findViewById<Button>(R.id.btnGoRegister)
        val progressBar = findViewById<ProgressBar>(R.id.progressBar)

        btnGoRegister.setOnClickListener {
            startActivity(Intent(this, RegisterActivity::class.java))
        }

        btnLogin.setOnClickListener {
            val email    = etEmail.text.toString().trim()
            val password = etPassword.text.toString().trim()

            if (email.isEmpty() || password.isEmpty()) {
                Toast.makeText(this, "Isi email dan password terlebih dahulu", Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }

            progressBar.visibility = View.VISIBLE
            btnLogin.isEnabled = false

            auth.signInWithEmailAndPassword(email, password)
                .addOnCompleteListener(this) { task ->
                    progressBar.visibility = View.GONE
                    btnLogin.isEnabled = true

                    if (task.isSuccessful) {
                        val user = auth.currentUser
                        Toast.makeText(this, "Selamat datang, ${user?.email}", Toast.LENGTH_SHORT).show()
                        startActivity(Intent(this, HomeActivity::class.java))
                        finish()
                    } else {
                        Toast.makeText(this, "Login gagal: ${task.exception?.message}", Toast.LENGTH_LONG).show()
                    }
                }
        }
    }
}

Pengenalan Cloud Firestore

Cloud Firestore adalah database NoSQL berbasis dokumen dari Firebase. Data disimpan dalam bentuk koleksi (collection) yang berisi dokumen (document), mirip seperti folder berisi file JSON.

Struktur data Koleksi → Dokumen → Field Pohon JSON

Offline support Otomatis Otomatis

Firestore menggunakan hierarki Collection → Document → Field. Bayangkan Collection seperti tabel, Document seperti baris, dan Field seperti kolom — tapi jauh lebih fleksibel karena setiap dokumen bisa punya field yang berbeda.

Create — Menambah Data

Ada dua cara menambah dokumen: dengan ID otomatis (add()) atau dengan ID yang ditentukan sendiri (set()).

Cara 1 — ID Otomatis dengan add()

Tambah dokumen dengan ID otomatis 
val db = Firebase.firestore

// Data yang akan disimpan
val note = hashMapOf(
    "title"     to "Belajar Firebase",
    "content"   to "Firestore sangat mudah digunakan!",
    "userId"    to Firebase.auth.currentUser?.uid,
    "createdAt" to com.google.firebase.Timestamp.now()
)

// Tambah ke collection "notes" — ID dokumen dibuat otomatis
db.collection("notes")
    .add(note)
    .addOnSuccessListener { documentReference ->
        Log.d("Firestore", "Dokumen ditambahkan dengan ID: ${documentReference.id}")
        Toast.makeText(this, "Catatan berhasil disimpan!", Toast.LENGTH_SHORT).show()
    }
    .addOnFailureListener { e ->
        Log.e("Firestore", "Error menambah dokumen", e)
        Toast.makeText(this, "Gagal menyimpan: ${e.message}", Toast.LENGTH_SHORT).show()
    }

Cara 2 — ID Manual dengan set()

Tambah dokumen dengan ID yang ditentukan 
val db = Firebase.firestore
val userId = Firebase.auth.currentUser?.uid ?: return

// Simpan profil user dengan UID sebagai ID dokumen
val userProfile = hashMapOf(
    "name"      to "Budi Santoso",
    "email"     to "budi@example.com",
    "createdAt" to com.google.firebase.Timestamp.now()
)

// Gunakan UID sebagai ID dokumen agar mudah dicari
db.collection("users")
    .document(userId)   // ← ID ditentukan manual
    .set(userProfile)
    .addOnSuccessListener {
        Log.d("Firestore", "Profil user berhasil disimpan")
    }
    .addOnFailureListener { e ->
        Log.e("Firestore", "Gagal menyimpan profil", e)
    }

Read — Membaca Data

Ada dua cara membaca data dari Firestore: sekali baca (get()) dan realtime listener (addSnapshotListener()) yang otomatis update saat data berubah.

Baca Semua Dokumen dalam Collection

Baca semua dokumen — sekali baca 
val db = Firebase.firestore

db.collection("notes")
    .get()
    .addOnSuccessListener { result ->
        val noteList = mutableListOf<Note>()

        for (document in result) {
            // Konversi dokumen ke data class
            val note = document.toObject(Note::class.java).copy(id = document.id)
            noteList.add(note)
            Log.d("Firestore", "${document.id} => ${document.data}")
        }

        // Update RecyclerView atau UI
        adapter.submitList(noteList)
    }
    .addOnFailureListener { e ->
        Log.e("Firestore", "Error membaca data", e)
    }

Baca dengan Filter (Query)

Query — filter berdasarkan field 
val db = Firebase.firestore
val currentUserId = Firebase.auth.currentUser?.uid ?: return

// Ambil hanya catatan milik user yang sedang login
db.collection("notes")
    .whereEqualTo("userId", currentUserId)   // filter
    .orderBy("createdAt", Query.Direction.DESCENDING)  // urutkan terbaru
    .get()
    .addOnSuccessListener { result ->
        val noteList = result.documents.mapNotNull { doc ->
            doc.toObject(Note::class.java)?.copy(id = doc.id)
        }
        adapter.submitList(noteList)
    }

Realtime Listener — Data Auto Update

addSnapshotListener — update otomatis saat data berubah 
val db = Firebase.firestore
val currentUserId = Firebase.auth.currentUser?.uid ?: return

// Simpan listener agar bisa di-remove nanti
private var listenerRegistration: ListenerRegistration? = null

override fun onStart() {
    super.onStart()

    listenerRegistration = db.collection("notes")
        .whereEqualTo("userId", currentUserId)
        .orderBy("createdAt", Query.Direction.DESCENDING)
        .addSnapshotListener { snapshot, error ->
            if (error != null) {
                Log.e("Firestore", "Listen failed", error)
                return@addSnapshotListener
            }

            val noteList = snapshot?.documents?.mapNotNull { doc ->
                doc.toObject(Note::class.java)?.copy(id = doc.id)
            } ?: emptyList()

            adapter.submitList(noteList)
        }
}

override fun onStop() {
    super.onStop()
    listenerRegistration?.remove()  // Selalu remove listener saat tidak dipakai!
}

Jangan Lupa Remove Listener

Listener yang tidak di-remove akan terus aktif di background dan menghabiskan kuota Firestore. Selalu panggil listenerRegistration?.remove() di onStop() atau onDestroy().

NoteListActivity.kt — Daftar Catatan dengan Realtime Listener

NoteListActivity.kt 
import android.content.Intent
import android.os.Bundle
import android.widget.Toast
import androidx.appcompat.app.AppCompatActivity
import androidx.recyclerview.widget.LinearLayoutManager
import androidx.recyclerview.widget.RecyclerView
import com.google.android.material.floatingactionbutton.FloatingActionButton
import com.google.firebase.Firebase
import com.google.firebase.auth.auth
import com.google.firebase.firestore.ListenerRegistration
import com.google.firebase.firestore.Query
import com.google.firebase.firestore.firestore

class NoteListActivity : AppCompatActivity() {

    private val db   = Firebase.firestore
    private val auth = Firebase.auth
    private var listenerReg: ListenerRegistration? = null
    private lateinit var adapter: NoteAdapter

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_note_list)

        val recyclerView = findViewById<RecyclerView>(R.id.recyclerView)
        val fab = findViewById<FloatingActionButton>(R.id.fab)

        // Setup adapter dengan callback edit dan delete
        adapter = NoteAdapter(
            onEdit = { note ->
                val intent = Intent(this, AddEditNoteActivity::class.java)
                intent.putExtra("noteId", note.id)
                intent.putExtra("noteTitle", note.title)
                intent.putExtra("noteContent", note.content)
                startActivity(intent)
            },
            onDelete = { note -> deleteNote(note.id) }
        )

        recyclerView.layoutManager = LinearLayoutManager(this)
        recyclerView.adapter = adapter

        fab.setOnClickListener {
            startActivity(Intent(this, AddEditNoteActivity::class.java))
        }
    }

    override fun onStart() {
        super.onStart()
        val userId = auth.currentUser?.uid ?: return

        // Realtime listener — list otomatis update
        listenerReg = db.collection("notes")
            .whereEqualTo("userId", userId)
            .orderBy("createdAt", Query.Direction.DESCENDING)
            .addSnapshotListener { snapshot, error ->
                if (error != null) {
                    Toast.makeText(this, "Error: ${error.message}", Toast.LENGTH_SHORT).show()
                    return@addSnapshotListener
                }
                val notes = snapshot?.documents?.mapNotNull { doc ->
                    doc.toObject(Note::class.java)?.copy(id = doc.id)
                } ?: emptyList()
                adapter.submitList(notes)
            }
    }

    override fun onStop() {
        super.onStop()
        listenerReg?.remove()
    }

    private fun deleteNote(noteId: String) {
        db.collection("notes").document(noteId)
            .delete()
            .addOnSuccessListener {
                Toast.makeText(this, "Catatan dihapus", Toast.LENGTH_SHORT).show()
            }
    }
}

AddEditNoteActivity.kt — Tambah & Edit Catatan

AddEditNoteActivity.kt 
import android.os.Bundle
import android.widget.Button
import android.widget.EditText
import android.widget.Toast
import androidx.appcompat.app.AppCompatActivity
import com.google.firebase.Firebase
import com.google.firebase.Timestamp
import com.google.firebase.auth.auth
import com.google.firebase.firestore.firestore

class AddEditNoteActivity : AppCompatActivity() {

    private val db   = Firebase.firestore
    private val auth = Firebase.auth

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_add_edit_note)

        val etTitle   = findViewById<EditText>(R.id.etTitle)
        val etContent = findViewById<EditText>(R.id.etContent)
        val btnSave   = findViewById<Button>(R.id.btnSave)

        // Cek apakah mode edit (ada noteId dari Intent)
        val noteId      = intent.getStringExtra("noteId")
        val isEditMode  = noteId != null

        if (isEditMode) {
            etTitle.setText(intent.getStringExtra("noteTitle"))
            etContent.setText(intent.getStringExtra("noteContent"))
            btnSave.text = "Simpan Perubahan"
        }

        btnSave.setOnClickListener {
            val title   = etTitle.text.toString().trim()
            val content = etContent.text.toString().trim()

            if (title.isEmpty()) {
                etTitle.error = "Judul tidak boleh kosong"
                return@setOnClickListener
            }

            if (isEditMode) {
                // Mode edit — update dokumen yang ada
                db.collection("notes").document(noteId!!)
                    .update("title", title, "content", content)
                    .addOnSuccessListener {
                        Toast.makeText(this, "Catatan diperbarui", Toast.LENGTH_SHORT).show()
                        finish()
                    }
            } else {
                // Mode tambah — buat dokumen baru
                val note = Note(
                    title     = title,
                    content   = content,
                    userId    = auth.currentUser?.uid ?: "",
                    createdAt = Timestamp.now()
                )
                db.collection("notes").add(note)
                    .addOnSuccessListener {
                        Toast.makeText(this, "Catatan disimpan", Toast.LENGTH_SHORT).show()
                        finish()
                    }
                    .addOnFailureListener { e ->
                        Toast.makeText(this, "Gagal: ${e.message}", Toast.LENGTH_SHORT).show()
                    }
            }
        }
    }
}

Firebase Realtime Database

Pengenalan Realtime Database

Firebase Realtime Database adalah database NoSQL berbasis cloud yang menyimpan data dalam format pohon JSON. Keunggulan utamanya adalah sinkronisasi data secara real-time ke semua client yang terhubung dalam hitungan milidetik.

Aspek Realtime Database Cloud Firestore
Struktur data Pohon JSON flat Collection → Document
Cocok untuk Chat, live score, status online Data terstruktur kompleks
Query Terbatas (satu field) Lebih powerful
Harga Berdasarkan bandwidth & storage Berdasarkan jumlah operasi

Menulis Data ke Realtime Database

Ada tiga cara menulis data: setValue() untuk menulis/menimpa node, push() untuk menambah data dengan key unik otomatis, dan updateChildren() untuk update beberapa field sekaligus.

Membaca Data & Listener

Sama seperti Firestore, Realtime Database mendukung dua cara baca: sekali baca (get()) dan listener real-time (addValueEventListener()).

get() — Baca Sekali

get() — baca data satu kali 
val database = Firebase.database

    .get()
    .addOnSuccessListener { snapshot ->
        if (snapshot.exists()) {
            val name   = snapshot.child("name").getValue(String::class.java)
            val online = snapshot.child("online").getValue(Boolean::class.java)
            Log.d("RTDB", "Nama: $name, Online: $online")
        } else {
            Log.d("RTDB", "Data tidak ditemukan")
        }
    }
    .addOnFailureListener { e ->
        Log.e("RTDB", "Gagal membaca: ${e.message}")
    }

addValueEventListener() — Listener Real-time

Listener ini dipanggil pertama kali saat data dibaca, lalu dipanggil lagi setiap kali data berubah. Cocok untuk fitur chat atau status online.

addValueEventListener() — update otomatis saat data berubah 
import com.google.firebase.database.DataSnapshot
import com.google.firebase.database.DatabaseError
import com.google.firebase.database.ValueEventListener

val database    = Firebase.database
val messagesRef = database.getReference("messages")

// Simpan referensi listener untuk di-remove nanti
private lateinit var messageListener: ValueEventListener

override fun onStart() {
    super.onStart()

    messageListener = object : ValueEventListener {
        override fun onDataChange(snapshot: DataSnapshot) {
            // Dipanggil setiap kali data berubah
            val messageList = mutableListOf<Message>()

            for (child in snapshot.children) {
                val message = child.getValue(Message::class.java)
                message?.let { messageList.add(it) }
            }

            // Update RecyclerView
            adapter.submitList(messageList)
            Log.d("RTDB", "Total pesan: ${messageList.size}")
        }

        override fun onCancelled(error: DatabaseError) {
            Log.e("RTDB", "Listener dibatalkan: ${error.message}")
        }
    }

    messagesRef.addValueEventListener(messageListener)
}

override fun onStop() {
    super.onStop()
    // Selalu remove listener saat Activity tidak aktif
    messagesRef.removeEventListener(messageListener)
}

addChildEventListener() — Dengarkan Perubahan Per Item

Lebih efisien dari ValueEventListener untuk list besar karena hanya dipanggil saat ada item yang ditambah, diubah, atau dihapus — bukan seluruh list.

addChildEventListener() — efisien untuk list 
import com.google.firebase.database.ChildEventListener

messagesRef.addChildEventListener(object : ChildEventListener {
    override fun onChildAdded(snapshot: DataSnapshot, previousChildName: String?) {
        // Dipanggil saat ada pesan baru ditambahkan
        val message = snapshot.getValue(Message::class.java)
        message?.let { adapter.addItem(it) }
    }

    override fun onChildChanged(snapshot: DataSnapshot, previousChildName: String?) {
        // Dipanggil saat pesan diubah
        val message = snapshot.getValue(Message::class.java)
        message?.let { adapter.updateItem(it) }
    }

    override fun onChildRemoved(snapshot: DataSnapshot) {
        // Dipanggil saat pesan dihapus
        val key = snapshot.key
        key?.let { adapter.removeItem(it) }
    }

    override fun onChildMoved(snapshot: DataSnapshot, previousChildName: String?) { }
    override fun onCancelled(error: DatabaseError) {
        Log.e("RTDB", "Error: ${error.message}")
    }
})

import android.os.Bundle
import android.widget.Button
import android.widget.EditText
import androidx.appcompat.app.AppCompatActivity
import androidx.recyclerview.widget.LinearLayoutManager
import androidx.recyclerview.widget.RecyclerView
import com.google.firebase.Firebase
import com.google.firebase.auth.auth
import com.google.firebase.database.DataSnapshot
import com.google.firebase.database.DatabaseError
import com.google.firebase.database.ValueEventListener
import com.google.firebase.database.database

class ChatActivity : AppCompatActivity() {

    private val auth        = Firebase.auth
    private val database    = Firebase.database
    private val messagesRef = database.getReference("messages")

    private lateinit var adapter      : ChatAdapter
    private lateinit var msgListener  : ValueEventListener

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_chat)

        val recyclerView = findViewById<RecyclerView>(R.id.recyclerView)
        val etMessage    = findViewById<EditText>(R.id.etMessage)
        val btnSend      = findViewById<Button>(R.id.btnSend)

        val currentUserId = auth.currentUser?.uid ?: ""

        adapter = ChatAdapter(currentUserId)
        recyclerView.layoutManager = LinearLayoutManager(this).apply {
            stackFromEnd = true   // scroll ke bawah otomatis
        }
        recyclerView.adapter = adapter

        btnSend.setOnClickListener {
            val text = etMessage.text.toString().trim()
            if (text.isEmpty()) return@setOnClickListener

            val newRef = messagesRef.push()
            val message = Message(
                key        = newRef.key ?: "",
                text       = text,
                senderName = auth.currentUser?.email ?: "Anonymous",
                senderId   = currentUserId,
                timestamp  = System.currentTimeMillis()
            )
            newRef.setValue(message)
            etMessage.text.clear()
        }
    }

    override fun onStart() {
        super.onStart()

        msgListener = object : ValueEventListener {
            override fun onDataChange(snapshot: DataSnapshot) {
                val messages = mutableListOf<Message>()
                for (child in snapshot.children) {
                    val msg = child.getValue(Message::class.java)
                        ?.copy(key = child.key ?: "")
                    msg?.let { messages.add(it) }
                }
                adapter.submitList(messages)
                // Scroll ke pesan terbaru
                if (messages.isNotEmpty()) {
                    findViewById<RecyclerView>(R.id.recyclerView)
                        .scrollToPosition(messages.size - 1)
                }
            }

            override fun onCancelled(error: DatabaseError) { }
        }

        // Ambil 50 pesan terakhir saja
        messagesRef.limitToLast(50)
            .addValueEventListener(msgListener)
    }

    override fun onStop() {
        super.onStop()
        messagesRef.removeEventListener(msgListener)
    }
}

uploadImage() — upload dengan progress tracking 
import com.google.firebase.Firebase
import com.google.firebase.auth.auth
import com.google.firebase.storage.storage

private fun uploadImage(imageUri: Uri) {
    val userId     = Firebase.auth.currentUser?.uid ?: return
    val storage    = Firebase.storage
    val storageRef = storage.reference

    // Buat path unik untuk file — gunakan timestamp agar tidak bentrok
    val fileName  = "profile_${System.currentTimeMillis()}.jpg"
    val imageRef  = storageRef.child("images/profile_photos/$userId/$fileName")

    val progressBar = findViewById<ProgressBar>(R.id.progressBar)
    val btnUpload   = findViewById<Button>(R.id.btnUpload)

    progressBar.visibility = View.VISIBLE
    btnUpload.isEnabled    = false

    // Mulai upload
    val uploadTask = imageRef.putFile(imageUri)

    // Pantau progress upload
    uploadTask.addOnProgressListener { taskSnapshot ->
        val progress = (100.0 * taskSnapshot.bytesTransferred / taskSnapshot.totalByteCount).toInt()
        progressBar.progress = progress
        Log.d("Storage", "Upload progress: $progress%")
    }

    // Selesai upload
    uploadTask.addOnSuccessListener {
        progressBar.visibility = View.GONE
        btnUpload.isEnabled    = true
        Toast.makeText(this, "Gambar berhasil diupload!", Toast.LENGTH_SHORT).show()

        // Ambil download URL setelah upload selesai
        imageRef.downloadUrl.addOnSuccessListener { downloadUri ->
            Log.d("Storage", "Download URL: $downloadUri")
            // Simpan URL ke Firestore/RTDB untuk ditampilkan nanti
            saveImageUrlToFirestore(downloadUri.toString())
        }
    }

    uploadTask.addOnFailureListener { e ->
        progressBar.visibility = View.GONE
        btnUpload.isEnabled    = true
        Toast.makeText(this, "Upload gagal: ${e.message}", Toast.LENGTH_LONG).show()
        Log.e("Storage", "Upload error", e)
    }
}

Langkah 3 — Buat CloudinaryHelper.kt

Buat helper class untuk menangani upload ke Cloudinary. Simpan kredensial di sini — untuk production sebaiknya disimpan di server, tapi untuk praktikum ini cukup di client.

CloudinaryHelper.kt 
import android.content.Context
import android.net.Uri
import okhttp3.*
import okhttp3.MediaType.Companion.toMediaTypeOrNull
import okhttp3.RequestBody.Companion.toRequestBody
import org.json.JSONObject
import java.io.IOException

object CloudinaryHelper {

    // Ganti dengan kredensial Cloudinary Anda
    private const val CLOUD_NAME   = "your_cloud_name"
    private const val UPLOAD_PRESET = "your_upload_preset"  // unsigned preset

    // URL endpoint upload Cloudinary
    private val UPLOAD_URL = "https://api.cloudinary.com/v1_1/$CLOUD_NAME/image/upload"

    /**
     * Upload gambar ke Cloudinary
     * @param context  Context Android
     * @param imageUri Uri gambar yang dipilih dari galeri
     * @param onSuccess Callback dengan URL gambar jika berhasil
     * @param onError   Callback dengan pesan error jika gagal
     */
    fun uploadImage(
        context: Context,
        imageUri: Uri,
        onSuccess: (imageUrl: String) -> Unit,
        onError: (message: String) -> Unit
    ) {
        // Baca bytes dari Uri
        val inputStream = context.contentResolver.openInputStream(imageUri)
            ?: return onError("Tidak bisa membaca file gambar")
        val imageBytes = inputStream.readBytes()
        inputStream.close()

        // Buat multipart request body
        val requestBody = MultipartBody.Builder()
            .setType(MultipartBody.FORM)
            .addFormDataPart(
                "file",
                "upload_${System.currentTimeMillis()}.jpg",
                imageBytes.toRequestBody("image/jpeg".toMediaTypeOrNull())
            )
            .addFormDataPart("upload_preset", UPLOAD_PRESET)
            .build()

        val request = Request.Builder()
            .url(UPLOAD_URL)
            .post(requestBody)
            .build()

        // Jalankan di background thread
        OkHttpClient().newCall(request).enqueue(object : Callback {
            override fun onFailure(call: Call, e: IOException) {
                onError("Upload gagal: ${e.message}")
            }

            override fun onResponse(call: Call, response: Response) {
                val body = response.body?.string()
                if (response.isSuccessful && body != null) {
                    // Parse URL dari response JSON
                    val json = JSONObject(body)
                    val secureUrl = json.getString("secure_url")
                    onSuccess(secureUrl)
                } else {
                    onError("Server error: ${response.code}")
                }
            }
        })
    }
}

Langkah 4 — Implementasi di Activity

ProfileActivity.kt — upload foto profil via Cloudinary 
import android.net.Uri
import android.os.Bundle
import android.view.View
import android.widget.Button
import android.widget.ImageView
import android.widget.ProgressBar
import android.widget.Toast
import androidx.activity.result.contract.ActivityResultContracts
import androidx.appcompat.app.AppCompatActivity
import com.bumptech.glide.Glide
import com.google.firebase.Firebase
import com.google.firebase.auth.auth
import com.google.firebase.firestore.firestore

class ProfileActivity : AppCompatActivity() {

    private val auth = Firebase.auth
    private val db   = Firebase.firestore
    private var selectedImageUri: Uri? = null

    private val pickImageLauncher = registerForActivityResult(
        ActivityResultContracts.GetContent()
    ) { uri: Uri? ->
        if (uri != null) {
            selectedImageUri = uri
            // Tampilkan preview sebelum upload
            findViewById<ImageView>(R.id.ivProfile).setImageURI(uri)
        }
    }

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_profile)

        val ivProfile   = findViewById<ImageView>(R.id.ivProfile)
        val btnPick     = findViewById<Button>(R.id.btnPickImage)
        val btnUpload   = findViewById<Button>(R.id.btnUpload)
        val progressBar = findViewById<ProgressBar>(R.id.progressBar)

        // Tampilkan foto profil yang sudah ada (jika ada)
        val userId = auth.currentUser?.uid ?: return
        db.collection("users").document(userId).get()
            .addOnSuccessListener { doc ->
                val photoUrl = doc.getString("photoUrl")
                if (!photoUrl.isNullOrEmpty()) {
                    Glide.with(this).load(photoUrl).circleCrop().into(ivProfile)
                }
            }

        btnPick.setOnClickListener {
            pickImageLauncher.launch("image/*")
        }

        btnUpload.setOnClickListener {
            val uri = selectedImageUri
            if (uri == null) {
                Toast.makeText(this, "Pilih gambar terlebih dahulu", Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }

            // Tampilkan loading
            progressBar.visibility = View.VISIBLE
            btnUpload.isEnabled    = false

            // Upload ke Cloudinary
            CloudinaryHelper.uploadImage(
                context   = this,
                imageUri  = uri,
                onSuccess = { imageUrl ->
                    // Kembali ke main thread untuk update UI
                    runOnUiThread {
                        progressBar.visibility = View.GONE
                        btnUpload.isEnabled    = true

                        // Simpan URL ke Firestore
                        db.collection("users").document(userId)
                            .update("photoUrl", imageUrl)
                            .addOnSuccessListener {
                                Toast.makeText(this, "Foto profil berhasil diperbarui!", Toast.LENGTH_SHORT).show()
                                // Tampilkan gambar baru
                                Glide.with(this).load(imageUrl).circleCrop().into(ivProfile)
                            }
                    }
                },
                onError = { message ->
                    runOnUiThread {
                        progressBar.visibility = View.GONE
                        btnUpload.isEnabled    = true
                        Toast.makeText(this, "Gagal upload: $message", Toast.LENGTH_LONG).show()
                    }
                }
            )
        }
    }
}
