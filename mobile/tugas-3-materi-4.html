Praktikum: Pengenalan Android Development

Tujuan Pembelajaran

  • Memahami lingkungan pengembangan Android
  • Mampu membuat aplikasi Hello World
  • Memahami struktur UI XML dan berbagai jenis Layout
  • Mengenal dan menggunakan Basic Widgets
  • Mampu membuat Style dan Theme
  • Memahami konsep Localization
  • Memahami konsep Activity dan Lifecycle
  • Mampu mengimplementasikan Intent untuk navigasi

Tugas Praktikum

Ketentuan Pengerjaan & Pengumpulan

  • Kerjakan salah satu saja dari soal yang tersedia (Tugas 1 atau Tugas 2)
  • Silakan membuat aplikasi sebaik mungkin — kreativitas dan kerapian tampilan akan menjadi nilai tambah
  • Kumpulkan ke repository GitHub dengan nama repo: T3-mobile
  • Link GitHub dikumpulkan melalui eLearning
  • Pastikan repository bersifat public dan project bisa di-build tanpa error
  • Sertakan file README.md berisi nama, NIM, deskripsi aplikasi, dan screenshot

Tugas 1: Aplikasi Form Biodata dengan Style & Localization

Objektif: Menguasai penggunaan Style, Theme, dan Localization dalam membangun UI Android yang konsisten dan mendukung multi bahasa.

  1. Buat MainActivity berisi form input:
    • Nama lengkap (EditText)
    • NIM (EditText, inputType number)
    • Program Studi (Spinner)
    • Jenis Kelamin (RadioButton)
    • Hobi (CheckBox — minimal 3 pilihan)
  2. Buat ProfileActivity yang menampilkan data dari form menggunakan CardView
  3. Navigasi antar Activity menggunakan Explicit Intent
  4. Validasi input — semua field wajib diisi sebelum berpindah halaman
  5. Terapkan Style pada komponen UI (tombol, input, teks) agar tampilan konsisten
  6. Implementasikan Localization minimal 2 bahasa (Indonesia & Inggris) — semua teks UI harus menggunakan string resource

Fokus Penilaian Tugas 1

  • Konsistensi Style di seluruh halaman
  • Kelengkapan file strings.xml untuk setiap bahasa
  • Tidak ada hardcoded string di layout XML maupun kode Kotlin

Tugas 2: Aplikasi Registrasi dengan Layout & Intent

Objektif: Menguasai penggunaan Layout XML, Basic Widgets, dan Intent untuk membangun alur navigasi antar Activity.

  1. Buat RegistrasiActivity berisi form input:
    • Nama lengkap (EditText)
    • NIM (EditText, inputType number)
    • Program Studi (Spinner)
    • Jenis Kelamin (RadioButton)
    • Hobi (CheckBox — minimal 3 pilihan)
  2. Buat ProfilActivity yang menampilkan data registrasi
  3. Navigasi menggunakan Explicit Intent dengan data dikirim via putExtra()
  4. Validasi input — nama dan NIM tidak boleh kosong
  5. Layout wajib menggunakan minimal 2 jenis layout berbeda (contoh: ConstraintLayout untuk halaman profil, LinearLayout untuk form)
  6. Tambahkan minimal 1 Implicit Intent di halaman profil — contoh: tombol "Bagikan Profil" (ACTION_SEND), atau tombol "Hubungi Dosen" yang membuka dialer/email

Fokus Penilaian Tugas 2

  • Penggunaan minimal 2 jenis layout yang tepat dan sesuai konteks
  • Data berhasil dikirim dan ditampilkan dengan benar di ProfilActivity
  • Implicit Intent berfungsi dan tidak crash jika app target tidak tersedia

Catatan Penting

  • Plagiasi akan mendapat nilai 0
  • Pahami setiap baris kode yang Anda tulis
  • Jangan ragu bertanya jika ada kesulitan
