Setup Android Studio

Setelah menguasai dasar-dasar Kotlin, sekarang kita akan beralih ke Android Studio untuk pengembangan aplikasi mobile.

Kebutuhan Sistem

Komponen Minimum Rekomendasi
OS Windows 8 64-bit / macOS 10.14 / Linux Windows 10/11 64-bit / macOS 12+
RAM 8 GB 16 GB atau lebih
Storage 8 GB (IDE + SDK) SSD 50 GB+
Resolusi Layar 1280 x 800 1920 x 1080 atau lebih

Langkah Instalasi Android Studio

1

Download Android Studio

Kunjungi website resmi:

https://developer.android.com/studio

Klik tombol "Download Android Studio" dan pilih versi sesuai OS Anda.

2

Install Android Studio

Windows:

  • Jalankan file .exe yang sudah didownload
  • Ikuti wizard instalasi
  • Pilih "Standard" installation
  • Tunggu hingga selesai download komponen

macOS:

  • Buka file .dmg
  • Drag Android Studio ke folder Applications
  • Buka Android Studio dari Applications
3

Setup Wizard

Saat pertama kali membuka Android Studio:

  • Pilih "Do not import settings" jika instalasi baru
  • Pilih "Standard" setup type
  • Pilih UI theme (Light atau Dark)
  • Verify SDK components yang akan diinstall
  • Accept license agreements
  • Tunggu download dan instalasi komponen selesai
4

Verifikasi Instalasi

Pastikan komponen berikut terinstall:

  • Android SDK
  • Android SDK Platform
  • Android SDK Build-Tools
  • Android Emulator
  • Intel HAXM (untuk percepatan emulator)
Perhatian

Pastikan koneksi internet stabil saat instalasi. Android Studio akan mengunduh komponen SDK yang cukup besar (sekitar 2-5 GB).

Setup Emulator atau Device Fisik

Option A: Android Emulator (AVD)

A

Membuat Virtual Device

  • Buka Tools - Device Manager
  • Klik "Create Device"
  • Pilih device (contoh: Pixel 6)
  • Pilih system image (contoh: API 34)
  • Klik Finish
  • Klik tombol Play untuk menjalankan emulator

Option B: Device Fisik (HP Android)

B

Mengaktifkan Developer Options

  • Buka Settings - About Phone
  • Tap "Build Number" 7 kali
  • Kembali ke Settings, akan muncul "Developer Options"
  • Aktifkan "USB Debugging"
  • Hubungkan HP ke komputer dengan kabel USB
  • Allow USB debugging di popup yang muncul

Tambahkan di file app/build.gradle.kts:

    }
}

Intent

Intent digunakan untuk berkomunikasi antar komponen Android, seperti membuka Activity baru atau mengirim data.

IntentExample.kt
Kotlin 
// === EXPLICIT INTENT ===
// Membuka Activity tertentu dalam aplikasi yang sama

// Di MainActivity.kt
class MainActivity : AppCompatActivity() {
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        binding = ActivityMainBinding.inflate(layoutInflater)
        setContentView(binding.root)
        
        binding.btnOpenSecond.setOnClickListener {
            // Membuat intent untuk membuka SecondActivity
            val intent = Intent(this, SecondActivity::class.java)
            
            // Mengirim data dengan putExtra
            intent.putExtra("KEY_NAME", "John Doe")
            intent.putExtra("KEY_AGE", 25)
            intent.putExtra("KEY_SCORE", 95.5)
            
            // Memulai activity
            startActivity(intent)
        }
    }
}

// Di SecondActivity.kt
class SecondActivity : AppCompatActivity() {
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        binding = ActivitySecondBinding.inflate(layoutInflater)
        setContentView(binding.root)
        
        // Menerima data dari intent
        val name = intent.getStringExtra("KEY_NAME") ?: "Unknown"
        val age = intent.getIntExtra("KEY_AGE", 0)
        val score = intent.getDoubleExtra("KEY_SCORE", 0.0)
        
        binding.tvInfo.text = "Name: $name\nAge: $age\nScore: $score"
        
        // Tombol kembali
        binding.btnBack.setOnClickListener {
            finish()  // Menutup activity ini
        }
    }
}

// === IMPLICIT INTENT ===
// Membuka aplikasi lain atau fitur sistem

// Membuka browser
val browserIntent = Intent(Intent.ACTION_VIEW, Uri.parse("https://www.google.com"))
startActivity(browserIntent)

// Membuka dialer
val dialIntent = Intent(Intent.ACTION_DIAL, Uri.parse("tel:08123456789"))
startActivity(dialIntent)

// Mengirim email
val emailIntent = Intent(Intent.ACTION_SENDTO).apply {
    data = Uri.parse("mailto:")
    putExtra(Intent.EXTRA_EMAIL, arrayOf("test@example.com"))
    putExtra(Intent.EXTRA_SUBJECT, "Hello")
    putExtra(Intent.EXTRA_TEXT, "This is email body")
}
startActivity(emailIntent)

// Share text
val shareIntent = Intent(Intent.ACTION_SEND).apply {
    type = "text/plain"
    putExtra(Intent.EXTRA_TEXT, "Check out this awesome app!")
}
startActivity(Intent.createChooser(shareIntent, "Share via"))

Langkah 4: Tulis Kode Kotlin

5

Edit MainActivity.kt

Buka file java/com.example.helloandroid/MainActivity.kt

MainActivity.kt
Kotlin 
package com.example.helloandroid

import android.os.Bundle
import android.widget.Toast
import androidx.appcompat.app.AppCompatActivity
import com.example.helloandroid.databinding.ActivityMainBinding
import java.util.Calendar

class MainActivity : AppCompatActivity() {
    
    // Deklarasi view binding
    private lateinit var binding: ActivityMainBinding
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // Inflate binding
        binding = ActivityMainBinding.inflate(layoutInflater)
        setContentView(binding.root)
        
        // Setup click listener untuk button
        setupClickListeners()
    }
    
    private fun setupClickListeners() {
        binding.btnGreet.setOnClickListener {
            val name = binding.etName.text.toString().trim()
            
            if (name.isEmpty()) {
                // Tampilkan pesan error
                Toast.makeText(
                    this,
                    "Nama tidak boleh kosong!",
                    Toast.LENGTH_SHORT
                ).show()
                return@setOnClickListener
            }
            
            // Buat greeting berdasarkan waktu
            val greeting = getGreetingByTime(name)
            
            // Tampilkan hasil
            binding.tvResult.text = greeting
            
            // Clear input
            binding.etName.text.clear()
        }
    }
    
    private fun getGreetingByTime(name: String): String {
        val hour = Calendar.getInstance().get(Calendar.HOUR_OF_DAY)
        
        val timeGreeting = when (hour) {
            in 0..10 -> "Selamat Pagi"
            in 11..14 -> "Selamat Siang"
            in 15..17 -> "Selamat Sore"
            else -> "Selamat Malam"
        }
        
        return "$timeGreeting, $name!"
    }
}

Latihan

Berikut adalah latihan-latihan untuk mengasah kemampuan Kotlin dan Android development Anda.

Latihan Kotlin Dasar

Latihan 1: Kalkulator

Buat program kalkulator console yang:

  • Meminta input 2 angka dari user
  • Menampilkan menu pilihan operasi (+, -, *, /, %)
  • Menghitung dan menampilkan hasil
  • Handle pembagian dengan nol
  • Tanya apakah ingin menghitung lagi
Latihan 2: Manajemen Mahasiswa

Buat program untuk mengelola data mahasiswa:

  • Data class Mahasiswa dengan property: nim, nama, jurusan, ipk
  • Fungsi untuk menambah mahasiswa ke list
  • Fungsi untuk mencari mahasiswa berdasarkan nim
  • Fungsi untuk filter mahasiswa dengan IPK di atas 3.0
  • Fungsi untuk menghitung rata-rata IPK
  • Fungsi untuk mengurutkan berdasarkan IPK
Latihan 3: Sistem Pemesanan

Buat program sistem pemesanan makanan:

  • Sealed class untuk MenuStatus (Available, SoldOut, ComingSoon)
  • Data class Menu dengan: id, nama, harga, status
  • Interface Orderable untuk item yang bisa dipesan
  • Class Order yang implement Orderable
  • Fungsi untuk menghitung total pesanan
  • Fungsi untuk menerapkan diskon dengan higher-order function

Latihan Android

Latihan 4: Aplikasi Kalkulator

Buat aplikasi kalkulator Android dengan:

  • Layout dengan 2 EditText untuk input angka
  • 4 tombol untuk operasi (+, -, x, /)
  • TextView untuk menampilkan hasil
  • Validasi input (tidak boleh kosong)

Latihan 5: Konversi Suhu

Buat aplikasi konversi suhu dengan:

  • EditText untuk input suhu
  • Spinner/RadioButton untuk pilih satuan asal (C/F/K)
  • Spinner/RadioButton untuk pilih satuan tujuan
  • Button untuk konversi
  • TextView untuk menampilkan hasil dengan 2 desimal
Latihan 6: Form Biodata

Buat aplikasi form biodata dengan 2 Activity:

  • Activity 1 (Form): Input nama, umur, email, alamat
  • Tombol "Lihat Biodata" untuk pindah ke Activity 2
  • Kirim data menggunakan Intent extras
  • Activity 2 (Display): Tampilkan semua data yang dikirim
  • Tombol kembali ke Activity 1
Latihan 7: Quiz App

Buat aplikasi kuis pilihan ganda dengan:

  • Minimal 5 pertanyaan
  • 4 pilihan jawaban per pertanyaan (RadioButton)
  • Tombol Next untuk ke pertanyaan berikutnya
  • Tampilkan skor di akhir quiz
  • Tombol untuk restart quiz
  • Gunakan data class untuk menyimpan pertanyaan
Tips Mengerjakan Latihan
  • Kerjakan secara bertahap, mulai dari yang sederhana
  • Pastikan memahami setiap baris kode yang ditulis
  • Gunakan debugging untuk mencari error
  • Referensi dokumentasi resmi Kotlin dan Android
  • Jangan ragu bertanya jika menemui kesulitan
