Tugas: Aplikasi Form Identitas Mahasiswa

Deskripsi Tugas

Buatlah sebuah aplikasi Android sederhana yang menampilkan form identitas mahasiswa. Ketika tombol ditekan, semua data yang diisi akan ditampilkan di bawah tombol menggunakan TextView.

Spesifikasi Aplikasi

1. Layout

Gunakan minimal 2 jenis layout berbeda dalam satu halaman. Contoh kombinasi yang bisa digunakan:

Layout Luar Layout Dalam Kegunaan
LinearLayout (vertical) LinearLayout (horizontal) Menyusun tombol berdampingan
LinearLayout (vertical) ConstraintLayout Area hasil dengan posisi fleksibel
ConstraintLayout LinearLayout (horizontal) Menyusun RadioButton dalam satu baris

2. Widget yang Wajib Ada

Widget Fungsi Jumlah
EditText Input nama lengkap 1
RadioButton dalam RadioGroup Pilihan jenis kelamin (Laki-laki / Perempuan) 1 group, 2 pilihan
CheckBox Pilihan hobi (minimal 2 pilihan) minimal 2
Button Tombol "Tampilkan" untuk memproses input 1
TextView Menampilkan hasil input di bawah tombol 1

3. Logika Kotlin

  • Ketika tombol diklik, ambil nilai dari semua widget
  • Validasi: nama tidak boleh kosong dan jenis kelamin harus dipilih
  • Tampilkan hasil di TextView dengan format yang rapi
  • Jika validasi gagal, tampilkan pesan error menggunakan Toast atau EditText.error

4. Contoh Tampilan Hasil

Contoh output di TextView setelah tombol diklik:

Nama     : Budi Santoso
Kelamin  : Laki-laki
Hobi     : Membaca, Coding

Kriteria Penilaian

Kriteria Poin
Menggunakan minimal 2 jenis layout yang berbeda dan tepat 25
Semua widget wajib tersedia dan berfungsi 25
Hasil input tampil di TextView setelah tombol diklik 25
Validasi input berjalan (nama kosong / gender tidak dipilih) 15
Kerapian tampilan dan kode 10

Catatan

  • Tidak perlu berpindah Activity — cukup satu halaman
  • Fokus pada penerapan layout yang benar, bukan fitur yang banyak
  • Pahami setiap baris kode yang kamu tulis
