Aktivitas Belajar 1 HTML

Website dan HTML

Website dapat dibayangkan seperti majalah digital yang bisa dibuka dari mana saja. Browser berperan sebagai pembaca, server berperan sebagai rak penyimpanan, dan HTML berperan sebagai struktur halaman yang menentukan bagian judul, paragraf, gambar, tabel, dan form.

Tujuan Pembelajaran:
  • Mahasiswa mampu menjelaskan pengertian website dan cara kerja web.
  • Mahasiswa mampu menjelaskan peran HTML dalam pembuatan halaman web.
  • Mahasiswa mampu membuat struktur dokumen HTML5 sederhana.

A. Pengertian Website

Website adalah kumpulan halaman web yang saling terhubung dan dapat diakses melalui internet atau jaringan lokal. Setiap halaman biasanya dibuat dengan HTML, dihias dengan CSS, dan dibuat interaktif dengan JavaScript.

Komponen Peran Contoh
Browser Menampilkan halaman web kepada pengguna Google Chrome, Firefox, Safari
Server Menyimpan dan mengirim file website Apache, Nginx, Node.js server
URL Alamat untuk membuka halaman https://unram.ac.id

B. Cara Kerja Web Client-Server

Ketika pengguna mengetik alamat website, browser mengirim permintaan ke server. Server membalas dengan file HTML, CSS, JavaScript, gambar, atau data lain. Browser kemudian membaca file tersebut dan menampilkannya sebagai halaman web.

1. Pengguna membuka alamat website di browser
2. Browser mengirim request ke server
3. Server mengirim response berupa file HTML, CSS, JavaScript, dan aset
4. Browser merender file tersebut menjadi halaman web

C. Struktur Dasar HTML5

HTML menggunakan tag untuk menandai bagian halaman. Tag seperti <h1>, <p>, dan <img> memberi tahu browser jenis konten yang sedang ditampilkan.

Contoh Kode

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Halaman Pertama Saya</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <p>Ini adalah halaman web pertama saya.</p>
    <a href="https://unram.ac.id">Kunjungi Universitas Mataram</a>
</body>
</html>
Perhatian: Simpan file HTML dengan ekstensi .html, misalnya index.html. Jika disimpan sebagai .txt, browser tidak akan membacanya sebagai halaman web.

D. Tag HTML Dasar

Tag Fungsi Contoh
<h1> sampai <h6> Membuat judul dan subjudul <h1>Judul</h1>
<p> Membuat paragraf <p>Isi paragraf</p>
<a> Membuat link <a href="profil.html">Profil</a>
<img> Menampilkan gambar <img src="foto.jpg" alt="Foto">

Latihan

Buat file biodata.html yang berisi nama, NIM, program studi, satu paragraf deskripsi diri, dan satu link menuju website kampus.

Aktivitas Belajar 2 Dasar

CSS dan JavaScript

Jika HTML adalah kerangka rumah, maka CSS adalah cat, tata ruang, dan dekorasinya. JavaScript adalah sistem listrik dan saklar yang membuat rumah dapat merespons tindakan penghuni. Dalam web, ketiganya bekerja bersama: HTML menyusun konten, CSS mengatur tampilan, dan JavaScript mengatur interaksi.

Tujuan Pembelajaran:
  • Mahasiswa mampu menjelaskan fungsi CSS dan JavaScript dalam halaman web.
  • Mahasiswa mampu menggunakan selector, property, dan value pada CSS.
  • Mahasiswa mampu membuat interaksi sederhana menggunakan JavaScript dan DOM.

A. Dasar CSS

CSS bekerja dengan memilih elemen HTML menggunakan selector, lalu memberi aturan tampilan menggunakan property dan value.

selector {
    property: value;
}

Contoh CSS

body {
    font-family: Arial, sans-serif;
    background-color: #f5f5f5;
}

h1 {
    color: #2c3e50;
}

.kartu {
    background: white;
    padding: 20px;
    border-radius: 8px;
}

B. Box Model

Setiap elemen HTML dianggap sebagai kotak. Kotak ini memiliki isi, jarak dalam atau padding, garis tepi atau border, dan jarak luar atau margin.

Bagian Fungsi Contoh
Content Isi utama elemen Teks pada paragraf
Padding Jarak antara isi dan border padding: 20px;
Border Garis tepi elemen border: 1px solid #ddd;
Margin Jarak elemen dengan elemen lain margin-bottom: 16px;

C. Dasar JavaScript

JavaScript digunakan untuk menjalankan logika. Contohnya membaca input pengguna, mengubah isi halaman, menghitung nilai, atau menampilkan pesan.

let nama = "Siti";
let nilai = 85;

function cekKelulusan(nilai) {
    if (nilai >= 60) {
        return "Lulus";
    }

    return "Tidak Lulus";
}

console.log(cekKelulusan(nilai));

D. DOM Manipulation

DOM atau Document Object Model adalah representasi halaman HTML di dalam JavaScript. Dengan DOM, JavaScript dapat memilih elemen lalu mengubah teks, atribut, atau tampilannya.

Contoh Kode Lengkap

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>CSS dan JavaScript</title>
</head>
<body>
    <h1 id="judul">Belajar Web</h1>
    <button onclick="ubahJudul()">Ubah Judul</button>

    <script>
        function ubahJudul() {
            document.getElementById('judul').textContent = 'JavaScript Berhasil!';
        }
    </script>
</body>
</html>
Tips: Pelajari HTML, CSS, dan JavaScript sebagai satu kesatuan. HTML tanpa CSS masih bisa tampil, tetapi belum rapi. HTML dan CSS tanpa JavaScript belum interaktif.

Latihan

Buat halaman profil sederhana. Gunakan CSS untuk mengatur warna judul dan kotak profil. Tambahkan tombol JavaScript untuk mengganti teks sapaan pada halaman.

Aktivitas Belajar 4 Praktikum

Praktikum Unit 1: Implementasi CSS dan JavaScript

Pada tahap ini, halaman HTML yang sudah dibuat akan diberi tampilan dan interaksi. CSS membuat halaman lebih mudah dibaca, sedangkan JavaScript membuat halaman merespons tindakan pengguna.

Tujuan Pembelajaran:
  • Mahasiswa mampu menghubungkan file CSS eksternal ke HTML.
  • Mahasiswa mampu menghubungkan file JavaScript eksternal ke HTML.
  • Mahasiswa mampu membuat event klik dan manipulasi DOM sederhana.

A. Tiga Cara Menulis CSS

Jenis CSS Lokasi Penulisan Kapan Dipakai
Inline CSS Di atribut style Hanya untuk percobaan kecil
Internal CSS Di dalam tag <style> Untuk satu halaman sederhana
External CSS Di file terpisah style.css Paling disarankan untuk proyek

B. Menghubungkan CSS dan JavaScript

CSS eksternal dihubungkan di dalam <head>. JavaScript eksternal umumnya diletakkan sebelum penutup </body> agar elemen HTML sudah terbaca terlebih dahulu.

<head>
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <script src="script.js"></script>
</body>

File: index.html

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Implementasi CSS dan JS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <main class="kartu">
        <h1 id="judul">Praktikum Unit 1</h1>
        <p id="pesan">Klik tombol untuk melihat pesan.</p>
        <button onclick="tampilkanPesan()">Tampilkan Pesan</button>
    </main>

    <script src="script.js"></script>
</body>
</html>

File: style.css

body {
    font-family: Arial, sans-serif;
    background-color: #f5f5f5;
}

.kartu {
    max-width: 600px;
    margin: 40px auto;
    padding: 24px;
    background: white;
    border: 1px solid #ddd;

File: script.js

Tips: Pisahkan HTML, CSS, dan JavaScript sejak awal agar proyek lebih rapi dan mudah diperbaiki.

Aktivitas Belajar 5 Vue.js

Vue.js: Menampilkan Data dan Conditional/Pengandaian

Vue dapat dibayangkan seperti papan informasi digital. Ketika data berubah, tampilan ikut diperbarui tanpa harus menulis manipulasi DOM satu per satu. Ini membuat kode lebih ringkas dan mudah dirawat.

Tujuan Pembelajaran:
  • Mahasiswa mampu menjalankan Vue 3 menggunakan CDN.
  • Mahasiswa mampu menampilkan data dengan template interpolation.
  • Mahasiswa mampu menggunakan v-bind, v-if, v-else, v-else-if, dan v-show.

A. Vue 3 via CDN

Untuk tahap awal, Vue cukup dipanggil melalui CDN. Mahasiswa tidak perlu memasang Vue CLI atau Vite. Cukup buat file HTML, tambahkan script Vue, lalu buat aplikasi dengan createApp.

Contoh Kode

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Vue Dasar</title>
</head>
<body>
    <div id="app">
        <h1>{{ judul }}</h1>
        <p>Nama mahasiswa: {{ mahasiswa.nama }}</p>
        <p>Program studi: {{ mahasiswa.prodi }}</p>
        <a :href="linkElearning">Buka eLearning</a>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
    <script>
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    judul: 'Belajar Vue 3',
                    mahasiswa: {
                        nama: 'Andi',
                        prodi: 'Informatika'
                    },
                    linkElearning: 'https://berajah-if.unram.ac.id/'
                };
            }
        }).mount('#app');
    </script>
</body>
</html>

B. Template Interpolation dan v-bind

Template interpolation menggunakan tanda {{ }} untuk menampilkan data sebagai teks. Jika data ingin dipasang pada atribut HTML seperti href, src, atau title, gunakan v-bind atau bentuk singkatnya :.

Kebutuhan Sintaks Contoh
Menampilkan teks {{ data }} {{ nama }}
Mengikat atribut v-bind:atribut v-bind:href="url"
Bentuk singkat bind :atribut :href="url"

C. Conditional Rendering

Conditional rendering digunakan untuk menampilkan atau menyembunyikan elemen berdasarkan kondisi. v-if benar-benar menambah atau menghapus elemen dari DOM, sedangkan v-show hanya mengatur tampil atau sembunyi melalui CSS.

<div id="app">
    <p v-if="nilai >= 85">Grade A</p>
    <p v-else-if="nilai >= 75">Grade B</p>
    <p v-else>Perlu belajar lagi</p>

    <p v-show="sudahLogin">Selamat datang, {{ nama }}</p>
</div>
Catatan: Gunakan v-if jika elemen jarang berubah. Gunakan v-show jika elemen sering ditampilkan dan disembunyikan.

Latihan

Buat halaman Vue yang menampilkan nama mahasiswa, nilai, dan status kelulusan. Gunakan v-if, v-else-if, dan v-else untuk menampilkan grade berdasarkan nilai.

Aktivitas Belajar 6 Vue.js

Data Binding dan Pengolahannya (1): Data Binding

Data binding dapat dibayangkan seperti cermin. Ketika data berubah, tampilan ikut berubah. Pada two-way binding, perubahan dari tampilan juga langsung memperbarui data. Vue menyediakan v-model untuk kebutuhan tersebut.

Tujuan Pembelajaran:
  • Mahasiswa mampu menjelaskan konsep reaktivitas Vue.
  • Mahasiswa mampu membedakan one-way binding dan two-way binding.
  • Mahasiswa mampu menggunakan v-model pada input form.

A. Reaktivitas Vue

Reaktivitas berarti Vue memantau data yang digunakan di tampilan. Saat data berubah, Vue memperbarui bagian tampilan yang terkait. Mahasiswa tidak perlu lagi memanggil document.getElementById untuk setiap perubahan kecil.

Jenis Binding Arah Data Contoh
One-way binding Data ke tampilan {{ nama }}, :href="url"
Two-way binding Data ke tampilan dan tampilan ke data v-model="nama"

B. Menggunakan v-model

v-model menghubungkan input form dengan data Vue. Ketika pengguna mengetik, nilai data langsung berubah. Ketika data diubah dari kode, isi input juga ikut berubah.

Contoh Kode Lengkap

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Data Binding Vue</title>
</head>
<body>
    <div id="app">
        <h1>Form Biodata Mahasiswa</h1>

        <label>Nama:
            <input v-model="nama" placeholder="Masukkan nama">
        </label>
        <br>

        <label>NIM:
            <input v-model="nim" placeholder="Masukkan NIM">
        </label>
        <br>

        <label>Semester:
            <input type="number" v-model.number="semester">
        </label>

        <h2>Preview</h2>
        <p>Nama: {{ nama }}</p>
        <p>NIM: {{ nim }}</p>
        <p>Semester: {{ semester }}</p>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
    <script>
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    nama: '',
                    nim: '',
                    semester: 1
                };
            }
        }).mount('#app');
    </script>
</body>
</html>

C. Modifier pada v-model

Vue menyediakan modifier untuk mengolah input sederhana. .number mengubah input menjadi angka, .trim menghapus spasi di awal dan akhir, sedangkan .lazy memperbarui data setelah input selesai berubah.

Modifier Fungsi Contoh
.number Mengubah input menjadi angka v-model.number="umur"
.trim Menghapus spasi awal dan akhir v-model.trim="nama"
.lazy Update setelah event change v-model.lazy="catatan"
Tips: Gunakan v-model untuk form, dan gunakan interpolation atau v-bind untuk menampilkan data.

Latihan

Buat form pendaftaran sederhana dengan input nama, email, program studi, dan semester. Tampilkan preview data di bawah form secara langsung menggunakan v-model.
