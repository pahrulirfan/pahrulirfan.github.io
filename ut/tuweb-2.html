Aktivitas Belajar 7 Vue.js

Data Binding dan Pengolahannya (2): Computed Properties, Methods

Bayangkan daftar nilai mahasiswa di spreadsheet. Ada kolom nilai tugas, UTS, dan UAS. Ketika satu nilai berubah, kolom nilai akhir ikut berubah otomatis. Di Vue, pola seperti ini cocok dibuat dengan computed properties, sedangkan aksi seperti menambah data, menghapus data, atau memproses tombol lebih cocok dibuat dengan methods.

Tujuan Pembelajaran:
  • Mahasiswa mampu menggunakan computed properties untuk menghasilkan data turunan.
  • Mahasiswa mampu menggunakan methods untuk menjalankan aksi atau proses tertentu.
  • Mahasiswa mampu membedakan kapan memakai computed dan kapan memakai methods.

A. Computed Properties

Computed properties adalah properti yang nilainya dihitung dari data lain. Vue akan menyimpan hasilnya sementara atau melakukan caching. Artinya, computed hanya dihitung ulang jika data yang dipakai di dalamnya berubah.

Aspek Computed Methods
Cara dipanggil di template {{ namaComputed }} {{ namaMethod() }}
Eksekusi ulang Hanya saat data sumber berubah Setiap kali dipanggil saat render
Cocok untuk Nilai turunan seperti total, rata-rata, status Aksi seperti klik tombol, tambah data, reset form

Contoh Kode: Kalkulator Nilai Akhir

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Computed dan Methods</title>
</head>
<body>
    <div id="app">
        <h1>Kalkulator Nilai Akhir</h1>

        <label>Tugas:
            <input type="number" v-model.number="tugas">
        </label>
        <br>

        <label>UTS:
            <input type="number" v-model.number="uts">
        </label>
        <br>

        <label>UAS:
            <input type="number" v-model.number="uas">
        </label>

        <p>Nilai Akhir: {{ nilaiAkhir }}</p>
        <p>Grade: {{ grade }}</p>
        <button @click="resetNilai">Reset</button>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
    <script>
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    tugas: 80,
                    uts: 75,
                    uas: 90
                };
            },
            computed: {
                nilaiAkhir() {
                    return (this.tugas * 0.3) + (this.uts * 0.3) + (this.uas * 0.4);
                },
                grade() {
                    if (this.nilaiAkhir >= 85) return 'A';
                    if (this.nilaiAkhir >= 75) return 'B';
                    if (this.nilaiAkhir >= 65) return 'C';
                    return 'D';
                }
            },
            methods: {
                resetNilai() {
                    this.tugas = 0;
                    this.uts = 0;
                    this.uas = 0;
                }
            }
        }).mount('#app');
    </script>
</body>
</html>
Tips: Jika nilai yang ditampilkan adalah hasil olahan dari data lain, gunakan computed. Jika yang dibutuhkan adalah aksi karena klik, submit, atau input pengguna, gunakan methods.

B. Contoh Kasus Konversi

Computed juga cocok untuk konversi satuan, misalnya mengubah menit ke jam atau rupiah ke format teks. Pengguna cukup mengubah satu input, lalu hasil turunannya ikut berubah.

computed: {
    jamKuliah() {
        return Math.floor(this.totalMenit / 60);
    },
    sisaMenit() {
        return this.totalMenit % 60;
    },
    durasiLengkap() {
        return `${this.jamKuliah} jam ${this.sisaMenit} menit`;
    }
}
Kesalahan umum: Jangan mengubah data utama dari dalam computed. Computed sebaiknya hanya menghitung dan mengembalikan nilai.

Latihan

Buat halaman Vue 3 untuk menghitung total pembayaran UKT sederhana. Input terdiri dari biaya UKT, potongan beasiswa, dan biaya administrasi. Tampilkan total bayar dengan computed, lalu buat tombol reset menggunakan methods.

Aktivitas Belajar 8 Vue.js

Watcher

Watcher dapat diibaratkan seperti petugas laboratorium yang mengamati perubahan suhu. Ketika suhu berubah melewati batas tertentu, petugas melakukan tindakan. Di Vue, watcher dipakai untuk mengamati perubahan data, lalu menjalankan proses lanjutan.

Tujuan Pembelajaran:
  • Mahasiswa mampu menjelaskan fungsi watcher dalam Vue.
  • Mahasiswa mampu menggunakan watch dan watchEffect.
  • Mahasiswa mampu membedakan kebutuhan watcher dan computed.

A. Konsep Watcher

Watcher digunakan ketika perubahan data harus memicu efek tertentu, misalnya validasi input, menyimpan data ke browser, memanggil API, atau menampilkan peringatan. Berbeda dengan computed yang menghasilkan nilai, watcher lebih tepat untuk menjalankan efek samping.

Kebutuhan Pilihan Alasan
Menghitung total nilai Computed Hasilnya ditampilkan sebagai data turunan
Menyimpan perubahan form Watcher Ada efek samping setelah data berubah
Mengambil data setelah keyword berubah Watcher Perubahan data memicu proses lain

Contoh Kode: Watch Input Pencarian

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Watcher Vue</title>
</head>
<body>
    <div id="app">
        <h1>Pencarian Mata Kuliah</h1>
        <input v-model="keyword" placeholder="Ketik nama mata kuliah">
        <p>Status: {{ status }}</p>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
    <script>
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    keyword: '',
                    status: 'Belum ada pencarian'
                };
            },
            watch: {
                keyword(newValue, oldValue) {
                    if (newValue.length === 0) {
                        this.status = 'Belum ada pencarian';
                    } else if (newValue.length < 3) {
                        this.status = 'Ketik minimal 3 huruf';
                    } else {
                        this.status = `Mencari data untuk: ${newValue}`;
                    }
                }
            }
        }).mount('#app');
    </script>
</body>
</html>

B. Opsi deep dan immediate

Jika data yang diamati berbentuk objek atau array, gunakan opsi deep agar perubahan di dalam objek ikut diamati. Gunakan immediate jika watcher perlu langsung berjalan saat halaman pertama kali dibuka.

watch: {
    mahasiswa: {
        handler(newValue) {
            console.log('Data mahasiswa berubah:', newValue);
        },
        deep: true,
        immediate: true
    }
}

C. watchEffect

watchEffect biasanya dipakai pada Composition API. Ia otomatis melacak data yang digunakan di dalam fungsinya. Untuk tahap awal, mahasiswa cukup memahami bahwa watch lebih eksplisit, sedangkan watchEffect lebih otomatis.

const { createApp, ref, watchEffect } = Vue;

createApp({
    setup() {
        const nama = ref('Budi');
        const prodi = ref('Informatika');

        watchEffect(() => {
            console.log(`Mahasiswa: ${nama.value}, Prodi: ${prodi.value}`);
        });

        return { nama, prodi };
    }
}).mount('#app');
Catatan: Jangan memakai watcher hanya untuk menghitung nilai yang langsung ditampilkan. Untuk kebutuhan itu, computed lebih sederhana dan lebih mudah dibaca.

Latihan

Buat form pendaftaran praktikum dengan input nama dan NIM. Gunakan watcher untuk menampilkan pesan validasi: NIM harus minimal 8 karakter dan nama tidak boleh kosong.

Praktikum Unit 2: Vue.js

Pada praktikum ini, mahasiswa membuat aplikasi kecil yang utuh: To-Do List. Aplikasi ini cocok untuk memahami alur Vue karena melibatkan input, array, event, computed, dan tampilan kondisional dalam satu halaman.

Tujuan Pembelajaran:
  • Mahasiswa mampu membuat aplikasi Vue 3 menggunakan CDN.
  • Mahasiswa mampu menambah, menghapus, dan mengubah status item pada array.
  • Mahasiswa mampu menghitung sisa tugas menggunakan computed.

A. Fitur Praktikum

Fitur Konsep Vue Keterangan
Tambah tugas v-model, @submit.prevent Mengambil teks dari input lalu menambahkannya ke array
Tandai selesai v-model pada checkbox Mengubah properti selesai
Hapus tugas @click, splice Menghapus item berdasarkan index
Hitung sisa tugas computed Menghitung item yang belum selesai

Kode Lengkap Praktikum

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>To-Do List Vue</title>
</head>
<body>
    <div id="app">
        <h1>To-Do List Praktikum</h1>

        <form @submit.prevent="tambahTugas">
            <input v-model="tugasBaru" placeholder="Contoh: Kerjakan laporan praktikum">
            <button type="submit">Tambah</button>
        </form>

        <p>Sisa tugas: {{ sisaTugas }}</p>

        <p v-if="daftarTugas.length === 0">Belum ada tugas.</p>

        <ul>
            <li v-for="(tugas, index) in daftarTugas" :key="tugas.id">
                <label>
                    <input type="checkbox" v-model="tugas.selesai">
                    <span>{{ tugas.nama }}</span>
                </label>
                <button @click="hapusTugas(index)">Hapus</button>
            </li>
        </ul>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
    <script>
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    tugasBaru: '',
                    daftarTugas: [
                        { id: 1, nama: 'Membaca materi computed', selesai: false },
                        { id: 2, nama: 'Mencoba contoh watcher', selesai: true }
                    ]
                };
            },
            computed: {
                sisaTugas() {
                    return this.daftarTugas.filter((tugas) => !tugas.selesai).length;
                }
            },
            methods: {
                tambahTugas() {
                    const nama = this.tugasBaru.trim();

                    if (nama === '') {
                        alert('Nama tugas tidak boleh kosong');
                        return;
                    }

                    this.daftarTugas.push({
                        id: Date.now(),
                        nama: nama,
                        selesai: false
                    });

                    this.tugasBaru = '';
                },
                hapusTugas(index) {
                    this.daftarTugas.splice(index, 1);
                }
            }
        }).mount('#app');
    </script>
</body>
</html>

Aktivitas Belajar 10 JavaScript

Array, Filter, dan Event Handling (1): Array

Array dapat dibayangkan seperti daftar hadir kelas. Setiap baris menyimpan satu data, dan daftar tersebut bisa ditambah, dikurangi, atau diubah. Di Vue, array sering dipakai untuk menampilkan daftar mahasiswa, daftar tugas, daftar produk, atau daftar mata kuliah.

Tujuan Pembelajaran:
  • Mahasiswa mampu menampilkan array menggunakan v-for.
  • Mahasiswa mampu menggunakan method mutasi array seperti push, pop, dan splice.
  • Mahasiswa mampu menggunakan :key saat melakukan looping data.

A. Menampilkan Array dengan v-for

v-for digunakan untuk mengulang elemen HTML berdasarkan isi array. Jika array berisi lima mahasiswa, Vue akan membuat lima baris tampilan.

<ul>
    <li v-for="(nama, index) in mahasiswa" :key="index">
        {{ index + 1 }}. {{ nama }}
    </li>
</ul>

B. Looping Array Objek

Dalam aplikasi nyata, data biasanya berbentuk objek. Satu mahasiswa tidak hanya punya nama, tetapi juga NIM, prodi, dan nilai.

C. Mengapa :key Penting?

:key membantu Vue mengenali identitas setiap item. Jika daftar berubah, Vue bisa memperbarui elemen yang tepat. Untuk data objek, gunakan id atau NIM yang unik, bukan index, jika tersedia.

Catatan: Index boleh dipakai sebagai key untuk contoh sederhana, tetapi untuk data yang bisa dihapus, diurutkan, atau difilter, gunakan key unik seperti id atau nim.

Latihan

Buat aplikasi daftar mata kuliah. Data minimal berisi kode, nama mata kuliah, dan jumlah SKS. Tampilkan dalam tabel, lalu buat tombol untuk menambah satu data dan menghapus data berdasarkan baris.

Aktivitas Belajar 11 JavaScript

Array, Filter, dan Event Handling (2): Filter, Event Handling

Setelah data bisa ditampilkan, kebutuhan berikutnya adalah mencari, menyaring, dan merespons tindakan pengguna. Misalnya dosen ingin mencari mahasiswa berdasarkan nama, menampilkan hanya mahasiswa yang lulus, atau menjalankan proses saat tombol diklik.

Tujuan Pembelajaran:
  • Mahasiswa mampu membuat fitur pencarian data menggunakan computed.
  • Mahasiswa mampu menggunakan event handling dengan v-on atau @.
  • Mahasiswa mampu menggunakan event modifier seperti .prevent, .stop, dan .enter.

A. Filter Data dengan Computed

Filter data sebaiknya dibuat dengan computed karena hasilnya adalah data turunan dari array asli dan kata kunci pencarian. Array asli tetap utuh, sedangkan tampilan menggunakan array hasil filter.

Contoh Kode: Pencarian dan Filter Mahasiswa

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Filter dan Event Vue</title>
</head>
<body>
    <div id="app">
        <h1>Data Nilai Mahasiswa</h1>

        <input
            v-model="keyword"
            @keyup.enter="catatPencarian"
            placeholder="Cari nama mahasiswa"
        >

        <select v-model="statusFilter">
            <option value="semua">Semua</option>
            <option value="lulus">Lulus</option>
            <option value="tidak-lulus">Tidak Lulus</option>
        </select>

        <p>Jumlah data tampil: {{ mahasiswaTampil.length }}</p>

        <ul>
            <li v-for="mhs in mahasiswaTampil" :key="mhs.nim">
                {{ mhs.nama }} - {{ mhs.nilai }}
                <button @click.stop="pilihMahasiswa(mhs)">Pilih</button>
            </li>
        </ul>

        <p v-if="pesan">{{ pesan }}</p>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
    <script>
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    keyword: '',
                    statusFilter: 'semua',
                    pesan: '',
                    mahasiswa: [
                        { nim: 'F1D001', nama: 'Andi', nilai: 82 },
                        { nim: 'F1D002', nama: 'Siti', nilai: 90 },
                        { nim: 'F1D003', nama: 'Raka', nilai: 58 },
                        { nim: 'F1D004', nama: 'Maya', nilai: 70 }
                    ]
                };
            },
            computed: {
                mahasiswaTampil() {
                    return this.mahasiswa.filter((mhs) => {
                        const cocokNama = mhs.nama
                            .toLowerCase()
                            .includes(this.keyword.toLowerCase());

                        const lulus = mhs.nilai >= 60;
                        const cocokStatus =
                            this.statusFilter === 'semua' ||
                            (this.statusFilter === 'lulus' && lulus) ||
                            (this.statusFilter === 'tidak-lulus' && !lulus);

                        return cocokNama && cocokStatus;
                    });
                }
            },
            methods: {
                pilihMahasiswa(mhs) {
                    this.pesan = `Mahasiswa dipilih: ${mhs.nama}`;
                },
                catatPencarian() {
                    this.pesan = `Pencarian dijalankan untuk: ${this.keyword}`;
                }
            }
        }).mount('#app');
    </script>
</body>
</html>

B. Event Handling

Event handling adalah cara Vue merespons tindakan pengguna. Bentuk panjangnya adalah v-on:click, sedangkan bentuk singkatnya adalah @click.

Event Contoh Fungsi
@click <button @click="simpan"> Menjalankan method saat tombol diklik
@input <input @input="ubahKeyword"> Merespons perubahan input
@submit.prevent <form @submit.prevent="kirim"> Mencegah reload halaman saat form dikirim
@keyup.enter <input @keyup.enter="cari"> Menjalankan method saat tombol Enter ditekan

C. Modifier prevent, stop, dan enter

.prevent mencegah perilaku bawaan browser, .stop menghentikan event agar tidak naik ke elemen induk, dan .enter membatasi event keyboard hanya saat tombol Enter ditekan.

<form @submit.prevent="simpanData">
    <input v-model="nama" @keyup.enter="simpanData">
    <button type="submit">Simpan</button>
</form>

<div @click="pilihKartu">
    <button @click.stop="hapusData">Hapus</button>
</div>
Tips: Untuk filter, jangan menghapus data asli. Simpan data asli di array utama, lalu tampilkan hasil filter dari computed.
