Tugas 3 — QMainWindow, Dialog & QSS

Implementasikan aplikasi desktop profesional dengan menu, toolbar, dialog, dan styling QSS

Indikator Penilaian

Setelah menyelesaikan tugas ini, mahasiswa mampu:
  1. Membangun QMainWindow lengkap dengan MenuBar, Toolbar, dan StatusBar
  2. Mengimplementasikan QAction dengan shortcut dan icon
  3. Menggunakan dialog bawaan dan membuat custom dialog
  4. Melakukan operasi file open/save
  5. Menerapkan QSS untuk styling tampilan aplikasi

Tugas Praktikum (Kerjakan 1 dari 3 soal tugas berikut)

Perhatian

Kerjakan hanya salah satu dari 3 tugas di bawah ini. Pilih tugas yang paling sesuai dengan kemampuan dan minat kamu. Tidak perlu mengerjakan semuanya.

Pengumpulan
  1. Simpan file Python dengan nama yang sesuai
  2. Test program dan pastikan berjalan tanpa error
  3. Ambil screenshot pada setiap aplikasi yang dijalankan
  4. Upload ke GitHub, buat repository dengan nama T3-week6 (upload file .py dan screenshot aplikasi berjalan)
  5. Submit link GitHub ke platform Berajah
Tambahan
  1. Tambahkan komentar Nama, NIM, dan Kelas di awal file
  2. Pastikan kode rapi dan mudah dibaca

Tugas 1: Notepad Clone

Buatlah aplikasi Notepad sederhana dengan fitur lengkap menggunakan QMainWindow dan QSS untuk styling.

Contoh Tampilan Aplikasi

📄 dokumen.txt * — Notepad — □ ✕
File Edit Format Help
| |
# Nama: Budi Santoso
# NIM: 12345678

Ini adalah contoh teks di dalam Notepad Clone.
Aplikasi ini mendukung fitur open, save, dan edit.
Teks yang diseleksi tampil seperti ini.
Baris: 5 | Karakter: 128 UTF-8 | Word Wrap: ON

Fitur yang Harus Diimplementasikan

  • QMainWindow dengan MenuBar (File, Edit, Format, Help), Toolbar, dan StatusBar
  • Operasi file: New, Open, Save, Save As menggunakan QFileDialog
  • Tracking perubahan — judul window menampilkan tanda * jika ada perubahan belum disimpan
  • Menu Edit: Undo, Redo, Cut, Copy, Paste, Select All dengan shortcut standar
  • Menu Format: Font dialog (QFontDialog) dan Word Wrap toggle
  • Custom dialog Find & Replace (QDialog) dengan tombol Find Next, Replace, Replace All
  • StatusBar menampilkan jumlah baris dan karakter secara real-time
  • Konfirmasi QMessageBox sebelum menutup jika ada perubahan belum disimpan
  • Styling QSS untuk tampilan yang menarik
FiturWidget/Class yang Digunakan
Window utamaQMainWindow, QTextEdit
Menu & aksiQMenuBar, QAction, QKeySequence
ToolbaraddToolBar(), QStyle.StandardPixmap
File dialogQFileDialog.getOpenFileName / getSaveFileName
Font dialogQFontDialog.getFont()
Find & ReplaceQDialog, QDialogButtonBox, QLineEdit
KonfirmasiQMessageBox.question()
StylingsetStyleSheet(), QSS

Tugas 2: Image Viewer

Buatlah aplikasi image viewer dengan tampilan profesional menggunakan QMainWindow, DockWidget, dan QSS.

Contoh Tampilan Aplikasi

🖼️ photo.jpg — Image Viewer — □ ✕
File View Image Help
| |
📁 File Browser
🖼 photo.jpg
🖼 sunset.png
🖼 avatar.png
🖼 logo.gif
🖼️
photo.jpg
800 × 600 px
photo.jpg | 800×600 px | 245 KB Zoom: 100%

Fitur yang Harus Diimplementasikan

  • QMainWindow dengan MenuBar, Toolbar, dan StatusBar
  • Dock widget (QDockWidget) di sisi kiri berisi QListWidget untuk daftar file gambar
  • Central widget menampilkan gambar menggunakan QLabel + QPixmap + QScrollArea
  • Open image dengan QFileDialog (filter: jpg, png, gif, bmp)
  • Zoom In / Zoom Out (shortcut Ctrl++ / Ctrl+-) dan Fit to Window
  • Rotate gambar 90°, 180°, 270° menggunakan QTransform
  • Custom dialog Image Properties menampilkan nama file, ukuran (px), format, dan ukuran file (KB)
  • StatusBar menampilkan info gambar dan persentase zoom
  • Styling QSS tema dark/modern
FiturWidget/Class yang Digunakan
Tampilan gambarQLabel, QPixmap, QScrollArea
File browserQDockWidget, QListWidget
Open fileQFileDialog.getOpenFileName()
Zoom & RotateQPixmap.scaled(), QTransform
Properties dialogQDialog, QFormLayout, QLabel
StylingsetStyleSheet(), QSS

Tugas 3: Task Manager

Buatlah aplikasi task manager dengan QTableWidget untuk menampilkan data, custom dialog untuk input, dan QSS untuk styling modern.

Contoh Tampilan Aplikasi

✅ Task Manager — □ ✕
File Task Help
| Filter:
No Judul Task Prioritas Status Due Date
1 Buat laporan praktikum High In Progress 2026-04-01
2 Review materi PySide6 Medium Todo 2026-04-05
3 Push code ke GitHub Low Done ✓ 2026-03-30
Total: 3 tasks | Done: 1 | In Progress: 1 | Todo: 1 tasks.json

Fitur yang Harus Diimplementasikan

  • QMainWindow dengan MenuBar, Toolbar, dan StatusBar
  • QTableWidget dengan kolom: No, Judul, Prioritas, Status, Due Date
  • Toolbar berisi tombol Add, Edit, Delete, filter QComboBox, dan search QLineEdit
  • Custom dialog Add/Edit Task: QLineEdit (judul), QComboBox (prioritas & status), QDateEdit (due date)
  • Warna baris tabel berbeda per prioritas — merah (High), kuning (Medium), hijau (Low)
  • Filter real-time berdasarkan prioritas atau status
  • Konfirmasi QMessageBox sebelum menghapus task
  • StatusBar menampilkan ringkasan jumlah task
  • Styling QSS untuk tampilan yang konsisten
FiturWidget/Class yang Digunakan
Tabel dataQTableWidget, QTableWidgetItem
Form inputQDialog, QFormLayout, QLineEdit, QComboBox, QDateEdit
Filter & SearchQComboBox, QLineEdit
KonfirmasiQMessageBox.question()
Warna prioritasQTableWidgetItem.setBackground()
StylingsetStyleSheet(), QSS
