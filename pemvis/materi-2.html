1. Konsep Event-Driven Programming

Event-Driven Programming adalah paradigma pemrograman di mana alur program ditentukan oleh events (kejadian) seperti aksi pengguna, sensor output, atau pesan dari program lain.

Definisi Event

Event adalah setiap aksi atau kejadian yang terdeteksi oleh program yang mungkin ditangani oleh program tersebut. Contoh: klik mouse, penekanan tombol keyboard, perubahan nilai input, dll.

Perbandingan Paradigma Pemrograman

Pemrograman Prosedural
  • Eksekusi berjalan secara linear
  • Program mengontrol alur eksekusi
  • Input diminta secara eksplisit
  • Cocok untuk batch processing
Event-Driven Programming
  • Eksekusi berdasarkan event
  • User mengontrol alur program
  • Program menunggu dan merespon event
  • Cocok untuk GUI application

Komponen Utama Event-Driven

Alur Event-Driven Programming
Event Source
(Button, Input)
→
Event Loop
(Mendeteksi)
→
Event Handler
(Memproses)
→
Response
(Aksi)
event_driven_basic.py Python 
import sys
from PySide6.QtWidgets import QApplication, QMainWindow, QPushButton

class MainWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Event-Driven Demo")
        self.setGeometry(100, 100, 300, 200)
        
        # Membuat button sebagai event source
        self.button = QPushButton("Klik Saya!", self)
        self.button.setGeometry(100, 80, 100, 30)
        
        # Menghubungkan event (signal) ke handler (slot)
        self.button.clicked.connect(self.on_button_click)
    
    def on_button_click(self):
        # Event handler - dipanggil saat button diklik
        print("Button telah diklik!")

if __name__ == "__main__":
    app = QApplication(sys.argv)  # Event loop dimulai
    window = MainWindow()
    window.show()
    sys.exit(app.exec())  # Menjalankan event loop
Penting!

app.exec() memulai event loop yang akan terus berjalan dan menunggu event hingga aplikasi ditutup. Tanpa event loop, aplikasi GUI tidak akan merespon interaksi user.

2. Signals & Slots (Observer Pattern)

Signals dan Slots adalah mekanisme inti Qt/PySide6 untuk komunikasi antar objek. Ini merupakan implementasi dari Observer Pattern yang memungkinkan loose coupling antar komponen.

Observer Pattern

Design pattern di mana sebuah objek (Subject/Observable) memelihara daftar dependennya (Observers) dan memberitahu mereka secara otomatis ketika terjadi perubahan state.

Anatomi Signal & Slot

Komponen Deskripsi Contoh
Signal Notifikasi yang dipancarkan ketika event terjadi clicked, textChanged
Slot Fungsi yang dipanggil sebagai respon terhadap signal def on_click(self)
Connect Menghubungkan signal ke slot signal.connect(slot)
Emit Memancarkan signal signal.emit()
Mekanisme Signal-Slot
Sender Object
(QPushButton)
→
Signal
(clicked)
→
connect()
→
Slot
(handler)
→
Receiver Object

Cara Menghubungkan Signal ke Slot

signal_slot_connection.py Python 
from PySide6.QtWidgets import (
    QApplication, QWidget, QVBoxLayout, 
    QPushButton, QLabel, QLineEdit
)

class SignalSlotDemo(QWidget):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Signal-Slot Demo")
        self.setup_ui()
    
    def setup_ui(self):
        layout = QVBoxLayout(self)
        
        # Label untuk menampilkan hasil
        self.label = QLabel("Ketik sesuatu di bawah...")
        layout.addWidget(self.label)
        
        # Input field
        self.input_field = QLineEdit()
        self.input_field.setPlaceholderText("Ketik di sini...")
        layout.addWidget(self.input_field)
        
        # Button
        self.button = QPushButton("Reset")
        layout.addWidget(self.button)
        
        # ===== MENGHUBUNGKAN SIGNALS KE SLOTS =====
        
        # Cara 1: Signal ke method
        self.button.clicked.connect(self.reset_text)
        
        # Cara 2: Signal ke lambda (inline function)
        self.input_field.textChanged.connect(
            lambda text: self.label.setText(f"Anda mengetik: {text}")
        )
        
        # Cara 3: Satu signal ke banyak slot
        self.button.clicked.connect(lambda: print("Button diklik!"))
    
    def reset_text(self):
        self.input_field.clear()
        self.label.setText("Text telah direset!")
Keuntungan Signal-Slot
  • Loose Coupling: Sender tidak perlu tahu siapa yang menerima signal
  • Multiple Connections: Satu signal bisa terhubung ke banyak slot
  • Type Safety: Qt memvalidasi tipe parameter signal dan slot
  • Thread Safety: Signal-slot aman digunakan antar thread

import sys
from PySide6.QtWidgets import (
    QApplication, QWidget, QVBoxLayout, QHBoxLayout,
    QPushButton, QLabel, QLineEdit, QSlider, 
    QSpinBox, QCheckBox, QComboBox
)
from PySide6.QtCore import Qt

class BuiltinSignalsDemo(QWidget):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Built-in Signals Demo")
        self.setMinimumWidth(400)
        self.setup_ui()
    
    def setup_ui(self):
        layout = QVBoxLayout(self)
        
        # === CLICKED SIGNAL ===
        self.click_label = QLabel("Jumlah klik: 0")
        self.click_count = 0
        self.click_btn = QPushButton("Klik Saya!")
        self.click_btn.clicked.connect(self.on_button_clicked)
        layout.addWidget(self.click_label)
        layout.addWidget(self.click_btn)
        
        # === TEXTCHANGED SIGNAL ===
        self.text_label = QLabel("Text: -")
        self.text_input = QLineEdit()
        self.text_input.setPlaceholderText("Ketik sesuatu...")
        self.text_input.textChanged.connect(self.on_text_changed)
        layout.addWidget(self.text_label)
        layout.addWidget(self.text_input)
        
        # === VALUECHANGED SIGNAL (Slider + SpinBox) ===
        slider_layout = QHBoxLayout()
        self.slider = QSlider(Qt.Horizontal)
        self.slider.setRange(0, 100)
        self.spinbox = QSpinBox()
        self.spinbox.setRange(0, 100)
        
        # Sinkronisasi slider dan spinbox
        self.slider.valueChanged.connect(self.spinbox.setValue)
        self.spinbox.valueChanged.connect(self.slider.setValue)
        
        slider_layout.addWidget(QLabel("Nilai:"))
        slider_layout.addWidget(self.slider)
        slider_layout.addWidget(self.spinbox)
        layout.addLayout(slider_layout)
        
        # === STATECHANGED SIGNAL ===
        self.checkbox = QCheckBox("Aktifkan Fitur")
        self.feature_label = QLabel("Status: Nonaktif")
        self.checkbox.stateChanged.connect(self.on_checkbox_changed)
        layout.addWidget(self.checkbox)
        layout.addWidget(self.feature_label)
        
        # === CURRENTINDEXCHANGED SIGNAL ===
        self.combo = QComboBox()
        self.combo.addItems(["Pilih Warna", "Merah", "Hijau", "Biru"])
        self.color_label = QLabel("Warna: -")
        self.combo.currentIndexChanged.connect(self.on_combo_changed)
        layout.addWidget(self.combo)
        layout.addWidget(self.color_label)
    
    # === SLOT METHODS ===
    def on_button_clicked(self):
        self.click_count += 1
        self.click_label.setText(f"Jumlah klik: {self.click_count}")
    
    def on_text_changed(self, text):
        self.text_label.setText(f"Text: {text}")
    
    def on_checkbox_changed(self, state):
        status = "Aktif" if state == Qt.Checked else "Nonaktif"
        self.feature_label.setText(f"Status: {status}")
    
    def on_combo_changed(self, index):
        if index > 0:
            warna = self.combo.currentText()
            self.color_label.setText(f"Warna: {warna}")

if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = BuiltinSignalsDemo()
    window.show()
    sys.exit(app.exec())

4. Custom Signals dengan Signal()

Selain signal bawaan, kita bisa membuat custom signal untuk komunikasi antar komponen sesuai kebutuhan aplikasi.

Kapan Menggunakan Custom Signal?
  • Komunikasi antar class/widget yang berbeda
  • Membuat komponen yang reusable
  • Mengirim data spesifik saat event terjadi
  • Implementasi arsitektur MVC/MVP

Membuat Custom Signal

1

Import Signal dari PySide6.QtCore

Pastikan mengimport class Signal untuk membuat custom signal.

2

Definisikan Signal sebagai Class Attribute

Signal didefinisikan di level class, bukan di dalam method.

3

Emit Signal saat diperlukan

Gunakan method emit() untuk memancarkan signal.

custom_signals_demo.py Python 
import sys
from PySide6.QtWidgets import (
    QApplication, QWidget, QVBoxLayout, QHBoxLayout,
    QPushButton, QLabel, QLineEdit, QProgressBar
)
from PySide6.QtCore import Signal, QObject

# ==========================================
# CONTOH 1: Custom Signal Sederhana
# ==========================================

class Counter(QObject):
    # Definisi custom signals
    count_changed = Signal(int)       # Signal dengan parameter int
    limit_reached = Signal()           # Signal tanpa parameter
    
    def __init__(self, limit=10):
        super().__init__()
        self._count = 0
        self._limit = limit
    
    def increment(self):
        self._count += 1
        self.count_changed.emit(self._count)  # Emit signal dengan nilai
        
        if self._count >= self._limit:
            self.limit_reached.emit()  # Emit signal tanpa parameter
    
    def reset(self):
        self._count = 0
        self.count_changed.emit(self._count)


# ==========================================
# CONTOH 2: Signal dengan Multiple Parameters
# ==========================================

class LoginManager(QObject):
    # Signal dengan multiple parameters
    login_attempted = Signal(str, str)      # username, password
    login_success = Signal(str)              # username
    login_failed = Signal(str)               # error message
    
    def __init__(self):
        super().__init__()
        self.valid_users = {"admin": "admin123", "user": "user123"}
    
    def attempt_login(self, username, password):
        self.login_attempted.emit(username, password)
        
        if username in self.valid_users:
            if self.valid_users[username] == password:
                self.login_success.emit(username)
            else:
                self.login_failed.emit("Password salah!")
        else:
            self.login_failed.emit("Username tidak ditemukan!")


# ==========================================
# APLIKASI UTAMA
# ==========================================

class MainWindow(QWidget):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Custom Signals Demo")
        self.setup_ui()
        self.setup_connections()
    
    def setup_ui(self):
        layout = QVBoxLayout(self)
        
        # Counter Section
        layout.addWidget(QLabel("<b>Counter Demo:</b>"))
        self.counter = Counter(limit=10)
        self.progress = QProgressBar()
        self.progress.setRange(0, 10)
        self.count_label = QLabel("Count: 0")
        self.increment_btn = QPushButton("Increment")
        self.reset_btn = QPushButton("Reset")
        
        counter_layout = QHBoxLayout()
        counter_layout.addWidget(self.increment_btn)
        counter_layout.addWidget(self.reset_btn)
        
        layout.addWidget(self.count_label)
        layout.addWidget(self.progress)
        layout.addLayout(counter_layout)
        
        # Login Section
        layout.addWidget(QLabel("<br><b>Login Demo:</b>"))
        self.login_manager = LoginManager()
        self.username_input = QLineEdit()
        self.username_input.setPlaceholderText("Username")
        self.password_input = QLineEdit()
        self.password_input.setPlaceholderText("Password")
        self.password_input.setEchoMode(QLineEdit.Password)
        self.login_btn = QPushButton("Login")
        self.status_label = QLabel("Status: Belum login")
        
        layout.addWidget(self.username_input)
        layout.addWidget(self.password_input)
        layout.addWidget(self.login_btn)
        layout.addWidget(self.status_label)
    
    def setup_connections(self):
        # Counter connections
        self.increment_btn.clicked.connect(self.counter.increment)
        self.reset_btn.clicked.connect(self.counter.reset)
        self.counter.count_changed.connect(self.update_count)
        self.counter.limit_reached.connect(self.show_limit_message)
        
        # Login connections
        self.login_btn.clicked.connect(self.do_login)
        self.login_manager.login_success.connect(self.on_login_success)
        self.login_manager.login_failed.connect(self.on_login_failed)
    
    def update_count(self, value):
        self.count_label.setText(f"Count: {value}")
        self.progress.setValue(value)
    
    def show_limit_message(self):
        self.count_label.setText("Limit tercapai! Reset untuk mulai lagi.")
        self.increment_btn.setEnabled(False)
    
    def do_login(self):
        username = self.username_input.text()
        password = self.password_input.text()
        self.login_manager.attempt_login(username, password)
    
    def on_login_success(self, username):
        self.status_label.setText(f"Selamat datang, {username}!")
        self.status_label.setStyleSheet("color: green;")
    
    def on_login_failed(self, message):
        self.status_label.setText(f"Error: {message}")
        self.status_label.setStyleSheet("color: red;")

if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = MainWindow()
    window.show()
    sys.exit(app.exec())
Perhatian!

Signal harus didefinisikan sebagai class attribute, bukan instance attribute. Signal tidak bisa didefinisikan di dalam __init__ method.

5. Event Handling: Mouse, Keyboard, Close

Selain menggunakan signals, kita bisa menangani event secara langsung dengan meng-override method event handler dari QWidget.

Perbedaan Event vs Signal

Event Signal
Low-level, langsung dari sistem High-level, abstraksi dari event
Override method (inheritance) Connect ke slot (composition)
Bisa difilter/diabaikan Selalu diteruskan ke slot
Contoh: mousePressEvent Contoh: clicked

Daftar Event Handler Penting

Kategori Method Deskripsi
Mouse mousePressEvent Saat tombol mouse ditekan
mouseReleaseEvent Saat tombol mouse dilepas
mouseMoveEvent Saat mouse bergerak
mouseDoubleClickEvent Saat double click
Keyboard keyPressEvent Saat tombol keyboard ditekan
keyReleaseEvent Saat tombol keyboard dilepas
Window closeEvent Saat window akan ditutup
resizeEvent Saat window di-resize
event_handling_demo.py Python 
import sys
from PySide6.QtWidgets import (
    QApplication, QWidget, QVBoxLayout, QLabel, QMessageBox
)
from PySide6.QtCore import Qt
from PySide6.QtGui import QMouseEvent, QKeyEvent, QCloseEvent

class EventHandlingDemo(QWidget):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Event Handling Demo")
        self.setGeometry(100, 100, 400, 300)
        self.setMouseTracking(True)  # Enable mouse tracking
        self.setup_ui()
    
    def setup_ui(self):
        layout = QVBoxLayout(self)
        
        self.info_label = QLabel("Gerakkan mouse, tekan keyboard, atau klik!")
        self.mouse_label = QLabel("Mouse: -")
        self.key_label = QLabel("Keyboard: -")
        self.click_label = QLabel("Click: -")
        
        layout.addWidget(self.info_label)
        layout.addWidget(self.mouse_label)
        layout.addWidget(self.key_label)
        layout.addWidget(self.click_label)
        layout.addStretch()
    
    # ============ MOUSE EVENTS ============
    
    def mouseMoveEvent(self, event: QMouseEvent):
        x = event.position().x()
        y = event.position().y()
        self.mouse_label.setText(f"Mouse Position: ({x:.0f}, {y:.0f})")
    
    def mousePressEvent(self, event: QMouseEvent):
        button = "Left" if event.button() == Qt.LeftButton else "Right"
        x = event.position().x()
        y = event.position().y()
        self.click_label.setText(f"{button} Click at ({x:.0f}, {y:.0f})")
    
    def mouseDoubleClickEvent(self, event: QMouseEvent):
        self.click_label.setText("Double Click Detected!")
    
    # ============ KEYBOARD EVENTS ============
    
    def keyPressEvent(self, event: QKeyEvent):
        key = event.key()
        text = event.text()
        
        # Cek modifier keys
        modifiers = []
        if event.modifiers() & Qt.ControlModifier:
            modifiers.append("Ctrl")
        if event.modifiers() & Qt.ShiftModifier:
            modifiers.append("Shift")
        if event.modifiers() & Qt.AltModifier:
            modifiers.append("Alt")
        
        mod_str = " + ".join(modifiers) + " + " if modifiers else ""
        
        # Handle special keys
        if key == Qt.Key_Escape:
            self.key_label.setText("Key: ESC pressed")
        elif key == Qt.Key_Return:
            self.key_label.setText("Key: ENTER pressed")
        else:
            self.key_label.setText(f"Key: {mod_str}{text} (Code: {key})")
    
    # ============ CLOSE EVENT ============
    
    def closeEvent(self, event: QCloseEvent):
        reply = QMessageBox.question(
            self,
            "Konfirmasi",
            "Apakah Anda yakin ingin keluar?",
            QMessageBox.Yes | QMessageBox.No,
            QMessageBox.No
        )
        
        if reply == QMessageBox.Yes:
            event.accept()  # Izinkan window ditutup
        else:
            event.ignore()  # Batalkan penutupan

if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = EventHandlingDemo()
    window.show()
    sys.exit(app.exec())
Tips Event Handling
  • Gunakan event.accept() untuk menerima event
  • Gunakan event.ignore() untuk meneruskan event ke parent
  • Aktifkan setMouseTracking(True) untuk mendeteksi mouse move tanpa klik
  • Selalu panggil method parent jika ingin behavior default tetap berjalan

6. Validasi Input Real-time

Validasi input real-time memberikan feedback langsung kepada user saat mereka mengetik, meningkatkan user experience dan mencegah error submission.

Pendekatan Validasi

Validasi Saat Submit
  • User mengisi semua field
  • Klik submit button
  • Menampilkan semua error sekaligus
  • User harus memperbaiki dan submit lagi
Validasi Real-time
  • Validasi saat user mengetik
  • Feedback visual langsung
  • User langsung tahu kesalahan
  • Submit button diaktifkan saat valid

Contoh: Form Registrasi dengan Validasi Real-time

realtime_validation.py Python 
import sys
import re
from PySide6.QtWidgets import (
    QApplication, QWidget, QVBoxLayout, QHBoxLayout,

from PySide6.QtGui import QRegularExpressionValidator

class ValidatedLineEdit(QLineEdit):
    """Custom QLineEdit dengan validasi real-time"""
    
    validation_changed = Signal(bool)  # Signal untuk status validasi
    
    def __init__(self, validator_func=None, error_message=""):
        super().__init__()
        self.validator_func = validator_func
        self.error_message = error_message
        self.is_valid = False
        
        # Connect textChanged ke validasi
        self.textChanged.connect(self.validate)
    
    def validate(self):
        if self.validator_func:
            self.is_valid = self.validator_func(self.text())
            self.update_style()
            self.validation_changed.emit(self.is_valid)
    
    def update_style(self):
        if len(self.text()) == 0:
            self.setStyleSheet("")
        elif self.is_valid:
            self.setStyleSheet("""
                QLineEdit {
                    border: 2px solid #27ae60;
                    border-radius: 5px;
                    padding: 5px;
                    background-color: #e8f8f0;
                }
            """)
        else:
            self.setStyleSheet("""
                QLineEdit {
                    border: 2px solid #e74c3c;
                    border-radius: 5px;
                    padding: 5px;
                    background-color: #fdeaea;
                }
            """)

            lambda valid: self.show_error(self.email_error, self.email_input, valid)
        )
        self.password_input.validation_changed.connect(
            lambda valid: self.show_error(self.password_error, self.password_input, valid)
        )
        self.confirm_input.validation_changed.connect(
            lambda valid: self.show_error(self.confirm_error, self.confirm_input, valid)
        )
        
        # Submit action
        self.submit_btn.clicked.connect(self.submit_form)
    
    # ============ VALIDATOR FUNCTIONS ============
    
    def validate_username(self, text):
        return len(text) >= 4 and text.isalnum()
    
    def validate_email(self, text):
        pattern = r'^[\w\.-]+@[\w\.-]+\.\w+$'
        return bool(re.match(pattern, text))
    
    def validate_password(self, text):
        has_letter = any(c.isalpha() for c in text)
        has_digit = any(c.isdigit() for c in text)
        return len(text) >= 8 and has_letter and has_digit
    
    def validate_confirm(self, text):
        return text == self.password_input.text() and len(text) > 0
    
    def check_password_strength(self, text):
        if len(text) == 0:
            self.strength_label.setText("Kekuatan: -")
            self.strength_label.setStyleSheet("")
            return
        
        score = 0
        if len(text) >= 8: score += 1
        if len(text) >= 12: score += 1
        if any(c.isupper() for c in text): score += 1
        if any(c.islower() for c in text): score += 1
        if any(c.isdigit() for c in text): score += 1
        if any(c in "!@#$%^&*" for c in text): score += 1
        
        if score <= 2:
            self.strength_label.setText("Kekuatan: Lemah")
            self.strength_label.setStyleSheet("color: #e74c3c;")
        elif score <= 4:
            self.strength_label.setText("Kekuatan: Sedang")
            self.strength_label.setStyleSheet("color: #f39c12;")
        else:
            self.strength_label.setText("Kekuatan: Kuat")
            self.strength_label.setStyleSheet("color: #27ae60;")
    
    def show_error(self, label, input_field, is_valid):
        if len(input_field.text()) == 0:
            label.setText("")
        elif not is_valid:
            label.setText(input_field.error_message)
        else:
            label.setText("")
    
    def check_form(self):
        all_valid = (
            self.username_input.is_valid and
            self.email_input.is_valid and
            self.password_input.is_valid and
            self.confirm_input.is_valid
        )
        self.submit_btn.setEnabled(all_valid)
    
    def submit_form(self):
        QMessageBox.information(
            self,
            "Berhasil",
            f"Registrasi berhasil!\n\nUsername: {self.username_input.text()}\nEmail: {self.email_input.text()}"
        )

if __name__ == "__main__":
