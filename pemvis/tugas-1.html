TUGAS PRAKTIKUM 1

Aplikasi Form Biodata Mahasiswa

Persyaratan yang Harus Dipenuhi:

  • Tambahkan input untuk NIM dengan placeholder "Masukkan NIM"
  • Tambahkan input untuk Kelas dengan placeholder "Contoh: TI-2A"
  • ComboBox jenis kelamin berisi: "Laki-laki" dan "Perempuan"
  • Tombol "Tampilkan" menampilkan semua data di label hasil
  • Tombol "Reset" mengosongkan semua input
  • Validasi: Semua field harus diisi

Hasil yang Diharapkan:

Kriteria Penilaian:

Komponen
Input NIM dan Kelas berfungsi dengan benar
ComboBox jenis kelamin berfungsi
Tombol Tampilkan menampilkan semua data
Tombol Reset mengosongkan semua field
Validasi field kosong berfungsi

TUGAS PRAKTIKUM 2

Aplikasi Konversi Suhu

Lengkapi kode konversi suhu berikut. Aplikasi harus bisa mengkonversi dari Celsius ke Fahrenheit, Kelvin, dan Reamur.

Persyaratan yang Harus Dipenuhi:

  • Input suhu dalam Celsius
  • 3 tombol konversi: ke Fahrenheit, Kelvin, dan Reamur
  • Rumus:
    • Fahrenheit = (Celsius x 9/5) + 32
    • Kelvin = Celsius + 273.15
    • Reamur = Celsius x 4/5
  • Validasi input harus berupa angka
  • Hasil ditampilkan dengan 2 angka desimal

Hasil yang Diharapkan:

Konversi Suhu
KONVERSI SUHU
Fahrenheit Kelvin Reamur
Hasil Konversi:

100 Celsius = 212.00 Fahrenheit

Kriteria Penilaian:

Komponen
Konversi ke Fahrenheit benar
Konversi ke Kelvin benar
Konversi ke Reamur benar
Validasi input dan format 2 desimal

TUGAS PRAKTIKUM 3

Aplikasi Form Login

Persyaratan yang Harus Dipenuhi:

  • Input username dan password
  • Password harus disembunyikan (EchoMode.Password)
  • Checkbox "Tampilkan Password" untuk toggle visibility
  • Validasi login:
    • Username: admin
    • Password: 12345
  • Tampilkan pesan sukses/gagal dengan warna berbeda
  • Tombol Reset untuk mengosongkan form

Hasil yang Diharapkan:

Login
LOGIN
Tampilkan Password
Login Reset
Login berhasil! Selamat datang, admin.
Login
LOGIN
Tampilkan Password
Login Reset
Login gagal! Username atau password salah.

Tampilan login sukses (kiri) dan gagal (kanan)

Kriteria Penilaian:

Komponen
Password tersembunyi dengan EchoMode
Checkbox toggle password berfungsi
Validasi login benar (admin/12345)
Pesan sukses/gagal dengan warna berbeda
Tombol reset berfungsi
