Tugas Praktikum — PySide6 + SQLite

Buat aplikasi CRUD sederhana dengan QMainWindow, MenuBar, Layout, dan Style

Ketentuan Umum

Aturan Tugas
  1. Buat aplikasi CRUD sederhana menggunakan PySide6 dan SQLite.
  2. Gunakan QMainWindow, Menu Bar, Layout, dan QTableWidget.
  3. Form input terdiri dari 3 field sesuai pembagian tugas.
  4. Sediakan tombol Simpan, Hapus, dan Bersihkan Form.
  5. Semua field wajib diisi. Tambahkan validasi sesuai tipe data.
  6. Terapkan setStyleSheet() minimal pada 3 widget.
  7. Tampilkan Nama dan NIM pada status bar serta menu Tentang.
  8. Tulis program dari awal, tanpa kerangka kode jadi.
