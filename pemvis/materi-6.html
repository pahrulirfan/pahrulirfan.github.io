1. Overview Visualisasi Data

Visualisasi data membantu user memahami pola, tren, perbandingan, dan distribusi data tanpa harus membaca tabel satu per satu. Pada aplikasi desktop, grafik biasanya dipakai untuk dashboard penjualan, monitoring stok, nilai mahasiswa, laporan keuangan, hasil sensor, atau ringkasan data CRUD.

Kapan Data Perlu Divisualisasikan?

Gunakan chart ketika user perlu membandingkan nilai, melihat perubahan dari waktu ke waktu, mencari komposisi kategori, atau membaca pola distribusi. Jika user perlu mengedit record satu per satu, tabel tetap lebih cocok; chart menjadi pelengkap untuk ringkasan dan analisis.

2. Plugin dan Setup Environment

Pada praktikum ini, visualisasi data dibuat dengan kombinasi PySide6, Matplotlib, Pandas, dan NumPy. PySide6 bertugas membuat tampilan aplikasi desktop, Matplotlib bertugas menggambar grafik, Pandas bertugas membaca dan mengolah data tabular, sedangkan NumPy membantu proses perhitungan numerik yang sering dipakai oleh Pandas dan Matplotlib.

Kenapa Perlu Plugin Visualisasi?

PySide6 menyediakan komponen antarmuka seperti tombol, tabel, layout, dan window, tetapi tidak menyediakan fitur chart lengkap secara langsung. Karena itu aplikasi desktop biasanya memakai library tambahan untuk menggambar grafik. Library grafik tersebut kemudian ditanam ke dalam layout PySide6 sehingga chart tampil sebagai bagian dari aplikasi, bukan sebagai window terpisah.

Library Peran Dipakai Untuk
PySide6 Framework GUI desktop berbasis Qt Membuat window, layout, tombol, tabel, combobox, dan event user
Matplotlib Library utama untuk membuat grafik Membuat bar chart, line chart, pie chart, scatter plot, judul, label, legenda, dan grid
FigureCanvasQTAgg Backend Matplotlib untuk aplikasi Qt Menanamkan objek grafik Matplotlib ke dalam layout PySide6 sebagai widget
Pandas Library pengolahan data tabular Membaca CSV, memfilter data, menghitung total, mengelompokkan data, dan menyiapkan data untuk chart
NumPy Library numerik untuk array dan perhitungan Mendukung operasi angka, perhitungan statistik sederhana, dan menjadi dependency penting untuk Pandas/Matplotlib
Keputusan Praktikum

Gunakan Matplotlib sebagai plugin visualisasi utama karena stabil, dokumentasinya banyak, mudah digabung dengan Pandas, dan bisa ditampilkan langsung di PySide6 melalui FigureCanvasQTAgg. Dengan pendekatan ini, chart menjadi bagian dari tampilan aplikasi desktop dan dapat diperbarui ketika user memilih filter atau menekan tombol refresh.

Alternatif Kelebihan Kapan Dipakai
PyQtGraph Sangat cepat untuk data realtime Monitoring sensor, sinyal, data streaming
Plotly Interaktif dan bagus untuk web Jika chart ditampilkan di browser atau QWebEngine

Setelah semua dependency terpasang, alur coding biasanya dimulai dari membaca data dengan Pandas, mengolah data menjadi ringkasan, membuat figure Matplotlib, memasukkan figure ke canvas Qt, lalu menambahkan canvas ke layout PySide6. 3. Embed Chart ke PySide6

Inti integrasinya adalah membuat objek Figure, memasukkannya ke FigureCanvasQTAgg, lalu menambahkan canvas tersebut ke layout PySide6 seperti widget biasa.

chart_basic.py MATPLOTLIB + PYSIDE6 
import sys

# Import komponen utama PySide6 untuk membuat aplikasi desktop.
from PySide6.QtWidgets import QApplication, QMainWindow, QWidget, QVBoxLayout

# FigureCanvasQTAgg adalah penghubung agar grafik Matplotlib bisa tampil di PySide6.
from matplotlib.backends.backend_qtagg import FigureCanvasQTAgg as FigureCanvas

# Figure adalah area kosong tempat grafik Matplotlib digambar.
from matplotlib.figure import Figure


class ChartWindow(QMainWindow):
    def __init__(self):
        super().__init__()

        # Mengatur judul window dan ukuran awal aplikasi.
        self.setWindowTitle("Grafik Penjualan")
        self.resize(800, 500)

        # QWidget dipakai sebagai container utama.
        # QVBoxLayout menyusun widget secara vertikal dari atas ke bawah.
        central = QWidget()
        layout = QVBoxLayout(central)
        self.setCentralWidget(central)

        # Membuat area figure Matplotlib.
        # tight_layout=True membantu agar judul dan label tidak terpotong.
        self.figure = Figure(figsize=(6, 4), tight_layout=True)

        # Figure dimasukkan ke canvas agar bisa dipasang ke layout PySide6.
        self.canvas = FigureCanvas(self.figure)
        layout.addWidget(self.canvas)

        # Memanggil method untuk menggambar grafik pertama kali.
        self.plot_sales()

    def plot_sales(self):
        # Data contoh yang akan divisualisasikan.
        months = ["Jan", "Feb", "Mar", "Apr", "Mei", "Jun"]
        sales = [12, 19, 14, 25, 22, 30]

        # Menambahkan area plot ke dalam figure.
        ax = self.figure.add_subplot(111)

        # Membuat line chart dengan marker titik pada setiap bulan.
        ax.plot(months, sales, marker="o", linewidth=2)

        # Memberi informasi pada grafik agar mudah dibaca.
        ax.set_title("Tren Penjualan Semester 1")
        ax.set_xlabel("Bulan")
        ax.set_ylabel("Jumlah Penjualan")
        ax.grid(True, alpha=0.3)

        # Memperbarui canvas supaya grafik muncul di window PySide6.
        self.canvas.draw()


if __name__ == "__main__":
    # Membuat object aplikasi Qt.
    app = QApplication(sys.argv)

    # Membuat dan menampilkan window utama.
    window = ChartWindow()
    window.show()

    # Menjalankan event loop aplikasi.
    sys.exit(app.exec())

import sys
from pathlib import Path

import pandas as pd
from PySide6.QtWidgets import QApplication, QComboBox, QLabel, QMainWindow, QWidget, QVBoxLayout
from matplotlib.backends.backend_qtagg import FigureCanvasQTAgg as FigureCanvas
from matplotlib.figure import Figure


class InteractiveChart(QMainWindow):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Chart Interaktif dari CSV")

        # Membaca file sales.csv dari folder yang sama dengan file Python ini.
        # Cara ini lebih aman daripada langsung pd.read_csv("sales.csv")
        # karena working directory bisa berbeda saat program dijalankan.
        csv_path = Path(__file__).parent / "sales.csv"
        self.df = pd.read_csv(csv_path)

        central = QWidget()
        layout = QVBoxLayout(central)
        self.setCentralWidget(central)

        # Combobox pertama dipakai untuk memfilter data berdasarkan kategori.
        layout.addWidget(QLabel("Pilih kategori:"))
        self.category_filter = QComboBox()
        self.category_filter.addItem("Semua")
        self.category_filter.addItems(sorted(self.df["kategori"].unique()))
        self.category_filter.currentTextChanged.connect(self.update_chart)
        layout.addWidget(self.category_filter)

        # Combobox kedua dipakai untuk memilih jenis grafik.
        layout.addWidget(QLabel("Pilih tipe chart:"))
        self.chart_type = QComboBox()
        self.chart_type.addItems(["Line Chart", "Bar Chart"])
        self.chart_type.currentTextChanged.connect(self.update_chart)
        layout.addWidget(self.chart_type)

        self.figure = Figure(tight_layout=True)
        self.canvas = FigureCanvas(self.figure)
        layout.addWidget(self.canvas)

        self.update_chart()

    def update_chart(self):
        self.figure.clear()
        ax = self.figure.add_subplot(111)

        selected_category = self.category_filter.currentText()

        # Jika user memilih kategori tertentu, data difilter dulu.
        # Jika memilih Semua, seluruh data dipakai.
        if selected_category == "Semua":
            filtered_df = self.df
        else:
            filtered_df = self.df[self.df["kategori"] == selected_category]

        # Data dikelompokkan per bulan agar total penjualan setiap bulan bisa digambar.
        summary = filtered_df.groupby("bulan", sort=False)["total"].sum().reset_index()
        months = summary["bulan"].tolist()
        totals = summary["total"].tolist()

        if self.chart_type.currentText() == "Bar Chart":
            ax.bar(months, totals, color="#27ae60")
        else:
            ax.plot(months, totals, marker="o", color="#3498db")

        ax.set_title("Penjualan per Bulan")
        ax.set_xlabel("Bulan")
        ax.set_ylabel("Total Penjualan")
        ax.grid(True, alpha=0.25)
        self.canvas.draw()


if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = InteractiveChart()
    window.resize(800, 500)
    window.show()
    sys.exit(app.exec())

import pandas as pd


MONTH_ORDER = ["Jan", "Feb", "Mar", "Apr", "Mei", "Jun"]


def load_sales_data(csv_path):
    # Membaca data mentah dari CSV.
    df = pd.read_csv(csv_path)

    # Memastikan kolom total bertipe angka agar bisa dihitung.
    df["total"] = pd.to_numeric(df["total"])
    return df


def get_categories(df):
    # Mengambil daftar kategori unik untuk isi combobox filter.
    categories = sorted(df["kategori"].unique())
    return ["Semua"] + categories


def filter_by_category(df, category):
    # Jika user memilih Semua, data tidak difilter.
    if category == "Semua":
        return df

    return df[df["kategori"] == category]


def summarize_by_month(df):
    # Mengelompokkan data per bulan agar chart tidak menampilkan data terlalu detail.
    summary = df.groupby("bulan", sort=False)["total"].sum().reset_index()

    # Menjaga urutan bulan agar tidak berubah secara alfabetis.
    summary["bulan"] = pd.Categorical(summary["bulan"], categories=MONTH_ORDER, ordered=True)
    summary = summary.sort_values("bulan")
    return summary
chart_widget.py CHART LAYER 
from PySide6.QtWidgets import QWidget, QVBoxLayout
from matplotlib.backends.backend_qtagg import FigureCanvasQTAgg as FigureCanvas
from matplotlib.figure import Figure


class SalesChartWidget(QWidget):
    def __init__(self):
        super().__init__()

        layout = QVBoxLayout(self)
        self.figure = Figure(figsize=(6, 4), tight_layout=True)
        self.canvas = FigureCanvas(self.figure)
        layout.addWidget(self.canvas)

    def plot(self, summary, chart_type, title):
        # Membersihkan chart lama sebelum menggambar chart baru.
        self.figure.clear()
        ax = self.figure.add_subplot(111)

        months = summary["bulan"].astype(str).tolist()
        totals = summary["total"].tolist()

        if chart_type == "Bar Chart":
            ax.bar(months, totals, color="#2c7be5")
        elif chart_type == "Line Chart":
            ax.plot(months, totals, marker="o", linewidth=2, color="#00a676")
        elif chart_type == "Pie Chart":
            ax.pie(totals, labels=months, autopct="%1.1f%%", startangle=90)
            ax.axis("equal")

        ax.set_title(title)

        # Pie chart tidak membutuhkan label sumbu dan grid.
        if chart_type != "Pie Chart":
            ax.set_xlabel("Bulan")
            ax.set_ylabel("Total Penjualan")
            ax.grid(True, alpha=0.25)

        self.canvas.draw()

class DashboardWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Dashboard Penjualan")
        self.resize(1000, 650)

        csv_path = Path(__file__).parent / "data" / "sales_dashboard.csv"
        self.df = load_sales_data(csv_path)

        self.setup_ui()
        self.update_dashboard()

    def setup_ui(self):
        central = QWidget()
        main_layout = QVBoxLayout(central)
        self.setCentralWidget(central)

        filter_layout = QHBoxLayout()

        filter_layout.addWidget(QLabel("Kategori:"))
        self.category_filter = QComboBox()
        self.category_filter.addItems(get_categories(self.df))
        self.category_filter.currentTextChanged.connect(self.update_dashboard)
        filter_layout.addWidget(self.category_filter)

        filter_layout.addWidget(QLabel("Jenis Chart:"))
        self.chart_type = QComboBox()
        self.chart_type.addItems(["Bar Chart", "Line Chart", "Pie Chart"])
        self.chart_type.currentTextChanged.connect(self.update_dashboard)
        filter_layout.addWidget(self.chart_type)

        filter_layout.addStretch()
        main_layout.addLayout(filter_layout)

        self.summary_label = QLabel()
        main_layout.addWidget(self.summary_label)

        content_layout = QHBoxLayout()

        self.table = QTableWidget()
        content_layout.addWidget(self.table, stretch=2)

        self.chart = SalesChartWidget()
        content_layout.addWidget(self.chart, stretch=3)

        main_layout.addLayout(content_layout)

    def update_dashboard(self):
        selected_category = self.category_filter.currentText()
        chart_type = self.chart_type.currentText()

        filtered_df = filter_by_category(self.df, selected_category)
        summary = summarize_by_month(filtered_df)

        self.update_summary(filtered_df)
        self.update_table(filtered_df)
        self.chart.plot(summary, chart_type, f"Penjualan per Bulan - {selected_category}")

    def update_summary(self, df):
        total_sales = df["total"].sum()
        total_rows = len(df)
        total_products = df["produk"].nunique()

        self.summary_label.setText(
            f"Total penjualan: Rp {total_sales:,.0f} | "
            f"Jumlah data: {total_rows} baris | "
            f"Jumlah produk: {total_products}"
        )

    def update_table(self, df):
        columns = ["bulan", "kategori", "produk", "total"]

        self.table.setRowCount(len(df))
        self.table.setColumnCount(len(columns))
        self.table.setHorizontalHeaderLabels(["Bulan", "Kategori", "Produk", "Total"])

        for row_index, row in df.reset_index(drop=True).iterrows():
            for col_index, column in enumerate(columns):
                value = row[column]

                if column == "total":
                    value = f"Rp {value:,.0f}"

                self.table.setItem(row_index, col_index, QTableWidgetItem(str(value)))

        self.table.resizeColumnsToContents()

7. Best Practices

Best practices membantu aplikasi visualisasi data tetap rapi, mudah dibaca, dan tidak lambat ketika data bertambah banyak. Pada aplikasi PySide6, masalah yang sering muncul biasanya berasal dari chart yang digambar berulang tanpa dibersihkan, data yang belum diringkas, atau proses membaca data besar yang dilakukan langsung di thread utama UI.

Hindari
  • Menggambar chart baru tanpa figure.clear()
  • Memakai pie chart untuk terlalu banyak kategori
  • Membaca file besar langsung di slot tombol tanpa thread
  • Warna chart terlalu ramai dan tidak konsisten
Lakukan
  • Pakai FigureCanvasQTAgg sebagai widget chart
  • Agregasi data dulu dengan Pandas
  • Tambahkan label sumbu, judul, dan grid seperlunya
  • Gunakan QThread untuk load data besar atau API lambat
Praktik Alasan Dampak pada Aplikasi
Bersihkan figure sebelum menggambar ulang Chart lama masih tersimpan di figure jika tidak dibersihkan Grafik tidak menumpuk dan tampilan tetap benar saat filter berubah
Agregasi data sebelum divisualisasikan Chart lebih mudah dibaca jika data sudah diringkas per bulan, kategori, atau produk User cepat memahami pola tanpa membaca ratusan baris data mentah
Pilih chart sesuai pertanyaan data Tidak semua chart cocok untuk semua jenis data Bar chart cocok untuk perbandingan, line chart untuk tren, pie chart untuk komposisi sederhana
Pisahkan data, chart, dan UI Setiap bagian punya tanggung jawab yang berbeda Kode lebih mudah diuji, diperbaiki, dan dikembangkan menjadi final project
Gunakan path file yang jelas Working directory bisa berbeda antara Terminal dan IDE Mengurangi error file CSV tidak ditemukan
Jangan proses data besar langsung di slot tombol Slot berjalan di thread utama UI Aplikasi tidak freeze saat membaca file besar atau mengambil data dari API
Pola Update Chart yang Aman

Setiap kali user mengganti filter atau tipe chart, gunakan pola yang sama: ambil pilihan user, filter data, ringkas data, bersihkan figure, gambar chart baru, lalu panggil canvas.draw(). Urutan ini membuat chart selalu sinkron dengan data yang sedang aktif.

Kesalahan yang Sering Terjadi

Jika chart tidak berubah setelah combobox dipilih, periksa apakah signal sudah dihubungkan ke method update, misalnya currentTextChanged.connect(self.update_chart). Jika data CSV tidak terbaca, gunakan Path(__file__).parent agar path file dihitung dari lokasi file Python, bukan dari lokasi Terminal saat program dijalankan.
