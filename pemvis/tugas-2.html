Event & Signal Handling

Materi lanjutan terkait Event-Driven Programming dan implementasi Signals & Slots

Tugas Praktikum (Kerjakan 1 dari 3 soal tugas berikut)

Pengumpulan
  1. Simpan file Python dengan nama yang sesuai
  2. Test program dan pastikan berjalan tanpa error
  3. Ambil screenshot pada setiap aplikasi yang dijalan
  4. Upload ke GitHub, buat repository dengan nama T2-Week4 (upload file .py dan screenshot aplikasi berjalan)
  5. Submit link GitHub ke platform Berajah
Tambahan
  1. Tambahkan komentar Nama, NIM, dan Kelas di awal file
  2. Pastikan kode rapi dan mudah dibaca

1: Kalkulator dengan Event Handling

Buatlah aplikasi kalkulator sederhana dengan fitur:

  1. Input angka pertama dan kedua menggunakan QLineEdit
  2. Pilihan operasi (+, -, ×, ÷) menggunakan QComboBox
  3. Validasi real-time bahwa input adalah angka
  4. Tombol "Hitung" yang hanya aktif jika input valid
  5. Keyboard shortcut: Enter untuk menghitung, Escape untuk clear
  6. Konfirmasi saat window akan ditutup

2: Drawing Canvas

Buatlah aplikasi canvas sederhana dengan fitur:

  1. Area kanvas yang bisa menggambar dengan mouse drag
  2. Pilihan warna menggunakan custom signal
  3. Pilihan ukuran brush dengan QSlider
  4. Tampilkan koordinat mouse di status bar
  5. Double click untuk clear canvas
  6. Tombol "Clear" dengan konfirmasi

3: Form Multi-Step

Buatlah form registrasi multi-step dengan fitur:

  1. Step 1: Data Pribadi (nama, tanggal lahir, jenis kelamin)
  2. Step 2: Kontak (email, telepon, alamat)
  3. Step 3: Akun (username, password, confirm password)
  4. Validasi real-time di setiap step
  5. Custom signal untuk notifikasi perpindahan step
  6. Progress indicator menggunakan signal
  7. Review data sebelum submit
