Packaging & Distribusi Aplikasi PySide6

Mengubah project Python menjadi aplikasi executable (.exe / .app / binary) yang siap didistribusikan

Capaian Pembelajaran

Setelah menyelesaikan materi ini, mahasiswa mampu:
  1. Menyiapkan project PySide6 untuk distribusi
  2. Menggunakan PyInstaller untuk membuat executable
  3. Menangani penyertaan file data (database, icon, resource)
  4. Membangun aplikasi untuk Windows, macOS, dan Linux
  5. Melakukan troubleshooting masalah packaging umum

1. Overview Distribusi Aplikasi

Saat kita menulis aplikasi Python, user harus menginstall Python dan semua dependency untuk menjalankannya. Ini tidak praktis untuk distribusi ke end-user. Solusinya adalah packaging — membungkus seluruh aplikasi beserta interpreter Python dan library ke dalam satu executable.

Dari Source Code ke Executable
Source Code
(.py files)
→
Dependencies
(PySide6, dll)
→
PyInstaller
(Bundling)
→
Executable
(.exe/.app)

Apa yang Dibundel?

Komponen Deskripsi Contoh
Python Interpreter Mesin Python itu sendiri python311.dll
Source Code File .py kita (dikompilasi ke .pyc) main.py → main.pyc
Dependencies Library pihak ketiga PySide6, sqlite3, joblib
Data Files File non-Python .db, .png, .ico, .pkl
Tanpa Packaging
  • User harus install Python
  • User harus pip install dependencies
  • Jalankan via command line
  • Rawan error versi library
  • Source code terlihat
Dengan Packaging
  • Double-click langsung jalan
  • Tidak perlu install apapun
  • Seperti aplikasi native
  • Versi library terkunci
  • Source code terlindungi

Output per Platform

Platform Output Distribusi
Windows .exe File .exe atau installer .msi / Inno Setup
macOS .app Bundle .app atau .dmg
Linux binary Binary executable, AppImage, atau .deb
Cross-Compilation Tidak Didukung

PyInstaller tidak bisa cross-compile. Untuk membuat .exe, Anda harus build di Windows. Untuk .app, build di macOS. Untuk binary Linux, build di Linux. Gunakan Virtual Machine atau CI/CD jika perlu build multi-platform.

File 1: database.py

database.py Database 
import sqlite3
import os
import sys

class DatabaseManager:
    """Manager untuk operasi database SQLite"""
    
    def __init__(self, db_name='mahasiswa.db'):
        # Penting: tentukan path database yang benar
        # Saat di-bundle, working directory bisa berbeda
        self.db_path = self.get_db_path(db_name)
        self.init_db()
    
    def get_db_path(self, db_name):
        """
        Menentukan path database yang benar.
        
        Saat dijalankan sebagai .exe (frozen), 
        file database disimpan di direktori yang sama 
        dengan executable agar bisa ditulis.
        """
        if getattr(sys, 'frozen', False):
            # Jika dijalankan sebagai executable (PyInstaller)
            app_dir = os.path.dirname(sys.executable)
        else:
            # Jika dijalankan sebagai script Python biasa
            app_dir = os.path.dirname(os.path.abspath(__file__))
        
        return os.path.join(app_dir, db_name)
    
    def get_connection(self):
        conn = sqlite3.connect(self.db_path)
        conn.row_factory = sqlite3.Row
        return conn
    
    def init_db(self):
        with self.get_connection() as conn:
            conn.execute('''
                CREATE TABLE IF NOT EXISTS mahasiswa (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    nim TEXT UNIQUE NOT NULL,
                    nama TEXT NOT NULL,
                    jurusan TEXT NOT NULL,
                    semester INTEGER DEFAULT 1
                )
            ''')
    
    def add(self, nim, nama, jurusan, semester):
        with self.get_connection() as conn:
            conn.execute(
                'INSERT INTO mahasiswa (nim, nama, jurusan, semester) VALUES (?,?,?,?)',
                (nim, nama, jurusan, semester)
            )
    
    def get_all(self):
        with self.get_connection() as conn:
            return conn.execute('SELECT * FROM mahasiswa ORDER BY nama').fetchall()
    
    def get_by_id(self, id):
        with self.get_connection() as conn:
            return conn.execute('SELECT * FROM mahasiswa WHERE id=?', (id,)).fetchone()
    
    def update(self, id, nim, nama, jurusan, semester):
        with self.get_connection() as conn:
            conn.execute(
                'UPDATE mahasiswa SET nim=?, nama=?, jurusan=?, semester=? WHERE id=?',
                (nim, nama, jurusan, semester, id)
            )
    
    def delete(self, id):
        with self.get_connection() as conn:
            conn.execute('DELETE FROM mahasiswa WHERE id=?', (id,))
    
    def search(self, keyword):
        with self.get_connection() as conn:
            return conn.execute(
                'SELECT * FROM mahasiswa WHERE nama LIKE ? OR nim LIKE ?',
                (f'%{keyword}%', f'%{keyword}%')
            ).fetchall()
PENTING: get_db_path()

Method get_db_path() adalah kunci agar database berfungsi setelah di-package. Saat aplikasi dijadikan .exe, working directory bisa berbeda dari lokasi .exe. Kita gunakan sys.frozen untuk mendeteksi apakah berjalan sebagai executable, lalu simpan database di folder yang sama dengan .exe.

class MahasiswaDialog(QDialog):
    def __init__(self, parent=None, data=None):
        super().__init__(parent)
        self.setWindowTitle("Edit Mahasiswa" if data else "Tambah Mahasiswa")
        self.setMinimumWidth(320)
        
        layout = QVBoxLayout(self)
        form = QFormLayout()
        
        self.nim_input = QLineEdit()
        self.nim_input.setPlaceholderText("8 digit angka")
        self.nim_input.setMaxLength(8)
        form.addRow("NIM:", self.nim_input)
        
        self.nama_input = QLineEdit()
        self.nama_input.setPlaceholderText("Nama lengkap")
        form.addRow("Nama:", self.nama_input)
        
        self.jurusan_combo = QComboBox()
        self.jurusan_combo.addItems([
            'Teknik Informatika', 'Sistem Informasi',
            'Teknik Komputer', 'Manajemen Informatika'
        ])
        form.addRow("Jurusan:", self.jurusan_combo)
        
        self.semester_spin = QSpinBox()
        self.semester_spin.setRange(1, 14)
        form.addRow("Semester:", self.semester_spin)
        
        layout.addLayout(form)
        
        buttons = QDialogButtonBox(QDialogButtonBox.Ok | QDialogButtonBox.Cancel)
        buttons.accepted.connect(self.validate_and_accept)
        buttons.rejected.connect(self.reject)
        layout.addWidget(buttons)
        
        if data:
            self.nim_input.setText(data['nim'])
            self.nama_input.setText(data['nama'])
            self.jurusan_combo.setCurrentText(data['jurusan'])
            self.semester_spin.setValue(data['semester'])
    
    def validate_and_accept(self):
        nim = self.nim_input.text().strip()
        nama = self.nama_input.text().strip()
        
        if not nim or not nim.isdigit() or len(nim) != 8:
            QMessageBox.warning(self, "Validasi", "NIM harus 8 digit angka!")
            self.nim_input.setFocus()
            return
        if not nama or len(nama) < 3:
            QMessageBox.warning(self, "Validasi", "Nama minimal 3 karakter!")
            self.nama_input.setFocus()
            return
        self.accept()
    
    def get_data(self):
        return {
            'nim': self.nim_input.text().strip(),
            'nama': self.nama_input.text().strip(),
            'jurusan': self.jurusan_combo.currentText(),
            'semester': self.semester_spin.value()
        }

class MainWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Sistem Data Mahasiswa v1.0")
        self.setGeometry(100, 100, 750, 500)
        
        self.db = DatabaseManager()
        self.create_actions()
        self.create_menus()
        self.create_toolbar()
        self.setup_ui()
        self.load_data()
    
    def create_actions(self):
        style = self.style()
        
        self.add_action = QAction("&Tambah", self)
        self.add_action.setIcon(style.standardIcon(QStyle.SP_FileIcon))
        self.add_action.setShortcut("Ctrl+N")
        self.add_action.triggered.connect(self.add_mahasiswa)
        
        self.edit_action = QAction("&Edit", self)
        self.edit_action.setShortcut("Ctrl+E")
        self.edit_action.triggered.connect(self.edit_mahasiswa)
        
        self.delete_action = QAction("&Hapus", self)
        self.delete_action.setIcon(style.standardIcon(QStyle.SP_TrashIcon))
        self.delete_action.setShortcut(QKeySequence.Delete)
        self.delete_action.triggered.connect(self.delete_mahasiswa)
        
        self.export_action = QAction("E&xport CSV", self)
        self.export_action.setIcon(style.standardIcon(QStyle.SP_DialogSaveButton))
        self.export_action.setShortcut("Ctrl+Shift+E")
        self.export_action.triggered.connect(self.export_csv)
        
        self.exit_action = QAction("E&xit", self)
        self.exit_action.setShortcut("Ctrl+Q")
        self.exit_action.triggered.connect(self.close)
    
    def create_menus(self):
        file_menu = self.menuBar().addMenu("&File")
        file_menu.addAction(self.add_action)
        file_menu.addAction(self.edit_action)
        file_menu.addAction(self.delete_action)
        file_menu.addSeparator()
        file_menu.addAction(self.export_action)
        file_menu.addSeparator()
        file_menu.addAction(self.exit_action)
    
    def create_toolbar(self):
        toolbar = self.addToolBar("Main")
        toolbar.addAction(self.add_action)
        toolbar.addAction(self.delete_action)
        toolbar.addSeparator()
        toolbar.addAction(self.export_action)
    
    def setup_ui(self):
        central = QWidget()
        self.setCentralWidget(central)
        layout = QVBoxLayout(central)
        
        # Search
        search_layout = QHBoxLayout()
        self.search_input = QLineEdit()
        self.search_input.setPlaceholderText("Cari nama atau NIM...")
        self.search_input.textChanged.connect(self.search_data)
        search_layout.addWidget(QLabel("Cari:"))
        search_layout.addWidget(self.search_input)
        layout.addLayout(search_layout)
        
        # Table
        self.table = QTableWidget()
        self.table.setColumnCount(5)
        self.table.setHorizontalHeaderLabels(['ID', 'NIM', 'Nama', 'Jurusan', 'Semester'])
        self.table.setSelectionBehavior(QTableWidget.SelectRows)
        self.table.setEditTriggers(QTableWidget.NoEditTriggers)
        self.table.horizontalHeader().setSectionResizeMode(QHeaderView.Stretch)
        self.table.setColumnWidth(0, 40)
        self.table.doubleClicked.connect(self.edit_mahasiswa)
        layout.addWidget(self.table)
        
        # Buttons
        btn_layout = QHBoxLayout()
        
        btn_add = QPushButton("Tambah")
        btn_add.clicked.connect(self.add_mahasiswa)
        btn_layout.addWidget(btn_add)
        
        btn_edit = QPushButton("Edit")
        btn_edit.clicked.connect(self.edit_mahasiswa)
        btn_layout.addWidget(btn_edit)
        
        btn_delete = QPushButton("Hapus")
        btn_delete.clicked.connect(self.delete_mahasiswa)
        btn_layout.addWidget(btn_delete)
        
        btn_refresh = QPushButton("Refresh")
        btn_refresh.clicked.connect(self.load_data)
        btn_layout.addWidget(btn_refresh)
        
        layout.addLayout(btn_layout)
        self.statusBar().showMessage("Ready")
    
    def load_data(self):
        data = self.db.get_all()
        self.populate_table(data)
    
    def populate_table(self, data):
        self.table.setRowCount(0)
        for row_data in data:
            row = self.table.rowCount()
            self.table.insertRow(row)
            self.table.setItem(row, 0, QTableWidgetItem(str(row_data['id'])))
            self.table.setItem(row, 1, QTableWidgetItem(row_data['nim']))
            self.table.setItem(row, 2, QTableWidgetItem(row_data['nama']))
            self.table.setItem(row, 3, QTableWidgetItem(row_data['jurusan']))
            self.table.setItem(row, 4, QTableWidgetItem(str(row_data['semester'])))
        self.statusBar().showMessage(f"Total: {len(data)} mahasiswa")
    
    def search_data(self):
        keyword = self.search_input.text().strip()
        data = self.db.search(keyword) if keyword else self.db.get_all()
        self.populate_table(data)
    
    def get_selected_id(self):
        row = self.table.currentRow()
        if row < 0:
            return None
        return int(self.table.item(row, 0).text())
    
    def add_mahasiswa(self):
        dialog = MahasiswaDialog(self)
        if dialog.exec() == QDialog.Accepted:
            d = dialog.get_data()
            try:
                self.db.add(d['nim'], d['nama'], d['jurusan'], d['semester'])
                self.load_data()
                QMessageBox.information(self, "Sukses", "Data berhasil ditambahkan!")
            except sqlite3.IntegrityError:
                QMessageBox.warning(self, "Error", "NIM sudah terdaftar!")
    
    def edit_mahasiswa(self):
        id = self.get_selected_id()
        if not id:
            QMessageBox.warning(self, "Peringatan", "Pilih data yang akan diedit!")
            return
        mhs = self.db.get_by_id(id)
        dialog = MahasiswaDialog(self, dict(mhs))
        if dialog.exec() == QDialog.Accepted:
            d = dialog.get_data()
            try:
                self.db.update(id, d['nim'], d['nama'], d['jurusan'], d['semester'])
                self.load_data()
                QMessageBox.information(self, "Sukses", "Data berhasil diupdate!")
            except sqlite3.IntegrityError:
                QMessageBox.warning(self, "Error", "NIM sudah digunakan!")
    
    def delete_mahasiswa(self):
        id = self.get_selected_id()
        if not id:
            QMessageBox.warning(self, "Peringatan", "Pilih data yang akan dihapus!")
            return
        reply = QMessageBox.question(self, "Konfirmasi", "Yakin ingin menghapus?",
            QMessageBox.Yes | QMessageBox.No)
        if reply == QMessageBox.Yes:
            self.db.delete(id)
            self.load_data()
    
    def export_csv(self):
        path, _ = QFileDialog.getSaveFileName(self, "Export CSV", "mahasiswa.csv", "CSV (*.csv)")
        if path:
            data = self.db.get_all()
            with open(path, 'w', newline='', encoding='utf-8') as f:
                writer = csv.writer(f)
                writer.writerow(['ID', 'NIM', 'Nama', 'Jurusan', 'Semester'])
                for row in data:
                    writer.writerow([row['id'], row['nim'], row['nama'], row['jurusan'], row['semester']])
            QMessageBox.information(self, "Sukses", f"Data di-export ke:\n{path}")


if __name__ == "__main__":

5. Mode One-File vs One-Folder

One-Folder (Default)
  • Banyak file dalam satu folder
  • Startup lebih cepat
  • Mudah update satu komponen
  • Cocok jika distribusi via folder/zip
One-File (--onefile)
  • Satu file .exe saja
  • Startup lebih lambat (extract ke temp)
  • Lebih mudah didistribusikan
  • Cocok untuk aplikasi kecil-sedang
Terminal - Build Commands Command 
# ===== BUILD ONE-FOLDER (Recommended untuk PySide6) =====
pyinstaller --windowed --name="SistemMahasiswa" --noconfirm main.py


# ===== BUILD ONE-FILE =====
pyinstaller --onefile --windowed --name="SistemMahasiswa" --noconfirm main.py


# ===== BUILD DENGAN ICON =====
pyinstaller --onefile --windowed --name="SistemMahasiswa" --icon=icon.ico --noconfirm main.py


# ===== BUILD LENGKAP (REKOMENDASI untuk project ini) =====
pyinstaller --windowed --name="SistemMahasiswa" --icon=icon.ico --noconfirm --clean main.py
One-File dan Database SQLite

Mode --onefile meng-extract file ke folder temporary saat dijalankan. File database tidak boleh di-bundle di dalam .exe karena akan hilang setelah ditutup. Itulah mengapa get_db_path() di database.py menyimpan database di lokasi executable, bukan di folder temporary.

7. Menyertakan Data Files

Jika aplikasi membutuhkan file tambahan (icon, gambar, template database, file .pkl), file tersebut harus disertakan secara eksplisit.

Kapan Menyertakan vs Tidak
  • Sertakan: Icon, gambar UI, template, file read-only
  • JANGAN sertakan: Database yang akan ditulis (buat saat runtime via init_db)
  • Sertakan jika read-only: File .pkl model ML

Cara Menyertakan File

Terminal - add-data Command 
# Windows: pemisah titik koma (;)
pyinstaller --windowed --add-data "icon.ico;." --add-data "templates;templates" main.py

# macOS / Linux: pemisah titik dua (:)
pyinstaller --windowed --add-data "icon.ico:." --add-data "templates:templates" main.py

# Format: "source_path;destination_path_in_bundle"
#   .  = root folder bundle
#   templates = folder 'templates' di dalam bundle

Mengakses Data Files di Kode

resource_path.py Helper 
import sys
import os

def resource_path(relative_path):
    """
    Mendapatkan path absolut ke resource file.
    
    Saat di-bundle oleh PyInstaller:
    - File yang di-add-data diekstrak ke sys._MEIPASS (folder temp)
    - Fungsi ini mengarahkan ke folder yang benar
    
    Saat development biasa:
    - Mengarah ke folder project
    """
    if getattr(sys, 'frozen', False):
        # Dijalankan sebagai executable PyInstaller
        base_path = sys._MEIPASS
    else:
        # Dijalankan sebagai script Python
        base_path = os.path.dirname(os.path.abspath(__file__))
    
    return os.path.join(base_path, relative_path)


# ===== Contoh Penggunaan =====

# Untuk icon yang di-bundle
icon_path = resource_path('icon.ico')

# Untuk model ML yang di-bundle
model_path = resource_path('models/iris_model.pkl')

# TAPI untuk database yang HARUS BISA DITULIS,
# gunakan get_db_path() dari database.py
# karena sys._MEIPASS adalah READ-ONLY
READ-ONLY vs WRITABLE
  • sys._MEIPASS (resource_path) → READ-ONLY. Untuk icon, gambar, file .pkl, template.
  • os.path.dirname(sys.executable) (get_db_path) → WRITABLE. Untuk database, log, user data.

Jangan simpan database di _MEIPASS karena folder itu dihapus saat aplikasi ditutup (mode --onefile)!

9. Build untuk macOS

macOS - Build Commands Terminal 
# Step 1: Buat virtual environment
python3 -m venv venv
source venv/bin/activate

# Step 2: Install dependencies
pip install PySide6 pyinstaller

# Step 3: Build .app bundle
pyinstaller --windowed --name="SistemMahasiswa" --noconfirm --clean main.py

# Hasilnya: dist/SistemMahasiswa.app

# Step 4: Test
open dist/SistemMahasiswa.app

# ===== Jika ingin single file =====
pyinstaller --onefile --windowed --name="SistemMahasiswa" --noconfirm main.py

# ===== Dengan icon .icns (macOS menggunakan .icns, bukan .ico) =====
pyinstaller --windowed --name="SistemMahasiswa" --icon=icon.icns --noconfirm main.py
macOS Spesifik
  • macOS menggunakan format icon .icns, bukan .ico
  • Hasil build adalah bundle .app (sebenarnya folder)
  • Untuk distribusi, bungkus dalam .dmg menggunakan tool seperti create-dmg
  • Mungkin perlu code signing untuk distribusi publik

10. Build untuk Linux

Linux - Build Commands Terminal 
# Step 1: Buat virtual environment
python3 -m venv venv
source venv/bin/activate

# Step 2: Install dependencies
pip install PySide6 pyinstaller

# Step 3: Build binary
pyinstaller --windowed --name="SistemMahasiswa" --noconfirm --clean main.py

# Hasilnya: dist/SistemMahasiswa/SistemMahasiswa (binary executable)

# Step 4: Test
./dist/SistemMahasiswa/SistemMahasiswa

# ===== Single file =====
pyinstaller --onefile --windowed --name="SistemMahasiswa" --noconfirm main.py

# Beri permission executable
chmod +x dist/SistemMahasiswa
Distribusi Linux
  • Sederhana: Tar.gz folder dist dan kirimkan
  • AppImage: Gunakan appimage-builder untuk portable app
  • .deb: Buat package Debian menggunakan fpm
  • Build di distro tertua yang ingin didukung (misal Ubuntu 20.04)

11. Troubleshooting

Masalah Penyebab Solusi
ModuleNotFoundError PyInstaller tidak mendeteksi import Tambahkan --hidden-import=module_name
Database error / file not found Path salah setelah bundling Gunakan get_db_path() dan resource_path()
.exe langsung tertutup Error tapi console tidak terlihat Build tanpa --windowed dulu untuk debug
Ukuran .exe terlalu besar Banyak library tidak terpakai Build di venv bersih, tambahkan --exclude-module
Antivirus memblokir .exe False positive dari packer Tambahkan exception, atau sign executable
PySide6 plugin error Plugin Qt tidak ter-bundle Tambahkan --hidden-import=PySide6.QtSvg

Tips Debug

Terminal - Debug Build Debug 
# Build TANPA --windowed untuk melihat error di console
pyinstaller --name="SistemMahasiswa" --noconfirm main.py

# Jalankan .exe dari command prompt untuk melihat output
dist\SistemMahasiswa\SistemMahasiswa.exe

# Setelah debug selesai, build ulang DENGAN --windowed
pyinstaller --windowed --name="SistemMahasiswa" --noconfirm main.py
Mengurangi Ukuran Build
  • Gunakan virtual environment bersih dengan hanya library yang dibutuhkan
  • Tambahkan --exclude-module matplotlib,numpy,scipy jika tidak dipakai
  • Install upx untuk kompresi otomatis
  • Pertimbangkan Nuitka jika ukuran sangat penting

12. Checklist Sebelum Distribusi

Checklist Distribusi
  1. Aplikasi berjalan normal saat python main.py
  2. Build berhasil tanpa error
  3. Executable berjalan di komputer lain (tanpa Python installed)
  4. Database terbuat otomatis dan bisa CRUD
  5. File data (icon, template) ter-load dengan benar
  6. Tidak ada console window muncul (--windowed)
  7. Icon aplikasi terlihat benar
  8. Export CSV berfungsi
  9. Tidak ada error saat close aplikasi
  10. Sertakan README untuk instruksi user

Tugas Praktikum

Tugas: Package Aplikasi Anda

Ambil salah satu project yang sudah dibuat di praktikum sebelumnya, kemudian package menjadi executable:

  1. Pastikan project menggunakan get_db_path() dan resource_path()
  2. Build menggunakan PyInstaller (mode one-folder)
  3. Test executable di komputer lain jika memungkinkan
  4. Buat file README.md berisi instruksi penggunaan
  5. Zip folder output dist/ dan kumpulkan
Ketentuan
  • Aplikasi harus berfungsi penuh tanpa install Python
  • Database harus terbuat otomatis saat pertama kali dijalankan
  • Tidak boleh ada console window muncul
  • Sertakan screenshot aplikasi yang berjalan sebagai .exe
  • Sertakan file .spec yang digunakan
