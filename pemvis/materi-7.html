Integrasi Machine Learning di Aplikasi Desktop PySide6

Membangun aplikasi prediksi menggunakan model scikit-learn (.pkl) dengan antarmuka PySide6

Capaian Pembelajaran

Setelah menyelesaikan materi ini, mahasiswa mampu:
  1. Membuat dan melatih model machine learning sederhana menggunakan scikit-learn
  2. Mengexport model ke file .pkl menggunakan joblib/pickle
  3. Memuat model .pkl di aplikasi PySide6
  4. Membangun UI interaktif untuk input data dan menampilkan hasil prediksi

1. Overview & Arsitektur

Dalam praktikum ini, kita akan membangun aplikasi desktop yang mampu melakukan prediksi menggunakan model machine learning. Alurnya terbagi menjadi dua fase: Training (membuat model) dan Deployment (menggunakan model di aplikasi).

Apa itu File .pkl?

File .pkl (Pickle)

Pickle adalah format serialisasi bawaan Python yang mengubah objek Python (termasuk model ML) menjadi byte stream yang bisa disimpan ke file. File .pkl menyimpan seluruh state model yang sudah dilatih, sehingga bisa langsung digunakan untuk prediksi tanpa perlu training ulang.

Tanpa .pkl (Training Ulang)
  • Training setiap kali aplikasi dibuka
  • Butuh dataset lengkap
  • Lambat saat startup
  • Memakan resource besar
Dengan .pkl (Load Model)
  • Training hanya sekali
  • Cukup load file .pkl kecil
  • Startup cepat (hitungan detik)
  • Resource minimal

Jenis Model yang Akan Dibuat

Jenis Tujuan Output Contoh
Klasifikasi Memprediksi kategori/label Kelas diskrit Jenis bunga, spam/not spam
Regresi Memprediksi nilai numerik Angka kontinu Harga rumah, suhu

from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, classification_report
import joblib

# ===== STEP 1: Load Dataset =====
iris = load_iris()
X = iris.data           # Fitur (150 x 4)
y = iris.target         # Target/Label (150,)

print(f"Dataset shape: {X.shape}")
print(f"Target names : {iris.target_names}")


# ===== STEP 2: Split Train/Test =====
# 80% untuk training, 20% untuk testing
X_train, X_test, y_train, y_test = train_test_split(
    X, y, 
    test_size=0.2,       # 20% untuk test
    random_state=42      # Agar hasil reproducible
)

print(f"\nTrain set: {X_train.shape[0]} sampel")
print(f"Test set : {X_test.shape[0]} sampel")


# ===== STEP 3: Buat dan Latih Model =====
model = RandomForestClassifier(
    n_estimators=100,     # Jumlah decision tree
    random_state=42
)

# Training model dengan data training
model.fit(X_train, y_train)

print("\nModel berhasil dilatih!")


# ===== STEP 4: Evaluasi Model =====
y_pred = model.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)

print(f"\nAkurasi: {accuracy * 100:.2f}%")
print("\nClassification Report:")
print(classification_report(y_test, y_pred, target_names=iris.target_names))


# ===== STEP 5: Simpan Model ke .pkl =====
joblib.dump(model, 'iris_model.pkl')
print("Model disimpan ke 'iris_model.pkl'")


# ===== STEP 6: Test Load Model =====
loaded_model = joblib.load('iris_model.pkl')

# Test prediksi dengan data contoh
sample = [[5.1, 3.5, 1.4, 0.2]]  # Seharusnya Setosa

Training Model Regresi (Harga Rumah)

train_housing_model.py Training 
from sklearn.datasets import fetch_california_housing
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import mean_absolute_error, r2_score
import joblib
import numpy as np

# ===== STEP 1: Load Dataset =====
housing = fetch_california_housing()
X = housing.data
y = housing.target  # Harga dalam $100,000

print(f"Dataset shape: {X.shape}")
print(f"Fitur: {housing.feature_names}")


# ===== STEP 2: Preprocessing - Normalisasi =====
# Scaler penting untuk beberapa algoritma
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)


# ===== STEP 3: Split Train/Test =====
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, random_state=42
)


# ===== STEP 4: Training =====
model = RandomForestRegressor(
    n_estimators=100,
    random_state=42,
    n_jobs=-1   # Gunakan semua CPU core
)
model.fit(X_train, y_train)

print("Model berhasil dilatih!")


# ===== STEP 5: Evaluasi =====
y_pred = model.predict(X_test)

mae = mean_absolute_error(y_test, y_pred)
r2 = r2_score(y_test, y_pred)

print(f"\nMean Absolute Error: ${mae * 100000:,.0f}")
print(f"R² Score: {r2:.4f}")


# ===== STEP 6: Simpan Model DAN Scaler =====
# PENTING: Simpan scaler juga karena input harus di-scale
joblib.dump(model, 'housing_model.pkl')
joblib.dump(scaler, 'housing_scaler.pkl')

print("\nModel disimpan ke 'housing_model.pkl'")
print("Scaler disimpan ke 'housing_scaler.pkl'")
Jangan Lupa Simpan Scaler!

Jika menggunakan StandardScaler atau preprocessing lain saat training, scaler tersebut juga harus disimpan ke .pkl dan di-load bersamaan dengan model. Input user di aplikasi harus melalui scaler yang sama sebelum diprediksi.

5. Export & Load File .pkl

Ada dua cara utama untuk menyimpan dan memuat model: joblib dan pickle.

save_load_model.py Serialisasi 
import joblib
import pickle

# ==========================================
# CARA 1: joblib (DIREKOMENDASIKAN)
# ==========================================

# Simpan model
joblib.dump(model, 'model.pkl')

# Load model
loaded_model = joblib.load('model.pkl')

# Langsung bisa dipakai prediksi
result = loaded_model.predict(data)


# ==========================================
# CARA 2: pickle (Bawaan Python)
# ==========================================

# Simpan model
with open('model.pkl', 'wb') as f:
    pickle.dump(model, f)

# Load model
with open('model.pkl', 'rb') as f:
    loaded_model = pickle.load(f)


# ==========================================
# SIMPAN MULTIPLE OBJECTS
# ==========================================

# Simpan model + scaler + metadata dalam satu file
model_package = {
    'model': model,
    'scaler': scaler,
    'feature_names': ['feature1', 'feature2'],
    'target_names': ['class_a', 'class_b'],
    'version': '1.0'
}

joblib.dump(model_package, 'model_package.pkl')

# Load
package = joblib.load('model_package.pkl')
model = package['model']
scaler = package['scaler']
Tips: Simpan Sebagai Package

Simpan model, scaler, feature names, dan metadata lain dalam satu dictionary, lalu dump ke satu file .pkl. Ini memudahkan distribusi dan memastikan semua komponen kompatibel.

6. Load Model di PySide6

Berikut cara memuat model .pkl dan menggunakannya di dalam aplikasi PySide6:

model_handler.py Handler 
import joblib
import numpy as np
import os

class ModelHandler:
    """
    Class untuk mengelola model ML.
    Memisahkan logic ML dari UI (Separation of Concerns).
    """
    
    def __init__(self):
        self.model = None
        self.scaler = None
        self.feature_names = []
        self.target_names = []
        self.is_loaded = False
    
    def load_model(self, model_path, scaler_path=None):
        """Load model dan opsional scaler dari file .pkl"""
        try:
            if not os.path.exists(model_path):
                raise FileNotFoundError(f"File tidak ditemukan: {model_path}")
            
            self.model = joblib.load(model_path)
            
            if scaler_path and os.path.exists(scaler_path):
                self.scaler = joblib.load(scaler_path)
            
            self.is_loaded = True
            return True, "Model berhasil dimuat"
        
        except Exception as e:
            self.is_loaded = False
            return False, f"Gagal memuat model: {str(e)}"
    
    def predict(self, features):
        """
        Melakukan prediksi.
        
        Args:
            features: list of float, contoh [5.1, 3.5, 1.4, 0.2]
        
        Returns:
            prediction result
        """
        if not self.is_loaded:
            raise RuntimeError("Model belum dimuat!")
        
        # Konversi ke numpy array 2D (model butuh 2D input)
        X = np.array(features).reshape(1, -1)
        
        # Apply scaler jika ada
        if self.scaler:
            X = self.scaler.transform(X)
        
        # Prediksi
        prediction = self.model.predict(X)
        
        return prediction[0]
    
    def predict_proba(self, features):
        """Prediksi dengan probabilitas (hanya untuk klasifikasi)"""
        if not self.is_loaded:
            raise RuntimeError("Model belum dimuat!")
        
        X = np.array(features).reshape(1, -1)
        
        if self.scaler:
            X = self.scaler.transform(X)
        
        if hasattr(self.model, 'predict_proba'):
            return self.model.predict_proba(X)[0]
        
        return None
1

np.array(features).reshape(1, -1)

Model scikit-learn menerima input 2D. reshape(1, -1) mengubah array 1D [5.1, 3.5, 1.4, 0.2] menjadi 2D [[5.1, 3.5, 1.4, 0.2]] (1 sampel, N fitur).

2

scaler.transform(X)

Jika saat training menggunakan scaler, input prediksi juga harus di-transform dengan scaler yang sama. Gunakan transform(), bukan fit_transform().

3

model.predict(X)

Mengembalikan array prediksi. Untuk 1 sampel, hasilnya [prediction], jadi kita ambil index 0.

7. UI Prediksi Sederhana

Contoh paling minimal untuk memuat model dan melakukan prediksi dari PySide6:

simple_prediction_ui.py Simple 
import sys
import joblib
import numpy as np
from PySide6.QtWidgets import (
    QApplication, QWidget, QVBoxLayout, QHBoxLayout,
    QPushButton, QLabel, QDoubleSpinBox, QFormLayout, QMessageBox
)

class SimplePredictionApp(QWidget):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("Prediksi Iris - Sederhana")
        self.setMinimumWidth(350)
        
        # Load model
        try:
            self.model = joblib.load('iris_model.pkl')
            self.target_names = ['Setosa', 'Versicolor', 'Virginica']
        except Exception as e:
            QMessageBox.critical(None, "Error", f"Gagal load model:\n{e}")
            sys.exit(1)
        
        self.setup_ui()
    
    def setup_ui(self):
        layout = QVBoxLayout(self)
        form = QFormLayout()
        
        # 4 input untuk 4 fitur iris
        self.sepal_length = QDoubleSpinBox()
        self.sepal_length.setRange(0.0, 10.0)
        self.sepal_length.setValue(5.1)
        self.sepal_length.setSingleStep(0.1)
        form.addRow("Sepal Length (cm):", self.sepal_length)
        
        self.sepal_width = QDoubleSpinBox()
        self.sepal_width.setRange(0.0, 10.0)
        self.sepal_width.setValue(3.5)
        self.sepal_width.setSingleStep(0.1)
        form.addRow("Sepal Width (cm):", self.sepal_width)
        
        self.petal_length = QDoubleSpinBox()
        self.petal_length.setRange(0.0, 10.0)
        self.petal_length.setValue(1.4)
        self.petal_length.setSingleStep(0.1)
        form.addRow("Petal Length (cm):", self.petal_length)
        
        self.petal_width = QDoubleSpinBox()
        self.petal_width.setRange(0.0, 10.0)
        self.petal_width.setValue(0.2)
        self.petal_width.setSingleStep(0.1)
        form.addRow("Petal Width (cm):", self.petal_width)
        
        layout.addLayout(form)
        
        # Tombol prediksi
        self.predict_btn = QPushButton("Prediksi!")
        self.predict_btn.clicked.connect(self.do_predict)
        self.predict_btn.setStyleSheet("""
            QPushButton {
                background-color: #3498db; color: white;
                padding: 12px; border: none; border-radius: 6px;
                font-size: 14px; font-weight: bold;
            }
            QPushButton:hover { background-color: #2980b9; }
        """)
        layout.addWidget(self.predict_btn)
        
        # Label hasil
        self.result_label = QLabel("Hasil prediksi akan muncul di sini")
        self.result_label.setStyleSheet("""
            QLabel {
                background: #f0f0f0; padding: 20px;
                border-radius: 8px; font-size: 16px;
                text-align: center;
            }
        """)
        self.result_label.setWordWrap(True)
        layout.addWidget(self.result_label)
    
    def do_predict(self):
        # Ambil nilai dari input
        features = [
            self.sepal_length.value(),
            self.sepal_width.value(),
            self.petal_length.value(),
            self.petal_width.value()
        ]
        
        # Prediksi
        X = np.array(features).reshape(1, -1)
        prediction = self.model.predict(X)[0]
        probabilities = self.model.predict_proba(X)[0]
        
        species = self.target_names[prediction]
        confidence = probabilities[prediction] * 100
        
        # Tampilkan hasil
        result_text = f"Spesies: {species}\n"
        result_text += f"Confidence: {confidence:.1f}%\n\n"
        
        for i, name in enumerate(self.target_names):
            result_text += f"  {name}: {probabilities[i]*100:.1f}%\n"
        
        self.result_label.setText(result_text)
        
        # Warna berdasarkan confidence
        if confidence > 90:
            color = "#27ae60"
        elif confidence > 70:
            color = "#f39c12"
        else:
            color = "#e74c3c"
        
        self.result_label.setStyleSheet(f"""
            QLabel {{
                background: #f0f0f0; padding: 20px;
                border-radius: 8px; font-size: 14px;
                border-left: 5px solid {color};
            }}
        """)


if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = SimplePredictionApp()
    window.show()
    sys.exit(app.exec())

        self.history_table.setHorizontalHeaderLabels([
            'SL', 'SW', 'PL', 'PW', 'Prediksi', 'Confidence'
        ])
        self.history_table.setEditTriggers(QTableWidget.NoEditTriggers)
        self.history_table.horizontalHeader().setSectionResizeMode(QHeaderView.Stretch)
        right_layout.addWidget(self.history_table)
        
        splitter.addWidget(right)
        splitter.setSizes([300, 500])
        
        main_layout.addWidget(splitter)
    
    def do_predict(self):
        # Ambil input
        features = [
            self.inputs['sepal_length'].value(),
            self.inputs['sepal_width'].value(),
            self.inputs['petal_length'].value(),
            self.inputs['petal_width'].value(),
        ]
        
        # Prediksi
        X = np.array(features).reshape(1, -1)
        prediction = self.model.predict(X)[0]
        proba = self.model.predict_proba(X)[0]
        
        species = self.target_names[prediction]
        confidence = proba[prediction] * 100
        
        # Update UI
        self.result_species.setText(f"Spesies: {species}")
        
        detail = f"Confidence: {confidence:.1f}%\n\n"
        for i, name in enumerate(self.target_names):
            bar = '█' * int(proba[i] * 20)
            detail += f"{name:12}: {proba[i]*100:5.1f}% {bar}\n"
        self.result_detail.setText(detail)
        
        # Simpan ke history
        self.add_to_history(features, species, confidence)
        
        self.statusBar().showMessage(
            f"Prediksi: {species} ({confidence:.1f}%)", 5000
        )
    
    def add_to_history(self, features, species, confidence):
        row = self.history_table.rowCount()
        self.history_table.insertRow(row)
        
        for col, val in enumerate(features):
            self.history_table.setItem(row, col, QTableWidgetItem(f"{val:.1f}"))
        
        self.history_table.setItem(row, 4, QTableWidgetItem(species))
        self.history_table.setItem(row, 5, QTableWidgetItem(f"{confidence:.1f}%"))
        
        self.history.append(features + [species, f"{confidence:.1f}%"])
        self.history_table.scrollToBottom()
    
    def export_csv(self):
        if not self.history:
            QMessageBox.warning(self, "Kosong", "Belum ada data untuk di-export.")
            return
        
        path, _ = QFileDialog.getSaveFileName(self, "Export CSV", "predictions.csv", "CSV (*.csv)")
        if path:
            with open(path, 'w', newline='') as f:
                writer = csv.writer(f)
                writer.writerow(['Sepal L', 'Sepal W', 'Petal L', 'Petal W', 'Species', 'Confidence'])
                writer.writerows(self.history)
            QMessageBox.information(self, "Sukses", f"Data di-export ke:\n{path}")
    
    def clear_history(self):
        self.history.clear()
        self.history_table.setRowCount(0)
        self.statusBar().showMessage("History cleared", 3000)

Checklist Best Practices
  • Pisahkan training dan inference. Script training jangan digabung dengan aplikasi.
  • Simpan scaler bersama model. Jika preprocessing digunakan, simpan juga scaler-nya.
  • Cek file .pkl saat startup. Beri pesan error yang jelas jika file tidak ditemukan.
  • Gunakan ModelHandler terpisah. Jangan masukkan logic ML langsung di UI class.
  • Validasi input sebelum prediksi. Pastikan nilai input berada dalam range yang wajar.
  • Tampilkan confidence/probability. User perlu tahu seberapa yakin model.
  • Gunakan QDoubleSpinBox. Lebih mudah mengontrol input numerik daripada QLineEdit.
  • Gunakan threading untuk model besar. Jika model lambat, load dan prediksi di thread terpisah.
Kesalahan Umum yang Harus Dihindari
  • Lupa reshape input: Selalu gunakan .reshape(1, -1) untuk single prediction
  • Lupa scale input: Jika training pakai scaler, prediksi juga harus pakai scaler yang sama
  • Pakai fit_transform saat prediksi: Gunakan transform() saja, bukan fit_transform()
  • Versi library berbeda: Pastikan versi scikit-learn saat training dan inference sama
  • Tidak handle FileNotFoundError: Selalu cek apakah file .pkl ada sebelum load

Tugas Praktikum

Tugas: Aplikasi Prediksi ML

Pilih SATU dari tugas berikut:

  1. Klasifikasi Wine: Gunakan dataset load_wine() dari scikit-learn. Buat model untuk memprediksi kelas wine berdasarkan komposisi kimianya. Buat UI input dan tampilkan hasil + probabilitas.
  2. Klasifikasi Diabetes: Gunakan dataset load_diabetes(). Buat model regresi untuk memprediksi progression diabetes. Tampilkan prediksi nilai dan kategori resiko (rendah/sedang/tinggi).
  3. Dataset Sendiri: Buat/cari dataset CSV sendiri (minimal 100 baris, 3 fitur). Training model, export .pkl, buat aplikasi PySide6 untuk prediksi.
Ketentuan Pengumpulan
  • Sertakan script training (train_model.py) dan aplikasi (main.py)
  • Sertakan file .pkl hasil training
  • ModelHandler terpisah dari UI
  • Tampilkan confidence/probability
  • Ada validasi input
  • Sertakan README.md dengan instruksi menjalankan
  • Kumpulkan dalam format ZIP
