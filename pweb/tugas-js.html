Tugas 3A: Kalkulator Sederhana

Buat aplikasi kalkulator dengan fitur:

  • Input 2 angka
  • Operasi: Tambah, Kurang, Kali, Bagi, Modulus, Pangkat
  • Tampilkan hasil di halaman
  • Validasi input (tidak boleh kosong, harus angka)
  • Untuk pembagian, tampilkan error jika dibagi 0
  • Bonus: Tambahkan history perhitungan
Hint Struktur:
// HTML
<input type="number" id="angka1">
<select id="operator">...</select>
<input type="number" id="angka2">
<button onclick="hitung()">Hitung</button>
<div id="hasil"></div>

// JavaScript
function hitung() {
    let a = Number(document.getElementById('angka1').value);
    let b = Number(document.getElementById('angka2').value);
    let op = document.getElementById('operator').value;
    let hasil;
    
    // Gunakan switch-case untuk operasi
    switch(op) {
        case '+': hasil = a + b; break;
        // ... dst
    }
    
    document.getElementById('hasil').innerHTML = hasil;
}

📝 Tugas 3B: To-Do List

Buat aplikasi To-Do List dengan fitur:

  • Input untuk menambah task baru
  • Tombol untuk menambah task ke daftar
  • Setiap task memiliki tombol "Selesai" dan "Hapus"
  • Task yang selesai diberi style berbeda (coret/warna berbeda)
  • Validasi: task tidak boleh kosong
  • Bonus: Simpan ke localStorage agar tidak hilang saat refresh
💡 Hint Struktur:
// HTML
<input type="text" id="taskInput" placeholder="Tambah task...">
<button onclick="tambahTask()">Tambah</button>
<ul id="taskList"></ul>

// JavaScript
let tasks = [];

function tambahTask() {
    let task = document.getElementById('taskInput').value;
    
    if (task.trim() === '') {
        alert('Task tidak boleh kosong!');
        return;
    }
    
    tasks.push({ text: task, done: false });
    renderTasks();
}

function renderTasks() {
    let html = '';
    tasks.forEach((task, index) => {
        html += `<li class="${task.done ? 'done' : ''}">
            ${task.text}
            <button onclick="toggleDone(${index})">✓</button>
            <button onclick="hapusTask(${index})">✗</button>
        </li>`;
    });
    document.getElementById('taskList').innerHTML = html;
}

Tugas 3C: Form Validasi & Tabel Data

Buat form pendaftaran dengan validasi dan tampilkan data ke tabel:

  • Form input: Nama, Email, Telepon, Tanggal Lahir, Jenis Kelamin
  • Validasi:
    • Nama minimal 3 karakter
    • Email harus valid (mengandung @ dan .)
    • Telepon hanya angka, minimal 10 digit
    • Semua field wajib diisi
  • Jika valid, data ditambahkan ke tabel di bawah form
  • Setiap baris tabel ada tombol "Hapus"
  • Bonus: Gunakan Class untuk menyimpan data pendaftar
💡 Hint Struktur:
// Class untuk data
class Pendaftar {
    constructor(nama, email, telepon, tglLahir, gender) {
        this.id = Date.now();
        this.nama = nama;
        this.email = email;
        this.telepon = telepon;
        this.tglLahir = tglLahir;
        this.gender = gender;
    }
}

let dataPendaftar = [];

function validasiForm() {
