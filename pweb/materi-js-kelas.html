2. Alur Mengajar di Kelas Struktur

Tahap 1: Pembukaan dan konteks

Durasi: 10 menit

Mulai dengan pertanyaan sederhana: “Kalau HTML membuat struktur dan CSS membuat tampilan, lalu siapa yang membuat tombol bisa bekerja?” Dari sini arahkan mahasiswa ke peran JavaScript sebagai pemberi logika dan interaksi.

Fokus dosen: bangun gambaran besar dulu, jangan langsung masuk sintaks panjang.

Tahap 2: Sintaks paling dasar

Durasi: 15 menit

Kenalkan alert(), console.log(), variabel let, dan tipe data sederhana. Pastikan mahasiswa paham bahwa JavaScript dibaca baris demi baris dan hasil bisa dilihat di browser console.

Fokus dosen: cukup satu contoh kecil, lalu minta mahasiswa memodifikasi nilainya.

Tahap 3: Logika program

Durasi: 20 menit

Lanjut ke percabangan if...else dan function. Jelaskan bahwa program mulai terasa berguna saat bisa mengambil keputusan dari input pengguna.

Fokus dosen: gunakan contoh kehidupan kampus seperti nilai, absensi, atau diskon kantin.

Tahap 4: JavaScript bertemu HTML

Durasi: 20 menit

Tunjukkan bagaimana JS mengambil elemen dengan document.getElementById(), membaca nilai input, lalu menampilkan hasil ke halaman.

Fokus dosen: tekankan alur input → proses → output.

Tahap 5: Latihan mandiri singkat

Durasi: 20 menit

Berikan kasus kecil yang masih satu pola dengan contoh. Jangan langsung lompat ke proyek besar. Tujuannya agar mahasiswa berhasil dulu, lalu percaya diri.

Fokus dosen: cek pemahaman logika, bukan sekadar hasil akhir.

6. Looping untuk Melatih Logika Penting

Looping adalah proses menjalankan perintah yang sama berulang kali selama syaratnya masih terpenuhi. Materi ini penting karena mahasiswa mulai belajar berpikir terstruktur: berapa kali sesuatu diulang, kapan berhenti, dan bagaimana hasil tiap langkah disusun.

Yang perlu ditekankan saat mengajar looping:
  • for cocok saat jumlah perulangan sudah diketahui
  • while cocok saat perulangan bergantung pada kondisi
  • Looping bukan hanya mengulang, tapi juga melatih urutan logika langkah demi langkah

Contoh 1: Hitungan Mundur

Contoh ini cocok untuk mengenalkan arah perulangan dari besar ke kecil. Hasil ditampilkan ke halaman dan ke console.

Yang dijelaskan dosen: nilai awal i = 5, syarat berhenti i >= 1, dan perubahan nilai i--.

Contoh 2: Membuat Pola Bintang

Contoh ini sangat baik untuk melatih logika karena mahasiswa harus memahami perulangan di dalam perulangan.

Yang dijelaskan dosen: loop luar mengatur jumlah baris, loop dalam mengatur jumlah bintang pada tiap baris.

Contoh 3: Looping dengan While

Contoh ini dipakai untuk menunjukkan bahwa perulangan juga bisa ditulis dengan while, terutama saat kita lebih fokus pada kondisi berhentinya.

Yang dijelaskan dosen: selama kondisi angka <= 5 masih benar, blok kode akan terus berjalan. Jangan lupa update nilai angka++ agar loop berhenti.

Pertanyaan Kasus untuk Mahasiswa

  1. Buat program hitungan mundur dari 10 sampai 1, lalu tampilkan teks Selesai di akhir.
  2. Buat pola bintang 5 baris seperti segitiga, lalu ubah menjadi 7 baris.
  3. Buat pola angka: baris pertama 1, baris kedua 12, baris ketiga 123, dan seterusnya.
  4. Buat perulangan yang menampilkan bilangan genap dari 2 sampai 20.
  5. Buat program yang menghitung total jumlah angka dari 1 sampai 10.
  6. Buat ulang contoh angka 1 sampai 5 menggunakan while, lalu ubah menjadi 1 sampai 10.
Saran mengajar: minta mahasiswa menjelaskan dulu tiga bagian loop, yaitu nilai awal, kondisi, dan perubahan. Kalau tiga bagian ini dipahami, logika looping biasanya lebih cepat masuk.

7. Latihan untuk Mahasiswa Coba Sendiri

Berikut beberapa perintah kasus singkat yang bisa langsung diberikan di kelas. Polanya dibuat bertahap dari mudah ke menengah.

Latihan A

Buat variabel nama, nim, dan jurusan, lalu tampilkan ke console.

Latihan B

Buat program yang mengecek apakah angka bernilai genap atau ganjil.

Latihan C

Buat function hitungDiskon() yang menerima harga dan persen diskon.

Latihan D

Buat input nilai tugas dan tampilkan pesan Lulus atau Remedial.

Contoh perintah yang bisa dibacakan dosen

  1. “Buat variabel untuk menyimpan nama kalian, lalu tampilkan dengan console.log().”
  2. “Ubah program agar jika nilai di bawah 75, tampil pesan remedial.”
  3. “Buat tombol hitung yang menjumlahkan dua angka dari input HTML.”
  4. “Tambahkan validasi: jika input kosong, tampilkan pesan error.”
  5. “Modifikasi kasus belanja agar ada potongan 10% jika total lebih dari 50000.”
Saran mengajar: setelah memberi satu latihan, jangan langsung tampilkan jawaban penuh. Minta 2-3 mahasiswa menjelaskan alur logikanya lebih dulu.

8. Penutup Kelas Ringkasan

Urutan yang paling aman untuk materi JavaScript pemula adalah: pahami fungsi JS, lanjut ke variabel dan output, masuk ke kondisi dan function, lalu hubungkan ke HTML/DOM. Dengan urutan ini, mahasiswa tidak langsung kewalahan.

Target akhir pertemuan: mahasiswa mampu membuat program kecil berbasis input, memproses data dengan logika sederhana, lalu menampilkan hasil ke halaman web.

3. Apa Itu Console Penting

Console adalah tempat untuk melihat pesan dari JavaScript. Saat kita menulis console.log(), hasilnya tidak muncul di halaman web, tetapi muncul di console browser. Ini sangat penting untuk belajar JavaScript karena mahasiswa bisa melihat nilai variabel, hasil perhitungan, dan error program.

Fungsi console dalam pembelajaran:
  • Melihat hasil program dengan cepat tanpa harus membuat tampilan HTML yang rumit
  • Mengecek isi variabel saat program berjalan
  • Membantu mencari letak kesalahan kode

Cara membuka console di browser

Browser Cara Cepat
Google Chrome Tekan F12 atau Ctrl + Shift + I, lalu pilih tab Console
Microsoft Edge Tekan F12 atau Ctrl + Shift + I, lalu pilih tab Console
Firefox Tekan F12 atau Ctrl + Shift + K
Jelaskan ke mahasiswa: output di halaman web ditujukan untuk pengguna, sedangkan output di console ditujukan untuk programmer saat mengecek proses program.
