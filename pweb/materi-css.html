1. Pengertian CSS CPMK 3.1

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web yang ditulis dalam HTML. CSS memisahkan konten (HTML) dari presentasi visual (desain).

Kata Kunci dalam CSS:
  • Cascading - Aturan CSS diterapkan secara bertingkat/berurutan
  • Style - Mengatur tampilan visual elemen
  • Sheets - Kumpulan aturan yang disusun dalam file terpisah

Mengapa Menggunakan CSS?

Tanpa CSS Dengan CSS
Tampilan polos dan membosankan Tampilan menarik dan profesional
Styling inline di setiap elemen Styling terpusat dan terorganisir
Sulit di-maintain Mudah diubah dan dipelihara
Tidak responsive Dapat responsive di berbagai device
Kode HTML panjang dan berantakan Kode HTML bersih dan rapi

Sejarah CSS

Versi Tahun Fitur Utama
CSS 1 1996 Font, warna, margin, padding dasar
CSS 2 1998 Positioning, z-index, media types
CSS 2.1 2011 Perbaikan bug dan standarisasi
CSS 3 2011+ Flexbox, Grid, Animation, Transform, dll
Keuntungan CSS:
  • Memisahkan konten dari desain
  • Satu file CSS untuk banyak halaman HTML
  • Memudahkan maintenance dan update
  • Meningkatkan kecepatan loading (caching)
  • Mendukung responsive design

2. Cara Penulisan CSS CPMK 3.2

Struktur Dasar CSS

/* Struktur penulisan CSS */
selector {
    property: value;
    property2: value2;
}

/* Contoh nyata */
h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}
Komponen CSS:
  • Selector - Menentukan elemen HTML yang akan di-style
  • Property - Jenis style yang akan diubah
  • Value - Nilai dari property tersebut
  • Declaration - Pasangan property:value

3 Cara Menyisipkan CSS

1️⃣ Inline CSS (Langsung di tag HTML)

<!-- CSS langsung di atribut style -->
<h1 style="color: red; font-size: 30px;">Judul</h1>
<p style="background-color: yellow;">Paragraf</p>
Kelemahan Inline CSS:
  • Tidak efisien (harus ditulis berulang)
  • Sulit di-maintain
  • Mencampur konten dengan presentasi
  • Hanya untuk perubahan cepat/debugging

2️⃣ Internal CSS (Di dalam tag <style>)

<!DOCTYPE html>
<html>
<head>
    <title>Internal CSS</title>
    <style>
        /* CSS ditulis di sini */
        body {
            background-color: #f5f5f5;
            font-family: Arial, sans-serif;
        }
        
        h1 {
            color: #333;
            text-align: center;
        }
        
        p {
            line-height: 1.6;
            color: #666;
        }
    </style>
</head>
<body>
    <h1>Judul Halaman</h1>
    <p>Ini adalah paragraf.</p>
</body>
</html>

3️⃣ External CSS (File terpisah) ⭐ DIREKOMENDASIKAN

File: style.css

/* File: style.css */

/* Reset dasar */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* Body styling */
body {
    font-family: 'Segoe UI', sans-serif;
    background-color: #f5f5f5;
    color: #333;
}

/* Heading */
h1 {
    font-size: 2.5em;
    color: #2c3e50;
    margin-bottom: 20px;
}

/* Paragraf */
p {
    line-height: 1.8;
    margin-bottom: 15px;
}

File: index.html

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>External CSS</title>
    
    <!-- Link ke file CSS eksternal -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Judul Halaman</h1>
    <p>Konten halaman...</p>
</body>
</html>
Keuntungan External CSS:
  • Satu file CSS untuk banyak halaman HTML
  • Mudah di-maintain dan diorganisir
  • Browser dapat meng-cache file CSS
  • Memisahkan struktur (HTML) dari presentasi (CSS)

3. Selector CSS CPMK 3.2

Selector digunakan untuk memilih elemen HTML yang akan diberi style.

Jenis-jenis Selector

1. Element Selector

/* Memilih semua elemen dengan tag tertentu */
h1 {
    color: blue;
}

p {
    font-size: 16px;
}

a {
    text-decoration: none;
}

2. Class Selector (.)

/* Memilih elemen dengan class tertentu */
.highlight {
    background-color: yellow;
}

.btn-primary {
    background: #3498db;
    color: white;
    padding: 10px 20px;
}

/* HTML: <p class="highlight">Text</p> */

3. ID Selector (#)

/* Memilih elemen dengan ID tertentu (unik) */
#header {
    background: #333;
    color: white;
}

#main-content {
    max-width: 1200px;
    margin: 0 auto;
}

/* HTML: <div id="header">...</div> */

4. Universal Selector (*)

/* Memilih SEMUA elemen */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

5. Grouping Selector (,)

/* Menggabungkan beberapa selector */
h1, h2, h3, h4, h5, h6 {
    font-family: 'Georgia', serif;
    color: #2c3e50;
}

.btn, .link, .nav-item {
    cursor: pointer;
}

6. Descendant Selector (spasi)

/* Memilih elemen di dalam elemen lain */
nav a {
    color: white;
}

.container p {
    line-height: 1.8;
}

ul li a {
    text-decoration: none;
}

7. Child Selector (>)

/* Memilih anak langsung */
ul > li {
    list-style: none;
}

.parent > .child {
    padding: 10px;
}

8. Adjacent Sibling Selector (+)

/* Memilih elemen tepat setelahnya */
h1 + p {
    font-size: 1.2em;
    color: #666;
}

9. Attribute Selector ([])

/* Memilih berdasarkan atribut */
input[type="text"] {
    border: 1px solid #ccc;
}

input[type="submit"] {
    background: #3498db;
    color: white;
}

a[target="_blank"] {
    color: red;
}

a[href^="https"] {
    color: green; /* dimulai dengan https */
}

a[href$=".pdf"] {
    color: orange; /* diakhiri .pdf */
}

Tabel Ringkasan Selector

Selector Contoh Keterangan
Element p { } Semua <p>
Class .intro { } class="intro"
ID #header { } id="header"
Universal * { } Semua elemen
Group h1, h2 { } h1 dan h2
Descendant div p { } p di dalam div
Child div > p { } p anak langsung div
Adjacent h1 + p { } p setelah h1
Attribute [type="text"] { } Berdasarkan atribut

7. Box Model Penting!

Setiap elemen HTML dianggap sebagai sebuah "box" yang terdiri dari 4 bagian:

📦 Visualisasi Box Model:
CONTENT
↑ PADDING
↑ BORDER
↑ MARGIN

Komponen Box Model

Komponen Keterangan Warna di Demo
Content Area konten (teks, gambar) Putih
Padding Jarak antara content dan border Biru
Border Garis tepi elemen Hijau
Margin Jarak luar elemen Orange

Margin

.box {
    /* Semua sisi sama */
    margin: 20px;
    
    /* Atas-bawah | Kiri-kanan */
    margin: 20px 40px;
    
    /* Atas | Kiri-kanan | Bawah */
    margin: 10px 20px 30px;
    
    /* Atas | Kanan | Bawah | Kiri (clockwise) */
    margin: 10px 20px 30px 40px;
    
    /* Sisi individual */
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 30px;
    margin-left: 40px;
    
    /* Center horizontal */
    margin: 0 auto;
}

Padding

.box {
    /* Sama seperti margin */
    padding: 20px;
    padding: 10px 20px;
    padding: 10px 20px 30px 40px;
    
    /* Individual */
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
}

Width dan Height

.box {
    width: 300px;
    height: 200px;
    
    /* Persentase */
    width: 50%;
    width: 100%;
    
    /* Min dan Max */
    min-width: 200px;
    max-width: 1200px;
    min-height: 100px;
    max-height: 500px;
}

Box-Sizing

/* SANGAT PENTING! */
* {
    box-sizing: border-box;
}

/*
content-box (default): width = content saja
border-box: width = content + padding + border
*/

10. Display & Position Penting!

Display Property

.element {
    display: block;        /* elemen blok, lebar 100% */
    display: inline;       /* elemen sebaris */
    display: inline-block; /* sebaris tapi bisa atur width/height */
    display: none;         /* sembunyikan elemen */
    display: flex;         /* flexbox container */
    display: grid;         /* grid container */
}
Display Karakteristik Contoh Elemen
block Lebar 100%, baris baru div, p, h1-h6, ul, li
inline Sesuai konten, tidak bisa atur width/height span, a, strong, em
inline-block Sesuai konten, bisa atur width/height img, button, input
none Elemen tidak ditampilkan -

Position Property

/* Static (default) - posisi normal */
.static {
    position: static;
}

/* Relative - relatif dari posisi normal */
.relative {
    position: relative;
    top: 20px;
    left: 30px;
}

/* Absolute - relatif dari parent terdekat yang positioned */
.absolute {
    position: absolute;
    top: 0;
    right: 0;
}

/* Fixed - relatif dari viewport (tetap saat scroll) */
.fixed {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
}

/* Sticky - kombinasi relative dan fixed */
.sticky {
    position: sticky;
    top: 0;

Z-Index (Layering)

Overflow

.container {

    overflow: hidden;    /* potong konten yang keluar */

11. Flexbox CSS3

Flexbox adalah layout model yang memudahkan pengaturan elemen dalam satu dimensi (baris atau kolom).

Flex Container

.flex-container {
    display: flex;
    
    /* Arah: row (default), row-reverse, column, column-reverse */
    flex-direction: row;
    
    /* Wrap: nowrap (default), wrap, wrap-reverse */
    flex-wrap: wrap;
    
    /* Shorthand */
    flex-flow: row wrap;
    
    /* Justify content (main axis) */
    justify-content: flex-start;   /* default */
    justify-content: flex-end;
    justify-content: center;
    justify-content: space-between;
    justify-content: space-around;
    justify-content: space-evenly;
    
    /* Align items (cross axis) */
    align-items: stretch;    /* default */
    align-items: flex-start;
    align-items: flex-end;
    align-items: center;
    align-items: baseline;
    
    /* Gap (jarak antar item) */
    gap: 20px;
}

Flex Items

.flex-item {
    /* Flex grow: proporsi space yang diambil */
    flex-grow: 1;
    
    /* Flex shrink: proporsi shrink saat space kurang */
    flex-shrink: 0;
    
    /* Flex basis: ukuran awal sebelum grow/shrink */
    flex-basis: 200px;
    
    /* Shorthand: grow shrink basis */
    flex: 1 0 200px;
    flex: 1;  /* flex: 1 1 0 */
    
    /* Align self: override align-items untuk item ini */
    align-self: center;
    
    /* Order: urutan tampil */
    order: 1;
}

Contoh Layout dengan Flexbox

/* Navbar dengan Flexbox */
.navbar {
    display: flex;

12. CSS Grid CSS3

CSS Grid adalah layout model 2 dimensi (baris DAN kolom sekaligus).

Grid Container

    grid-template-columns: repeat(3, 1fr);    /* sama dengan di atas */
    grid-template-columns: 1fr 2fr 1fr;        /* kolom tengah 2x */

    /* Gap */
    gap: 20px;
    row-gap: 20px;
    column-gap: 20px;
    
    /* Alignment */
    justify-items: center;   /* horizontal */
    align-items: center;     /* vertical */
}

Device Breakpoint Media Query

15. Transition & Animation CSS3

Transition

Membuat perubahan style menjadi smooth/halus.

.button {
    background: #3498db;
    color: white;
    padding: 10px 20px;
    border: none;
    
    /* Transition: property duration timing-function delay */
    transition: all 0.3s ease;
    
    /* Atau terpisah */
    transition-property: background, transform;
    transition-duration: 0.3s;
    transition-timing-function: ease-in-out;
    transition-delay: 0s;
}

.button:hover {
    background: #2980b9;
    transform: scale(1.05);
}

Timing Functions

Timing Function Keterangan
ease Default, mulai lambat - cepat - lambat
linear Kecepatan konstan
ease-in Mulai lambat
ease-out Akhir lambat
ease-in-out Mulai dan akhir lambat
cubic-bezier() Custom curve

Transform

.element {
    /* Rotate */
    transform: rotate(45deg);
    
    /* Scale */
    transform: scale(1.5);      /* 150% */
    transform: scale(2, 0.5);  /* scaleX, scaleY */
    
    /* Translate (move) */
    transform: translateX(50px);
    transform: translateY(-20px);
    transform: translate(50px, -20px);
    
    /* Skew */
    transform: skew(10deg, 5deg);
    
    /* Multiple transforms */
    transform: rotate(45deg) scale(1.2) translateX(10px);
}

Animation

/* Definisi keyframes */
@keyframes bounce {
    0% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-30px);
    }
    100% {
        transform: translateY(0);
    }
}

@keyframes fadeIn {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}

/* Menggunakan animation */
.animated {
    /* animation: name duration timing-function delay iteration-count direction */
    animation: bounce 1s ease infinite;
    
    /* Atau terpisah */
    animation-name: bounce;
    animation-duration: 1s;
    animation-timing-function: ease;
    animation-delay: 0s;
    animation-iteration-count: infinite;  /* atau angka */
    animation-direction: alternate;  /* normal, reverse */
    animation-fill-mode: forwards;  /* tetap di posisi akhir */
}
