1. Cara Kerja Server Side CPMK 5.1

Client Side vs Server Side

Dalam pengembangan web, kode bisa dijalankan di dua tempat: client (browser) atau server. Pada pertemuan sebelumnya kita sudah belajar HTML, CSS, dan JavaScript yang berjalan di sisi client. Sekarang kita akan belajar PHP yang berjalan di sisi server.

Aspek Client Side Server Side
Eksekusi di Browser pengguna Web Server
Bahasa HTML, CSS, JavaScript PHP, Python, Java, Node.js, Ruby
Kode bisa dilihat user? Ya (View Source) Tidak (hanya output HTML yang dikirim)
Akses Database Tidak langsung Ya (MySQL, PostgreSQL, dll)
Akses File Server Tidak Ya
Contoh Animasi, validasi form, DOM Login, simpan data, kirim email

Alur detail:
  1. User membuka URL di browser → browser mengirim HTTP Request
  2. Request diterima oleh Web Server (Apache/Nginx)
  3. Jika file berekstensi .php, web server meneruskan ke PHP Engine
  4. PHP Engine mengeksekusi kode PHP, bisa juga mengakses Database
  5. PHP menghasilkan output berupa HTML
  6. Web Server mengirim HTML sebagai HTTP Response ke browser
  7. Browser menampilkan HTML (user hanya melihat HTML, bukan kode PHP)

Penting: File PHP tidak bisa dibuka langsung di browser seperti HTML. Harus dijalankan melalui web server! Simpan file di folder htdocs (XAMPP) lalu akses melalui http://localhost/namafile.php

2. Kelebihan & Kekurangan Server Side CPMK 5.2

Kelebihan Server Side

  • Keamanan - Kode tidak bisa dilihat oleh user
  • Akses Database - Bisa menyimpan dan mengambil data
  • Akses File System - Bisa membaca/menulis file di server
  • Otentikasi - Bisa mengelola login dan session
  • Konsisten - Hasilnya sama di semua browser
  • Tidak bergantung pada browser - Output sudah HTML final
  • Proses data besar - Bisa menangani data dalam jumlah besar

Kekurangan Server Side

  • Beban server - Semua request diproses di server
  • Latency - Setiap interaksi butuh request ke server
  • Biaya hosting - Butuh server yang cukup kuat
  • Tidak interaktif - Tidak bisa manipulasi DOM langsung
  • Page reload - Perlu refresh halaman untuk update (tanpa AJAX)
  • Butuh environment khusus - Harus ada web server + interpreter

Perbandingan Script Server Side

Bahasa Kelebihan Kekurangan Contoh Framework
PHP Mudah dipelajari, hosting murah, komunitas besar Desain bahasa kurang konsisten Laravel, CodeIgniter
Node.js JavaScript di server, cepat, real-time Callback hell, single-threaded Express, NestJS
Python Sintaks bersih, powerful, AI/ML Lebih lambat dari compiled Django, Flask
Java Enterprise, scalable, type-safe Verbose, setup kompleks Spring Boot
Ruby Elegant syntax, developer-friendly Performa lebih rendah Ruby on Rails
💡 Mengapa Belajar PHP?
  • Digunakan oleh ~77% website di dunia (termasuk WordPress, Facebook awal)
  • Mudah dipelajari untuk pemula
  • Hosting sangat murah dan tersedia di mana-mana
  • Ekosistem framework matang (Laravel, CodeIgniter)
  • Cocok untuk pemrograman web fundamental

3. Pengenalan PHP CPMK 5.3

Apa itu PHP?

PHP (PHP: Hypertext Preprocessor) adalah bahasa pemrograman server-side yang dirancang khusus untuk pengembangan web. PHP diciptakan oleh Rasmus Lerdorf pada tahun 1994.

Struktur Dasar PHP

<?php
    // Kode PHP ditulis di antara tag php
    echo "Hello, World!";
?>

<!DOCTYPE html>

    <meta charset="UTF-8">
    <title>PHP dalam HTML</title>

        // Variabel PHP

4. Variabel & Tipe Data PHP CPMK 5.3

Apa itu Variabel?

Variabel adalah wadah untuk menyimpan data sementara di memori selama program berjalan. Bayangkan variabel seperti sebuah kotak berlabel — labelnya adalah nama variabel, isinya adalah nilainya.

Ciri khas variabel PHP:
  • Selalu diawali tanda $ — ini wajib, bukan opsional
  • Case-sensitive — $Nama dan $nama adalah dua variabel berbeda
  • Loosely typed — tidak perlu deklarasi tipe, PHP menentukan tipe secara otomatis dari nilai yang diisi
  • Nilai bisa diubah kapan saja, bahkan berganti tipe

Aturan Penamaan Variabel

Aturan Contoh Benar ✅ Contoh Salah ❌
Dimulai dengan $ $nama nama
Karakter pertama: huruf atau _ $_data, $namaUser $1nama, $9x
Hanya huruf, angka, dan _ $nama_depan $nama-depan, $nama depan
Case-sensitive $Nama ≠ $nama —

Variabel vs Konstanta

Variabel

  • Nilainya bisa berubah
  • Diawali $
  • Berlaku di scope tertentu (lokal/global)
  • Contoh: $harga = 50000;

Konstanta

  • Nilainya tidak bisa diubah setelah ditetapkan
  • Tanpa $, biasanya HURUF_BESAR
  • Berlaku global di seluruh script
  • Contoh: define("PAJAK", 0.11);

Variabel PHP

<?php
// Variabel dimulai dengan tanda $
$nama = "John Doe";
$umur = 20;
$ipk = 3.75;
$aktif = true;

// PHP bersifat loosely typed (tipe otomatis)
$x = 10;       // integer
$x = "hello";  // sekarang string (otomatis berubah)

// Aturan penamaan variabel:
$namaDepan = "John";     // camelCase
$nama_depan = "John";    // snake_case
$_private = "data";      // boleh dimulai _
// $1nama = "salah";       // tidak boleh dimulai angka

// Konstanta
define("PI", 3.14159);
const NAMA_APP = "My Website";

echo PI;        // 3.14159
echo NAMA_APP;  // My Website
?>

Tipe Data PHP

PHP memiliki 8 tipe data yang dibagi menjadi 3 kelompok:

Scalar
String, Integer,
Float, Boolean
Compound
Array, Object
Special
NULL, Resource
Tipe Kelompok Keterangan Contoh
String Scalar Teks, diapit tanda kutip $s = "Hello";
Integer Scalar Bilangan bulat (positif/negatif) $i = 42;
Float Scalar Bilangan desimal / pecahan $f = 3.14;
Boolean Scalar Hanya dua nilai: true atau false $b = true;
Array Compound Kumpulan banyak nilai dalam satu variabel $a = [1, 2, 3];
Object Compound Instance dari sebuah class $o = new User();
NULL Special Variabel tanpa nilai / kosong $n = null;
Resource Special Referensi ke sumber eksternal (file, koneksi DB) $f = fopen("file.txt", "r");

Penjelasan Tiap Tipe Data

1. String

String adalah urutan karakter — bisa berupa kata, kalimat, angka dalam tanda kutip, bahkan HTML. Panjangnya tidak terbatas.

<?php
$nama    = "John Doe";       // double quotes
$kota    = 'Mataram';        // single quotes
$kosong  = "";               // string kosong
$angka   = "42";             // ini STRING, bukan integer!
$panjang = strlen($nama);    // 8 (jumlah karakter)
?>

2. Integer

Integer adalah bilangan bulat tanpa koma. PHP mendukung notasi desimal, heksadesimal, oktal, dan biner.

<?php
$positif = 100;
$negatif = -50;
$hex     = 0x1A;    // heksadesimal = 26
$oktal   = 0755;   // oktal = 493
$biner   = 0b1010; // biner = 10

echo PHP_INT_MAX; // nilai integer maksimum di sistem
?>

3. Float

Float (juga disebut double) adalah bilangan dengan koma desimal. Hati-hati dengan presisi float saat membandingkan nilai.

<?php
$ipk   = 3.75;
$suhu  = -12.5;
$sains = 1.5e3;   // notasi ilmiah = 1500

// ⚠️ Hati-hati perbandingan float!
$a = 0.1 + 0.2;
var_dump($a == 0.3);   // false! (masalah presisi floating point)
var_dump(round($a, 1) == 0.3); // true (gunakan round)
?>

4. Boolean

Boolean hanya punya dua nilai: true atau false. Sering digunakan dalam kondisi percabangan. PHP juga mengenal nilai yang dianggap falsy (dianggap false) dan truthy.

<?php
$login  = true;
$aktif  = false;

// Nilai yang dianggap FALSE (falsy):
var_dump((bool) 0);        // false
var_dump((bool) "");       // false (string kosong)
var_dump((bool) "0");      // false (string "0")
var_dump((bool) []);       // false (array kosong)
var_dump((bool) null);    // false

// Nilai yang dianggap TRUE (truthy):
var_dump((bool) 1);        // true
var_dump((bool) -1);       // true (angka non-nol)
var_dump((bool) "hello");  // true (string non-kosong)
var_dump((bool) [1]);      // true (array tidak kosong)
?>

5. NULL

NULL berarti variabel tidak memiliki nilai. Sebuah variabel bernilai NULL jika: belum diisi, di-assign null, atau di-unset().

<?php
$a = null;
$b;                        // variabel belum diisi = NULL

var_dump(is_null($a));     // true
var_dump(isset($a));       // false (NULL dianggap tidak di-set)

$c = "ada";
unset($c);                 // hapus variabel → jadi NULL
var_dump(isset($c));       // false
?>
isset() vs empty() vs is_null()
  • isset($x) — true jika variabel ada dan nilainya bukan NULL
  • empty($x) — true jika variabel kosong, 0, "", null, false, atau array kosong
  • is_null($x) — true hanya jika nilainya benar-benar NULL

String: Single vs Double Quotes

Perbedaan paling penting: double quotes memproses variabel di dalamnya, single quotes tidak. Pilih sesuai kebutuhan.

<?php
$nama = "John";

// Double quotes - variabel diparse ⭐
echo "Halo, $nama!";        // Output: Halo, John!
echo "Halo, {$nama}!";      // Output: Halo, John!

// Single quotes - variabel TIDAK diparse
echo 'Halo, $nama!';        // Output: Halo, $nama!

// Concatenation (titik .)
echo 'Halo, ' . $nama . '!'; // Output: Halo, John!

// Heredoc (multi-line, variabel diparse)
$html = <<<HTML
    <h1>Selamat Datang</h1>
    <p>Halo, $nama!</p>
HTML;

echo $html;
?>

Type Casting & Checking

Karena PHP loosely typed, kadang kita perlu memaksa konversi tipe (casting) atau memeriksa tipe sebuah variabel — terutama saat menerima input dari user.

<?php
// Mengecek tipe data
$x = 42;
var_dump($x);              // int(42)
echo gettype($x);          // "integer"
echo is_int($x);            // true
echo is_string($x);         // false

// Type casting (konversi paksa)
$str = "123";
$int = (int) $str;         // 123
$float = (float) "3.14";   // 3.14
$bool = (bool) 1;          // true
$arr = (array) "hello";    // ["hello"]

// Konversi function
$angka = intval("42abc");   // 42
$desimal = floatval("3.14"); // 3.14
$teks = strval(100);        // "100"
?>
💡 Ringkasan Penting:
  • Variabel PHP selalu diawali $ dan bersifat case-sensitive
  • PHP loosely typed — tipe ditentukan otomatis, tapi bisa berubah
  • Gunakan var_dump() untuk debug tipe + nilai sekaligus
  • Double quotes "..." memproses variabel; single quotes '...' tidak
  • Gunakan === (bukan ==) untuk perbandingan yang memeriksa tipe juga
  • isset() untuk cek variabel ada; empty() untuk cek variabel kosong/falsy
  • Konstanta (define / const) nilainya tidak bisa diubah dan berlaku global

5. Operator PHP CPMK 5.3

Operator adalah simbol yang digunakan untuk memproses data. Operator dipakai untuk menghitung, membandingkan, menggabungkan kondisi, memperbarui nilai variabel, dan menyusun string.

Hampir semua logika program bergantung pada operator. Jika mahasiswa paham operator, maka materi percabangan, perulangan, dan function akan jauh lebih mudah dipahami.

Jenis Operator yang Sering Dipakai

Jenis Contoh Fungsi
Aritmatika +, -, *, /, % Melakukan perhitungan
Perbandingan ==, ===, >, < Menghasilkan nilai true atau false
Logika &&, ||, ! Menggabungkan beberapa kondisi
Assignment =, +=, -= Mengisi atau mengubah nilai variabel
String ., .= Menggabungkan teks 
<?php
// === OPERATOR ARITMATIKA ===
$a = 10; $b = 3;
echo $a + $b;    // 13
echo $a - $b;    // 7
echo $a * $b;    // 30
echo $a / $b;    // 3.333...
echo $a % $b;    // 1 (modulus)
echo $a ** $b;   // 1000 (pangkat)

// === OPERATOR PERBANDINGAN ===
$x = 5; $y = "5";
var_dump($x == $y);     // true (sama nilai)
var_dump($x === $y);    // false (beda tipe) ⭐
var_dump($x != $y);     // false
var_dump($x !== $y);    // true ⭐
var_dump($x > 3);       // true
var_dump($x <=> 10);    // -1 (spaceship operator PHP7)

// === OPERATOR LOGIKA ===
var_dump(true && false);  // false (AND)
var_dump(true || false);  // true (OR)
var_dump(!true);           // false (NOT)

// === OPERATOR STRING ===
$str = "Hello" . " World";  // concatenation (titik)
$str .= "!";                // "Hello World!"

// === ASSIGNMENT ===
$x = 10;
$x += 5;   // $x = $x + 5  → 15
$x -= 3;   // $x = $x - 3  → 12
$x *= 2;   // $x = $x * 2  → 24
$x /= 4;   // $x = $x / 4  → 6

// === NULL COALESCING (PHP 7) ===
$nama = $_GET['nama'] ?? "Guest";
// Jika $_GET['nama'] null/tidak ada, gunakan "Guest"
?>
Catatan penting:
  • == mengecek kesamaan nilai, sedangkan === mengecek nilai sekaligus tipe data.
  • Untuk string, PHP memakai operator ., bukan tanda +.
  • && berarti semua kondisi harus benar, sedangkan || berarti cukup salah satu benar.
  • ?? berguna saat membaca data dari $_GET, $_POST, atau array yang mungkin belum ada.

Contoh Kasus Nyata

<?php
$harga = 150000;
$diskon = 20;
$member = true;

$potongan = $harga * $diskon / 100;
$total = $harga - $potongan;

if ($member && $total > 100000) {
    $total -= 10000;
}

echo "Total bayar: Rp " . number_format($total, 0, ",", ".");
?>

6. Percabangan CPMK 5.3

Percabangan adalah cara program memilih aksi berdasarkan kondisi tertentu. Jika kondisi benar, jalankan blok A. Jika salah, jalankan blok B.

Konsep ini dipakai di hampir semua aplikasi web: login berhasil atau gagal, nilai lulus atau tidak, stok tersedia atau habis, admin atau user biasa.

If - Else If - Else

📄 grade.php 
<?php
$nilai = 85;

if ($nilai >= 90) {
    $grade = "A";
    $keterangan = "Sangat Baik";
} elseif ($nilai >= 80) {
    $grade = "B";
    $keterangan = "Baik";
} elseif ($nilai >= 70) {
    $grade = "C";
    $keterangan = "Cukup";
} elseif ($nilai >= 60) {
    $grade = "D";
    $keterangan = "Kurang";
} else {
    $grade = "E";
    $keterangan = "Sangat Kurang";
}

echo "Nilai: $nilai → Grade: $grade ($keterangan)";
?>
Cara kerja struktur ini:
  1. Program mengecek kondisi pada if terlebih dahulu.
  2. Jika salah, program lanjut mengecek elseif berikutnya.
  3. Jika semua salah, maka blok else dijalankan.

If di dalam HTML (Alternative Syntax)

<?php $isLogin = true; $nama = "John"; ?>

<div class="header">
    <?php if ($isLogin): ?>
        <p>Selamat datang, <?= $nama ?>!</p>
        <a href="logout.php">Logout</a>
    <?php else: ?>
        <a href="login.php">Login</a>
        <a href="register.php">Daftar</a>
    <?php endif; ?>
</div>

Switch

<?php
$hari = date("l"); // nama hari saat ini

switch ($hari) {
    case "Monday":
        echo "Hari Senin - Semangat awal pekan!";
        break;
    case "Friday":
        echo "Hari Jumat - Weekend segera!";
        break;
    case "Saturday":
    case "Sunday":
        echo "Weekend! Istirahat 😊";
        break;
    default:
        echo "Hari biasa, tetap semangat!";
}
?>

Ternary Operator & Match (PHP 8)

<?php
// Ternary
$umur = 20;
$status = ($umur >= 17) ? "Dewasa" : "Anak-anak";

// Match expression (PHP 8+)
$nilai = 85;
$grade = match(true) {
    $nilai >= 90 => "A",
    $nilai >= 80 => "B",
    $nilai >= 70 => "C",
    default      => "D",
};
?>

Kapan Menggunakan yang Mana?

Bentuk Cocok untuk
if / elseif / else Kondisi yang fleksibel dan kompleks
switch Satu variabel dibandingkan dengan banyak pilihan tetap
ternary Keputusan singkat dalam satu baris
match Alternatif yang lebih ringkas dan tegas pada PHP 8+

Contoh Kasus Nyata

<?php
$stok = 0;

if ($stok > 0) {
    echo "Produk tersedia";
} else {
    echo "Produk habis";
}
?>

7. Perulangan CPMK 5.3

Perulangan digunakan untuk menjalankan blok kode yang sama berkali-kali. Ini membuat kode lebih singkat dibanding menulis perintah yang sama secara manual.

Perulangan sangat berguna saat membuat daftar, tabel, nomor urut, menghitung total, atau memproses sekumpulan data dari array dan database.

For Loop

📄 tabel_perkalian.php 
<?php
// Membuat tabel perkalian
echo "<table border='1'>";
echo "<caption>Tabel Perkalian</caption>";

for ($i = 1; $i <= 5; $i++) {
    echo "<tr>";
    for ($j = 1; $j <= 5; $j++) {
        $hasil = $i * $j;
        echo "<td>$i × $j = $hasil</td>";
    }
    echo "</tr>";
}

echo "</table>";
?>

for biasanya dipakai jika jumlah perulangan sudah diketahui dengan jelas, misalnya dari 1 sampai 10.

While & Do-While

<?php
// While
$i = 1;
while ($i <= 5) {
    echo "Iterasi ke-$i<br>";
    $i++;
}

// Do-While
$j = 1;
do {
    echo "Angka: $j<br>";
$j++;
} while ($j <= 5);
?>

while

  • Kondisi dicek di awal
  • Bisa tidak berjalan sama sekali
  • Cocok saat jumlah iterasi belum pasti

do-while

  • Blok dijalankan minimal sekali
  • Kondisi dicek di akhir
  • Cocok jika aksi harus dicoba dulu

Foreach (untuk Array) ⭐

<?php
// Foreach - indexed array
$buah = ["Apel", "Jeruk", "Mangga"];

foreach ($buah as $item) {
    echo "Buah: $item<br>";
}

// Foreach - dengan index
foreach ($buah as $index => $item) {
    echo "$index: $item<br>";
}

// Foreach - associative array
$mahasiswa = [
    "nama" => "John",
    "nim" => "12345",
    "jurusan" => "Informatika"
];

foreach ($mahasiswa as $key => $value) {
    echo "$key: $value<br>";
}
?>

Foreach di dalam HTML (Membuat Tabel Dinamis)

<?php
$data_mahasiswa = [
    ["nama" => "John",  "nim" => "001", "jurusan" => "Informatika"],
    ["nama" => "Jane",  "nim" => "002", "jurusan" => "Sistem Informasi"],
    ["nama" => "Ahmad", "nim" => "003", "jurusan" => "Teknik Komputer"],
];
?>

<table border="1">
    <thead>
        <tr>
            <th>No</th>
            <th>Nama</th>
            <th>NIM</th>
            <th>Jurusan</th>
        </tr>
    </thead>
    <tbody>
        <?php $no = 1; ?>
        <?php foreach ($data_mahasiswa as $mhs): ?>
        <tr>
            <td><?= $no++ ?></td>
            <td><?= $mhs['nama'] ?></td>
            <td><?= $mhs['nim'] ?></td>
            <td><?= $mhs['jurusan'] ?></td>
        </tr>
        <?php endforeach; ?>
    </tbody>
</table>

Kapan Memakai Loop Tertentu?

Loop Cocok untuk
for Jumlah iterasi sudah diketahui
while Berjalan selama kondisi masih benar
do-while Harus dijalankan minimal satu kali
foreach Menelusuri isi array atau daftar data
Jebakan umum:
  • Lupa menambah counter seperti $i++, sehingga loop tidak berhenti.
  • Salah menentukan batas loop, misalnya memakai < padahal seharusnya <=.
  • Memakai for untuk array padahal foreach lebih sederhana dan aman.

8. Function PHP CPMK 5.4

Function adalah sekumpulan kode yang diberi nama agar bisa dipanggil kembali saat dibutuhkan. Dengan function, kita tidak perlu menulis logika yang sama berulang-ulang.

Manfaat function adalah membuat kode lebih rapi, mudah dibaca, mudah diuji, dan mudah diperbaiki jika ada perubahan aturan.

<?php
// Function dasar
function sapa() {
    echo "Halo, Selamat Datang!<br>";
}
sapa();

// Function dengan parameter
function sapaUser($nama) {
    echo "Halo, $nama!<br>";
}
sapaUser("John");

// Function dengan return
function tambah($a, $b) {
    return $a + $b;
}
$hasil = tambah(5, 3); // 8

// Default parameter
function hitungDiskon($harga, $diskon = 10) {
    return $harga - ($harga * $diskon / 100);
}
echo hitungDiskon(100000);      // 90000
echo hitungDiskon(100000, 25); // 75000

// Type declaration (PHP 7+)
function jumlah(int $a, int $b): int {
    return $a + $b;
}

// Nullable type
function cariUser(int $id): ?string {
    if ($id === 1) return "John";
    return null;
}

// Arrow function (PHP 7.4+)
$kuadrat = fn($x) => $x * $x;
echo $kuadrat(5); // 25
?>

Bagian Penting Function

Bagian Penjelasan
Nama function Nama proses yang ingin dipanggil, misalnya hitungDiskon()
Parameter Data masukan yang dibutuhkan function
Argumen Nilai nyata yang dikirim saat function dipanggil
return Nilai hasil yang dikembalikan oleh function

Contoh Kasus Nyata

<?php
function hitungNilaiAkhir($tugas, $uts, $uas) {
    return ($tugas * 0.3) + ($uts * 0.3) + ($uas * 0.4);
}

$nilaiAkhir = hitungNilaiAkhir(80, 75, 90);
echo $nilaiAkhir; // 82.5
?>
Jebakan umum:
  • Jika hasil function ingin dipakai lagi, gunakan return, bukan hanya echo.
  • Gunakan nama function yang jelas agar maksudnya langsung terlihat.
  • Jangan membuat satu function mengerjakan terlalu banyak tugas sekaligus.

9. Array PHP CPMK 5.4

10. String & Fungsi Bawaan PHP CPMK 5.4

String adalah tipe data untuk menyimpan teks, misalnya nama user, email, alamat, judul artikel, atau isi pesan. Dalam PHP, string sangat sering diproses sebelum ditampilkan ke halaman, disimpan ke database, atau dicek isinya.

Karena itu, memahami fungsi bawaan sangat penting. Fungsi bawaan adalah fungsi yang sudah disediakan oleh PHP, sehingga kita tidak perlu membuat semuanya dari nol. Tinggal panggil fungsi yang sesuai, lalu kirim data yang ingin diproses.

Konsep Dasar String

<?php
$nama = "Budi";
$umur = 20;

// 1. Concatenation (menggabungkan string)
$pesan1 = "Halo, " . $nama . "!";

// 2. Interpolation dengan double quote
$pesan2 = "Halo, $nama! Umur kamu $umur tahun.";

// 3. Single quote tidak membaca variabel secara langsung
$pesan3 = 'Halo, $nama!';

echo $pesan1; // Halo, Budi!
echo $pesan2; // Halo, Budi! Umur kamu 20 tahun.
echo $pesan3; // Halo, $nama!
?>
Poin penting:
  • Gunakan tanda titik . untuk menggabungkan string.
  • "double quote" bisa membaca variabel langsung di dalam string.
  • 'single quote' lebih literal, sehingga isi string dianggap teks biasa.

String Functions

Fungsi-fungsi berikut dipakai untuk kebutuhan yang berbeda. Ada yang untuk menghitung panjang teks, membersihkan spasi, mengubah huruf besar-kecil, mencari kata tertentu, sampai memotong sebagian isi string.

<?php
$str = "  Hello World  ";

strlen($str);                // 15 (panjang string)
trim($str);                  // "Hello World"
strtoupper($str);             // "  HELLO WORLD  "
strtolower($str);             // "  hello world  "
ucfirst("hello");             // "Hello"
ucwords("hello world");       // "Hello World"
str_replace("World", "PHP", $str); // "Hello PHP"
strpos($str, "World");       // 8 (posisi ditemukan)
substr($str, 2, 5);          // "Hello"
str_contains($str, "Hello"); // true (PHP 8)
str_starts_with($str, "  H"); // true (PHP 8)
nl2br("baris1\nbaris2");     // "baris1<br>baris2"
number_format(1234567.89, 2, ",", "."); // "1.234.567,89"
str_pad("5", 3, "0", STR_PAD_LEFT); // "005"
?>
Fungsi Kegunaan Contoh Hasil
strlen() Menghitung jumlah karakter strlen("PHP") → 3
trim() Menghapus spasi di awal dan akhir " Budi " → "Budi"
strtoupper() Mengubah semua huruf menjadi kapital "php" → "PHP"
strtolower() Mengubah semua huruf menjadi kecil "PHP" → "php"
str_replace() Mengganti teks tertentu "Hello World" → "Hello PHP"
strpos() Mencari posisi teks strpos("Hello", "e") → 1
substr() Mengambil sebagian string substr("Informatika", 0, 5) → "Infor"
str_contains() Mengecek apakah teks mengandung substring true atau false

Contoh Pemakaian String dalam Kasus Nyata

Biasanya string tidak diproses satu fungsi saja. Dalam program nyata, satu data bisa dibersihkan dulu, dicek, lalu diformat sebelum ditampilkan.

<?php
$namaInput = "  aNiSa putri  ";

// Bersihkan spasi di awal/akhir
$namaBersih = trim($namaInput);

// Ubah jadi huruf kecil semua
$namaLower = strtolower($namaBersih);

// Jadikan setiap kata diawali huruf kapital
$namaTampil = ucwords($namaLower);

echo $namaTampil; // "Anisa Putri"
?>
Alur logikanya:
  1. trim() dipakai agar input tidak memiliki spasi berlebih.
  2. strtolower() membuat format huruf menjadi konsisten.
  3. ucwords() dipakai agar tampilan nama lebih rapi.

Jebakan Umum Saat Mengolah String

<?php
$teks = "Halo Dunia";

// Hati-hati: strpos() bisa mengembalikan 0
if (strpos($teks, "Halo") !== false) {
    echo "Kata ditemukan";
}

// Jangan pakai == false, karena posisi 0 bisa dianggap false
?>

strpos() mengembalikan posisi karakter pertama. Jika teks ditemukan di posisi awal, hasilnya adalah 0. Karena 0 sering dianggap false, maka pengecekan yang aman adalah !== false.

Date & Time Functions

Selain string, PHP juga menyediakan banyak fungsi bawaan lain. Dua kelompok yang paling sering dipakai adalah fungsi tanggal/waktu dan fungsi matematika.

<?php
echo date("Y-m-d");         // 2024-01-15
echo date("d/m/Y");         // 15/01/2024
echo date("H:i:s");         // 14:30:00
echo date("l, d F Y");     // Monday, 15 January 2024
echo time();                 // Unix timestamp
echo strtotime("next Monday"); // timestamp Senin depan

// Format: Y=tahun4, m=bulan2, d=tanggal2, H=jam24, i=menit, s=detik
// l=nama hari, F=nama bulan lengkap, D=nama hari singkat
?>

Fungsi tanggal berguna untuk membuat timestamp posting, deadline tugas, log aktivitas, atau menampilkan waktu saat ini. Biasanya time() menghasilkan angka timestamp, lalu date() dipakai untuk memformatnya agar mudah dibaca manusia.

Math Functions

<?php
echo abs(-5);         // 5
echo round(3.7);      // 4
echo ceil(3.2);       // 4
echo floor(3.9);      // 3
echo max(1, 5, 3);   // 5
echo min(1, 5, 3);   // 1
echo pow(2, 3);      // 8
echo sqrt(16);       // 4
echo rand(1, 100);   // random 1-100
?>

Fungsi matematika umum dipakai saat menghitung total harga, rata-rata nilai, pembulatan diskon, atau membuat angka acak untuk simulasi dan kuis.

12. Hands-On: Pengiriman Data HTML → PHP Praktek!

Kita akan berlatih mengirim data dari form HTML ke PHP melalui 3 contoh bertahap. Fokus pada cara kerja pengiriman data — belum perlu CSS.

Persiapan:
  1. Jalankan Apache di XAMPP Control Panel
  2. Buat folder C:\xampp\htdocs\latihan\
  3. Simpan semua file latihan di folder tersebut
  4. Akses via browser: http://localhost/latihan/namafile.php

1 Form GET — Sapa Pengguna

Form paling sederhana: input nama, kirim via GET, tampilkan di halaman baru. Perhatikan nama muncul di URL setelah submit.

📄 sapa.php (form) 
<!DOCTYPE html>
<html>
<body>
    <h2>Form Sapa</h2>
    <form action="hasil_sapa.php" method="GET">
        Nama: <input type="text" name="nama">
        <button type="submit">Kirim</button>
    </form>
</body>
</html>
📄 hasil_sapa.php (proses) 
<?php
$nama = $_GET['nama'] ?? 'Tamu';
echo "<h2>Halo, $nama!</h2>";
echo '<a href="sapa.php">Kembali</a>';
?>
Coba perhatikan: setelah submit, URL berubah menjadi hasil_sapa.php?nama=John — itulah cara kerja GET, data terlihat di URL.

2 Form POST — Data Biodata

Kirim beberapa field sekaligus via POST. Data tidak muncul di URL. Cocok untuk data yang lebih sensitif. Pada contoh ini, PHP hanya memproses data jika tombol submit benar-benar diklik.

📄 biodata.php (form) 
<!DOCTYPE html>
<html>
<body>
    <h2>Form Biodata</h2>
    <form action="hasil_biodata.php" method="POST">
        Nama : <input type="text" name="nama"><br><br>
        NIM  : <input type="text" name="nim"><br><br>
        Umur : <input type="number" name="umur"><br><br>
        <button type="submit" name="kirim">Kirim</button>
    </form>
</body>
</html>
📄 hasil_biodata.php (proses) 
<?php
if (isset($_POST['kirim'])) {
    $nama = trim($_POST['nama'] ?? '');
    $nim  = trim($_POST['nim']  ?? '');
    $umur = trim($_POST['umur'] ?? '');

    echo "<h2>Data Biodata</h2>";

    if ($nama === '' || $nim === '' || $umur === '') {
        echo "Semua field wajib diisi.<br>";
    } else {
        echo "Nama : $nama<br>";
        echo "NIM  : $nim<br>";
        echo "Umur : $umur tahun<br>";
    }
} else {
    echo "Data belum dikirim melalui tombol submit.<br>";
}

echo '<br><a href="biodata.php">Kembali</a>';
?>
Catatan: pengecekan isset($_POST['kirim']) memastikan file proses hanya berjalan ketika form dikirim lewat tombol bernama kirim, bukan saat file dibuka langsung.

3 Self-Processing Form — Satu File

Form dan proses PHP digabung dalam satu file. Pada versi ini, PHP memproses data hanya jika request berupa POST dan tombol submit benar-benar diklik.

📄 nilai.php (form + proses) 
<?php
$hasil = '';
$error = '';

if ($_SERVER['REQUEST_METHOD'] === 'POST' && isset($_POST['kirim'])) {
    $nama  = trim($_POST['nama']  ?? '');
    $nilai = (int) ($_POST['nilai'] ?? 0);

    if ($nama === '') {
        $error = 'Nama wajib diisi.';
    } else {
        if ($nilai >= 80) {
            $grade = 'A';
        } elseif ($nilai >= 70) {
            $grade = 'B';
        } elseif ($nilai >= 60) {
            $grade = 'C';
        } else {
            $grade = 'D';
        }

        $hasil = "$nama mendapat nilai $nilai → Grade: $grade";
    }
}
?>

<!DOCTYPE html>
<html>
<body>
    <h2>Cek Nilai</h2>

    <?php if ($error): ?>
        <p><strong><?= $error ?></strong></p>
    <?php endif; ?>

    <?php if ($hasil): ?>
        <p><strong><?= $hasil ?></strong></p>
    <?php endif; ?>

    <form method="POST">
        Nama  : <input type="text" name="nama"><br><br>
        Nilai : <input type="number" name="nilai" min="0" max="100"><br><br>
        <button type="submit" name="kirim">Cek Grade</button>
    </form>
</body>
</html>
Cara kerjanya: Saat halaman pertama dibuka, REQUEST_METHOD adalah GET sehingga blok PHP dilewati dan hanya form yang tampil. Setelah tombol bernama kirim diklik, request menjadi POST dan $_POST['kirim'] akan terisi, lalu PHP memvalidasi input dan menampilkan hasilnya.
Poin penting dari 3 latihan ini:
  • GET → data dikirim lewat URL, cocok untuk pencarian/filter
  • POST → data dikirim di body request, tidak terlihat di URL
  • Self-processing → satu file menangani form sekaligus prosesnya, cek dengan $_SERVER['REQUEST_METHOD']
  • Atribut name di input HTML = kunci untuk mengambil data di PHP ($_GET['name'] / $_POST['name'])

14. Class & OOP PHP CPMK 5.4

Apa itu OOP?

Object-Oriented Programming (OOP) adalah paradigma pemrograman yang mengorganisir kode berdasarkan objek — representasi dari benda atau konsep di dunia nyata. Setiap objek memiliki data (properti) dan perilaku (method).

Analogi Sederhana:
Bayangkan sebuah Mobil:
  • Properti (data): warna, merek, kecepatan, bahan bakar
  • Method (perilaku): nyalakan(), rem(), tambahKecepatan(), isi_bensin()
Dalam OOP, kita membuat blueprint (class) Mobil, lalu dari blueprint itu kita bisa membuat banyak objek mobil yang berbeda-beda.

Pemrograman Prosedural vs OOP

Aspek Prosedural OOP
Struktur kode Urutan fungsi dan variabel terpisah Data dan fungsi dibungkus dalam class
Reusability Fungsi bisa dipakai ulang Class bisa dipakai ulang & diwariskan
Skalabilitas Sulit untuk proyek besar Lebih mudah dikembangkan
Contoh hitungNilai($nama, $uts, $uas) $mhs->hitungNilai()

4 Pilar OOP

1. Encapsulation (Enkapsulasi)

Menyembunyikan detail internal dan hanya mengekspos apa yang perlu. Data dilindungi dari akses langsung luar.

Analogi: Mesin mobil tersembunyi di balik kap, kita cukup pakai setir & pedal.

2. Inheritance (Pewarisan)

Class baru bisa mewarisi properti dan method dari class yang sudah ada, lalu menambah atau mengubahnya.

Analogi: Mobil Sport mewarisi sifat Mobil, tapi punya turbo tambahan.

3. Polymorphism (Polimorfisme)

Satu method yang sama bisa berperilaku berbeda tergantung objeknya (method overriding).

Analogi: Hewan berbeda-beda cara "bersuara()" — anjing gonggong, kucing mengeong.

4. Abstraction (Abstraksi)

Menyederhanakan kompleksitas dengan hanya menampilkan fitur penting, menyembunyikan detail implementasi.

Analogi: Kita tahu tombol "ON" menyalakan TV, tanpa perlu tahu rangkaian elektroniknya.

Konsep Dasar: Class & Object

→
new Mahasiswa()

📦 Object
Instansi lain

Satu class bisa menghasilkan banyak object. Seperti cetakan kue yang sama bisa menghasilkan banyak kue.

Anatomi Sebuah Class

Sebuah class PHP terdiri dari:
  • Properties — variabel milik class (data/atribut)
  • Constructor — method khusus __construct() yang otomatis dipanggil saat object dibuat
  • Methods — fungsi milik class (perilaku/aksi)
  • Access Modifier — public, protected, private untuk mengatur siapa yang bisa mengakses

Getter & Setter (Enkapsulasi)

Karena properti private tidak bisa diakses langsung dari luar, kita gunakan getter (untuk membaca) dan setter (untuk mengubah) sebagai "pintu resmi" akses data.

<?php
// TANPA enkapsulasi (berbahaya)
class BankBuruk {
    public float $saldo = 0;
}
$akun = new BankBuruk();
$akun->saldo = -999999; // ❌ bisa diisi nilai negatif!

// DENGAN enkapsulasi (aman)
class Bank {
    private float $saldo = 0;

    // Getter
    public function getSaldo(): float {
        return $this->saldo;
    }

    // Setter dengan validasi
    public function setor(float $jumlah): void {
        if ($jumlah > 0) {
            $this->saldo += $jumlah;
        }
    }
}
$akun = new Bank();
$akun->setor(500000);
echo $akun->getSaldo(); // 500000
?>

Membuat Class

<?php
class Mahasiswa {
    // Properties
    public string $nama;
    public string $nim;
    private float $ipk;
    
    // Constructor
    public function __construct(string $nama, string $nim, float $ipk = 0) {
        $this->nama = $nama;
        $this->nim  = $nim;
        $this->ipk  = $ipk;
    }
    
    // Methods
    public function perkenalan(): string {
        return "Halo, saya {$this->nama} (NIM: {$this->nim})";
    }
    
    public function setIpk(float $ipk): void {
        if ($ipk >= 0 && $ipk <= 4) {
            $this->ipk = $ipk;
        }
    }
    
    public function getIpk(): float {
        return $this->ipk;
    }
    
    public function getPredikat(): string {
        return match(true) {
            $this->ipk >= 3.5 => "Cum Laude",
            $this->ipk >= 3.0 => "Sangat Memuaskan",
            $this->ipk >= 2.5 => "Memuaskan",
            default             => "Cukup",
        };
    }
}

// Membuat object
$mhs1 = new Mahasiswa("John Doe", "12345", 3.75);
$mhs2 = new Mahasiswa("Jane Doe", "12346");

echo $mhs1->perkenalan() . "<br>";
echo "IPK: " . $mhs1->getIpk() . " → " . $mhs1->getPredikat();

$mhs2->setIpk(3.2);
echo $mhs2->perkenalan() . "<br>";
?>

Inheritance (Pewarisan)

Inheritance memungkinkan sebuah class mewarisi properti dan method dari class lain. Class yang mewarisi disebut child class, class yang diwarisi disebut parent class. Gunakan keyword extends.

👤 Orang
nama, umur
sapa()
↓
extends
🎓 Mahasiswa
+ nim, jurusan
+ getInfo()
👨‍🏫 Dosen
+ nip, matkul
+ getMengajar()
<?php
class Orang {
    public function __construct(
        protected string $nama,
        protected int $umur
    ) {}
    
    public function sapa(): string {
        return "Halo, saya {$this->nama}";
    }
}

class Mahasiswa extends Orang {
    public function __construct(
        string $nama,
        int $umur,
        private string $nim,
        private string $jurusan
    ) {
        parent::__construct($nama, $umur);
    }
    
    // Override method parent
    public function sapa(): string {
        return parent::sapa() . ", mahasiswa {$this->jurusan}";
    }
    
    public function getInfo(): string {
        return "{$this->nama} | NIM: {$this->nim} | {$this->jurusan}";
    }
}

$mhs = new Mahasiswa("John", 20, "12345", "Informatika");
echo $mhs->sapa();      // "Halo, saya John, mahasiswa Informatika"
echo $mhs->getInfo();   // "John | NIM: 12345 | Informatika"
?>

Polymorphism (Polimorfisme)

Polymorphism terjadi ketika child class mengubah perilaku method yang diwarisi dari parent class (method overriding). Satu nama method, perilaku berbeda tergantung objeknya.

<?php
class Hewan {
    public function bersuara(): string {
        return "...";
    }
}

class Anjing extends Hewan {
    public function bersuara(): string {
        return "Guk guk!";
    }
}

class Kucing extends Hewan {
    public function bersuara(): string {
        return "Meow!";
    }
}

// Polimorfisme: method sama, hasil berbeda
$hewan = [new Anjing(), new Kucing(), new Anjing()];
foreach ($hewan as $h) {
    echo get_class($h) . ": " . $h->bersuara() . "<br>";
}
// Anjing: Guk guk!
// Kucing: Meow!
// Anjing: Guk guk!
?>

Visibility (Access Modifier)

Access modifier menentukan siapa yang boleh mengakses properti atau method sebuah class.

Modifier Dari Class Sendiri Dari Child Class Dari Luar Class public ✅ ✅ ✅ protected ✅ ✅ ❌ private ✅ ❌ ❌
💡 Ringkasan OOP PHP:
  • Class = blueprint/cetakan; Object = hasil cetakan (instansi)
  • $this = merujuk ke object itu sendiri di dalam class
  • __construct() = otomatis dipanggil saat new NamaClass()
  • Encapsulation = lindungi data dengan private + getter/setter
  • Inheritance = extends untuk mewarisi class lain
  • parent:: = memanggil method/constructor dari parent class
  • Polymorphism = override method parent di child class

Tugas 4A: Konversi Suhu

Buat aplikasi konversi suhu dengan PHP:

  • Form input angka suhu dan pilihan konversi (Celsius → Fahrenheit / Fahrenheit → Celsius / Celsius → Kelvin)
  • Proses konversi menggunakan PHP (self-processing form)
  • Tampilkan hasil di halaman yang sama
  • Gunakan function PHP untuk setiap konversi
  • Validasi: input harus angka

Rumus:

  • C → F: (C × 9/5) + 32
  • F → C: (F - 32) × 5/9
  • C → K: C + 273.15

Tugas 4B: Biodata dengan Form Multi-Page

Buat aplikasi biodata dengan 2 halaman:

  • Halaman 1 (form.php): Form input biodata (Nama, NIM, Email, Telepon, Alamat, Jenis Kelamin, Hobi [checkbox multiple], Jurusan [dropdown])
  • Halaman 2 (hasil.php): Tampilkan semua data yang diinputkan dalam format tabel yang rapi
  • Gunakan method POST
  • Validasi semua input wajib
  • Tampilkan pesan error jika ada field kosong
  • Bonus: Tambahkan tombol "Kembali ke Form" dengan data sebelumnya tetap terisi

Tugas 4C: Tabel Nilai Mahasiswa

Buat halaman yang menampilkan tabel nilai:

  • Buat array multidimensi berisi data minimal 5 mahasiswa (Nama, NIM, Nilai UTS, Nilai UAS)
  • Tampilkan dalam tabel HTML yang di-generate PHP
  • Hitung Nilai Akhir secara otomatis: NA = (UTS × 40%) + (UAS × 60%)
  • Tentukan Grade otomatis (A/B/C/D/E) menggunakan function
  • Beri warna berbeda pada baris yang nilainya < 60 (merah/kuning)
  • Tampilkan rata-rata kelas di bawah tabel
  • Bonus: Buat class Mahasiswa untuk menyimpan data
